### 视频模板
```
主画面（配音时长）→ 结尾LOGO画面（3秒）
     ↓                    ↓
  宣传图内容           LOGO居中放大（300px）
  + 配音解说            + 店铺信息
                       + 蓝色背景
```

### 视频制作规则（永恒记住）
1. **照片数量**：3-5张连贯照片或相近照片
2. **配音**：使用 edge_tts（zh-CN-XiaoxiaoNeural）
3. **背景音乐**：轻音乐（从 Pixabay 下载）
4. **输出格式**：MP4，1080×1920，24fps
5. **结尾画面**：LOGO居中 + 店铺信息（3秒）

### 视频制作流程
```
1. 选取3-5张连贯/相近照片
2. 生成配音文本（edge_tts）
3. 下载轻音乐背景（Pixabay）
4. 合成视频（MoviePy）
5. 添加结尾LOGO画面
6. 输出MP4文件
```

### 推荐轻音乐（Pixabay）
| 音乐名称 | 风格 | 适用场景 |
|----------|------|----------|

### 制作规则（永恒执行）

2. **照片圆角**：照片区域圆角处理（radius=30）
