# API_DEPRECATION_POLICY.md - API 下线与废弃规则

## 目的
定义 API 下线与废弃规则，确保平台平稳演进。

## 适用范围
所有对外公开的 API 接口。

---

## 一、废弃触发条件

### 1.1 废弃原因
| 原因 | 说明 |
|------|------|
| 功能替代 | 有更好的替代接口 |
| 安全问题 | 存在安全风险 |
| 性能问题 | 影响系统性能 |
| 合规要求 | 不符合法规要求 |
| 技术演进 | 技术架构变更 |

### 1.2 废弃评估
| 评估项 | 说明 |
|--------|------|
| 使用量 | 当前使用量统计 |
| 影响范围 | 受影响用户数量 |
| 替代方案 | 是否有替代接口 |
| 迁移成本 | 用户迁移成本 |
| 风险等级 | 废弃风险等级 |

---

## 二、废弃公告

### 2.1 公告内容
| 内容 | 说明 |
|------|------|
| 废弃接口 | 被废弃的接口列表 |
| 废弃原因 | 废弃的原因说明 |
| 替代接口 | 推荐的替代接口 |
| 时间线 | 废弃时间安排 |
| 迁移指南 | 迁移步骤说明 |

### 2.2 公告渠道
| 渠道 | 优先级 | 说明 |
|------|--------|------|
| 开发者邮件 | 高 | 直达受影响用户 |
| 文档更新 | 高 | 更新 API 文档 |
| Changelog | 中 | 版本变更记录 |
| 控制台通知 | 中 | 登录时提示 |
| 社区公告 | 低 | 社区渠道发布 |

### 2.3 公告时间线
```
T-18月: 首次公告（stable 接口）
T-12月: 发布替代版本
T-6月:  强制迁移提醒
T-3月:  最终警告
T-1月:  停止新用户访问
T-0:    完全下线
```

---

## 三、替代接口

### 3.1 替代接口要求
| 要求 | 说明 |
|------|------|
| 功能对等 | 功能不低于原接口 |
| 性能更优 | 性能优于原接口 |
| 向前兼容 | 尽量向前兼容 |
| 文档完整 | 有完整迁移文档 |

### 3.2 替代接口发布
```
发布替代接口 → 标记为推荐 → 逐步迁移用户 → 废弃旧接口
```

### 3.3 迁移对比文档
```
旧接口: POST /v1/chat
新接口: POST /v2/chat/completions

参数对比:
- message → messages (数组格式)
- 新增: model, temperature
- 移除: legacy_param

响应对比:
- response.text → response.choices[0].message.content
```

## 四、过渡期

### 4.1 过渡期时长
| 稳定性级别 | 最低过渡期 |
|------------|------------|
| stable | 12 个月 |
| beta | 6 个月 |
| experimental | 1 个月 |

### 4.2 过渡期行为
| 阶段 | 行为 |
|------|------|
| 公告后 | 正常服务 + 警告头 |
| 过渡中期 | 正常服务 + 强警告 |
| 过渡末期 | 限流 + 强警告 |
| 下线前 | 只读 + 最终警告 |

### 4.3 警告响应头
```http
X-API-Deprecated: true
X-API-Sunset: 2027-04-07T00:00:00Z
X-API-Replacement: /v2/chat/completions
X-API-Migration-Guide: https://docs.openclaw.ai/migration/v1-to-v2
```

## 六、租户豁免

### 6.1 豁免条件
| 条件 | 说明 |
|------|------|
| 企业合同 | 合同中有特殊条款 |
| 迁移困难 | 技术迁移确实困难 |

```
提交申请 → 评估审核 → 批准/拒绝 → 设置豁免期限
```

## 七、强制关停条件

### 7.1 强制关停触发
| 条件 | 说明 |
|------|------|
| 安全漏洞 | 严重安全漏洞 |
| 合规违规 | 严重合规违规 |
| 系统风险 | 影响系统稳定 |
| 过渡期结束 | 过渡期已结束 |

### 7.2 强制关停流程
```
触发条件 → 紧急评估 → 通知用户 → 执行关停 → 后续处理
   ↓           ↓           ↓           ↓           ↓
 即时        24h内       48h内       72h内       7天内
```

### 7.3 强制关停通知
```
紧急通知 → 邮件 + 短信 + 控制台
说明原因 → 提供替代方案
后续支持 → 技术支持通道
```

---

## 八、下线后处理

### 8.1 下线后行为
| 行为 | 说明 |
|------|------|
| 返回错误 | 返回 410 Gone |
| 错误信息 | 说明已下线 + 替代接口 |
| 文档保留 | 保留历史文档 |
| 日志保留 | 保留访问日志 |

### 8.2 下线响应
```json
{
  "error": {
    "code": "API_RETIRED",
    "message": "此接口已下线，请使用新接口",
    "details": {
      "retired_at": "2027-04-07T00:00:00Z",
      "replacement": "/v2/chat/completions",
      "migration_guide": "https://docs.openclaw.ai/migration/v1-to-v2"
    }
  }
}
```
