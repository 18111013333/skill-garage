# CONNECTOR_MARKETPLACE_POLICY.md - 连接器市场规则

## 目的
定义连接器市场规则，规范连接器分发。

## 适用范围
所有连接器市场的连接器。

---

## 一、连接器可见性

### 1.1 可见性分类
| 分类 | 说明 | 适用场景 |
|------|------|----------|
| public | 公开可见 | 所有租户可安装 |
| enterprise | 企业可见 | 仅企业版可用 |
| private | 私有可见 | 仅特定租户可用 |
| hidden | 隐藏 | 不可搜索 |

### 1.2 可见性规则
| 连接器类型 | 默认可见性 |
|------------|------------|
| 官方连接器 | public |
| 认证连接器 | public |
| 企业连接器 | enterprise |
| 私有连接器 | private |

---

## 二、上架审批

### 2.1 上架要求
| 要求 | 说明 |
|------|------|
| 认证通过 | 通过连接器认证 |
| 文档完整 | 有完整文档 |
| 测试通过 | 通过功能测试 |
| 安全审核 | 通过安全审核 |

### 2.2 上架流程
```
提交上架 → 自动检查 → 人工审核 → 批准上架 → 发布市场
    ↓           ↓           ↓           ↓           ↓
  开发者      系统        审核        审核        系统
```

### 2.3 上架时间
| 连接器类型 | 审核时间 |
|------------|----------|
| 低风险 | 3-5 天 |
| 中风险 | 5-7 天 |
| 高风险 | 7-14 天 |

---

## 三、下架规则

### 3.1 下架触发
| 触发条件 | 说明 |
|----------|------|
| 安全问题 | 发现安全漏洞 |
| 合规问题 | 合规违规 |
| 质量问题 | 质量不达标 |
| 作者请求 | 作者主动下架 |
| 长期不维护 | 超过 1 年未更新 |
| 外部系统下线 | 外部系统不可用 |

### 3.2 下架流程
```
触发下架 → 评估影响 → 决定下架 → 执行下架 → 通知用户
    ↓           ↓           ↓           ↓           ↓
  系统        产品        产品        系统        系统
```

### 3.3 下架后处理
- 已安装用户可继续使用
- 新用户无法安装
- 提供迁移建议

## 五、区域限制

### 5.1 区域支持
| 区域 | 说明 |
|------|------|
| global | 全球可用 |
| cn | 仅中国可用 |
| us | 仅美国可用 |
| eu | 仅欧洲可用 |

### 5.2 区域配置
```json
{
  "connector_id": "conn_xxx",
  "supported_regions": ["cn", "us"],
  "restricted_regions": ["eu"]
}
```

### 5.3 区域检查
```
租户区域 → 连接器支持区域 → 匹配 → 显示/隐藏
```
