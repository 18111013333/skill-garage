# REVENUE_SHARE_POLICY.md - 终极鸽子王渠道分成规则

## 目的
定义渠道销售、服务、市场资产的分成规则。

## 适用范围
所有渠道伙伴的收入分成计算与结算。

---

## 一、分成类型

### 1.1 首单分成
| 伙伴等级 | 新客户分成 | 说明 |
|----------|------------|------|
| 注册级 | 10% | 基础分成 |
| 认证级 | 15% | 标准分成 |
| 高级级 | 20% | 优厚分成 |
| 战略级 | 25% | 最高分成 |

### 1.2 续费分成
| 伙伴等级 | 续费分成 | 说明 |
|----------|----------|------|
| 注册级 | 5% | 基础分成 |
| 认证级 | 8% | 标准分成 |
| 高级级 | 10% | 优厚分成 |
| 战略级 | 12% | 最高分成 |

### 1.3 服务分成
| 服务类型 | 伙伴等级 | 分成比例 |
|----------|----------|----------|
| 实施服务 | 认证级+ | 10-20% |
| 支持服务 | 认证级+ | 10-15% |
| 培训服务 | 认证级+ | 10-15% |

### 1.4 市场资产分成
| 资产类型 | 开发者分成 | 平台分成 |
|----------|------------|----------|
| 标准资产 | 70% | 30% |
| 高级资产 | 75% | 25% |
| 认证资产 | 80% | 20% |

---

## 二、退款/争议处理

### 2.1 退款扣除
| 场景 | 处理方式 |
|------|----------|
| 客户退款 | 扣除已结算分成 |
| 部分退款 | 按比例扣除分成 |
| 服务退款 | 扣除服务分成 |

### 2.2 争议处理
| 争议类型 | 处理方式 |
|----------|----------|
| 归属争议 | 暂停分成，争议解决后处理 |
| 金额争议 | 按确认金额结算 |
| 服务争议 | 协商或仲裁解决 |

---

## 三、违规扣减

### 3.1 违规类型
| 违规类型 | 扣减比例 | 说明 |
|----------|----------|------|
| 服务质量不达标 | 10-30% | 根据严重程度 |
| 客户投诉 | 5-20% | 根据投诉严重程度 |
| 违规操作 | 20-50% | 根据违规严重程度 |
| 严重违规 | 100% | 取消全部分成 |

### 3.2 扣减流程
```
违规扣减流程
    ↓
┌─────────────────────────────────────┐
│ 1. 违规认定                          │
│    - 调查核实                        │
│    - 确定违规等级                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 扣减决定                          │
│    - 确定扣减比例                    │
│    - 通知伙伴                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 扣减执行                          │
│    - 从待结算扣除                    │
│    - 记录扣减原因                    │
└─────────────────────────────────────┘
```

## 四、分成计算示例

### 4.1 计算公式
```
伙伴分成 = 销售金额 × 分成比例 × 等级系数 - 扣减金额
```
