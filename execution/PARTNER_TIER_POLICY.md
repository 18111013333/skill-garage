# PARTNER_TIER_POLICY.md - 代理商等级规则

## 目的
定义代理商/服务商的等级体系与权益边界。

---

## 等级体系

### 注册级 (Registered)
| 维度 | 权限 |
|------|------|
| **可售范围** | 标准产品，无折扣 |
| **可实施范围** | 无 |
| **可支持范围** | 基础问题转达 |
| **可访问资源** | 公开文档、基础培训 |
| **分成比例** | 首单10%，续费5% |

### 认证级 (Certified)
| 维度 | 权限 |
|------|------|
| **可售范围** | 标准产品，标准折扣 |
| **可实施范围** | 标准实施包 |
| **可支持范围** | 一线问题处理 |
| **可访问资源** | 认证培训、实施工具 |
| **分成比例** | 首单20%，续费10%，服务30% |

### 高级级 (Advanced)
| 维度 | 权限 |
|------|------|
| **可售范围** | 全产品，灵活折扣 |
| **可实施范围** | 标准+定制实施 |
| **可支持范围** | 全级别支持 |
| **可访问资源** | 高级培训、定制工具 |
| **分成比例** | 首单30%，续费15%，服务40% |

### 战略级 (Strategic)
| 维度 | 权限 |
|------|------|
| **可售范围** | 全产品，自主定价 |
| **可实施范围** | 全权实施+质量审核 |
| **可支持范围** | 全权支持+SLA管理 |
| **可访问资源** | 全部资源+联合研发 |
| **分成比例** | 首单40%，续费20%，服务50% |

---

## 等级权益对比

| 权益 | 注册级 | 认证级 | 高级级 | 战略级 |
|------|--------|--------|--------|--------|
| 销售权限 | 推荐 | 独立销售 | 灵活定价 | 自主定价 |
| 实施权限 | 无 | 标准实施 | 定制实施 | 全权实施 |
| 支持权限 | 转达 | 一线支持 | 全级别 | 全权+SLA |
| 培训资源 | 基础 | 认证 | 高级 | 全部 |
| 技术支持 | 无 | 标准响应 | 优先响应 | 专属团队 |
| 市场支持 | 无 | 基础物料 | 定制物料 | 联合营销 |
| 财务支持 | 标准结算 | 月度结算 | 周期灵活 | 定制结算 |

---

## 等级升级路径

```
注册级
    ↓ 完成认证考试 + 1个成功案例
认证级
    ↓ 高级认证 + 3个成功案例 + 业绩达标
高级级
    ↓ 战略认证 + 10个成功案例 + 业绩超标
战略级
```

---

## 等级降级规则

| 触发条件 | 降级处理 |
|----------|----------|
| 年度业绩未达标 | 降一级 |
| 客户投诉≥3次 | 暂停资格 |
| 认证过期未续 | 降至注册级 |
| 违规操作 | 吊销资格 |
