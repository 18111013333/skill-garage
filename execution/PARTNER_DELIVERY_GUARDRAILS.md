# PARTNER_DELIVERY_GUARDRAILS.md - 终极鸽子王伙伴实施边界

## 目的
定义伙伴实施时的权限边界，防止越权操作。

## 适用范围
所有伙伴为客户实施交付时的操作边界。

---

## 一、伙伴可做配置

### 1.1 允许配置项
| 配置类型 | 说明 | 审批要求 |
|----------|------|----------|
| 业务配置 | 业务规则配置 | 无需审批 |
| 界面配置 | 界面样式配置 | 无需审批 |
| 通知配置 | 通知模板配置 | 无需审批 |
| 工作流配置 | 审批流程配置 | 无需审批 |
| 角色配置 | 角色权限配置 | 租户Owner审批 |

### 1.2 配置限制
| 限制项 | 说明 |
|--------|------|
| 配置范围 | 仅限授权租户范围 |
| 配置备份 | 配置变更需备份 |
| 配置审计 | 所有变更需审计 |

---

## 二、需厂商批准操作

### 2.1 高风险操作
| 操作类型 | 说明 | 审批人 |
|----------|------|--------|
| 系统配置变更 | 影响系统行为的配置 | 平台技术团队 |
| 安全配置变更 | 安全相关配置 | 平台安全团队 |
| 集成配置变更 | 外部集成配置 | 平台集成团队 |
| 数据迁移 | 大规模数据迁移 | 平台数据团队 |

### 2.2 审批流程
```
高风险操作审批
    ↓
┌─────────────────────────────────────┐
│ 1. 操作申请                          │
│    - 说明操作内容                    │
│    - 说明操作原因                    │
│    - 提供风险评估                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 平台审核                          │
│    - 技术可行性审核                  │
│    - 安全风险评估                    │
│    - 影响范围评估                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 审批决定                          │
│    - 批准/拒绝/有条件批准            │
│    - 说明审批意见                    │
└─────────────────────────────────────┘
```

---

## 三、需Tenant Owner批准操作

### 3.1 租户级操作
| 操作类型 | 说明 | 审批人 |
|----------|------|--------|
| 角色权限变更 | 角色权限调整 | Tenant Owner |
| 用户管理 | 用户创建/删除 | Tenant Owner |
| 数据导出 | 数据批量导出 | Tenant Owner |
| 高风险功能开启 | 高风险功能启用 | Tenant Owner |

### 3.2 审批要求
| 要求项 | 说明 |
|--------|------|
| 书面申请 | 需提交书面申请 |
| 审批记录 | 需保留审批记录 |
| 操作日志 | 需记录操作日志 |

---

## 四、禁止访问对象

### 4.1 禁止直接访问
| 对象类型 | 说明 |
|----------|------|
| 平台核心配置 | 平台级系统配置 |
| 其他租户数据 | 非授权租户数据 |
| 系统日志 | 平台系统日志 |
| 安全凭据 | 密钥/证书等 |

### 4.2 访问违规处理
| 违规类型 | 处理 |
|----------|------|
| 无意访问 | 警告+培训 |
| 有意访问 | 暂停权限+调查 |
| 恶意访问 | 终止合作+法律追究 |

---

## 五、审计要求

### 5.1 审计内容
| 审计项 | 记录内容 | 保留期限 |
|--------|----------|----------|
| 配置变更 | 变更内容、时间、操作人 | 1年 |
| 数据访问 | 访问内容、时间、操作人 | 1年 |
| 审批记录 | 审批内容、审批人、结果 | 永久 |
| 异常操作 | 异常类型、处理结果 | 1年 |

### 5.2 审计检查
| 检查项 | 频率 | 说明 |
|--------|------|------|
| 权限检查 | 每周 | 验证权限合规 |
| 操作审计 | 每月 | 审查操作记录 |
| 异常检测 | 实时 | 检测异常操作 |
