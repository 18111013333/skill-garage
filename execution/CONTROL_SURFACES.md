# CONTROL_SURFACES.md - 控制面定义

## 目的
定义控制面，明确哪些功能属于展示、哪些属于控制。

## 适用范围
平台控制台的所有设置、配置、操作功能。

## 控制面分类

### 展示类 (Display Only)
| 类型 | 说明 | 示例 |
|------|------|------|
| 状态展示 | 系统和服务状态 | 服务健康状态 |
| 指标展示 | 运行指标和统计 | 使用量统计 |
| 日志展示 | 日志和记录查看 | 审计日志查看 |
| 报表展示 | 报表和分析结果 | 运营报表 |
| 配置展示 | 配置信息查看 | 当前配置查看 |

### 控制类 (Controllable)
| 类型 | 说明 | 示例 |
|------|------|------|
| 配置修改 | 可修改的配置 | 功能开关设置 |
| 操作执行 | 可执行的操作 | 服务重启 |
| 权限管理 | 权限相关操作 | 用户权限分配 |
| 资源管理 | 资源相关操作 | 配额调整 |

## 控制面矩阵

### 按修改方式
| 控制面 | 控制台修改 | 变更控制 | 只读展示 |
|--------|------------|----------|----------|
| 功能开关 | ✅ 低风险 | ✅ 高风险 | - |
| 配额设置 | ✅ 范围内 | ✅ 超范围 | - |
| 权限配置 | ✅ 标准权限 | ✅ 高级权限 | - |
| 系统参数 | ❌ | ✅ | ✅ |
| 安全策略 | ❌ | ✅ | ✅ |
| 合规规则 | ❌ | ✅ | ✅ |
| 审计日志 | ❌ | ❌ | ✅ |
| 运行状态 | ❌ | ❌ | ✅ |

### 按风险等级
| 风险等级 | 控制台修改 | 变更控制 | 只读 |
|----------|------------|----------|------|
| 低风险 | ✅ | - | - |
| 中风险 | ✅ + 确认 | ✅ | - |
| 高风险 | ❌ | ✅ | ✅ |
| 极高风险 | ❌ | ✅ + 审批 | ✅ |

## 可通过控制台修改的设置

### 低风险设置
| 设置 | 说明 | 修改方式 |
|------|------|----------|
| 个人偏好 | 用户个人设置 | 直接修改 |
| 通知设置 | 通知偏好设置 | 直接修改 |
| 界面配置 | 界面显示配置 | 直接修改 |
| 快捷方式 | 快捷方式配置 | 直接修改 |

### 中风险设置
| 设置 | 说明 | 修改方式 |
|------|------|----------|
| 功能开关 | 功能启用/禁用 | 二次确认 |
| 配额调整 | 范围内配额调整 | 二次确认 |
| 成员管理 | 团队成员管理 | 二次确认 |
| 集成配置 | 外部集成配置 | 二次确认 |

### 高风险设置
| 设置 | 说明 | 修改方式 |
|------|------|----------|
| 安全策略 | 安全相关策略 | 变更控制 |
| 合规规则 | 合规相关规则 | 变更控制 |
| 系统参数 | 核心系统参数 | 变更控制 |
| 权限策略 | 权限相关策略 | 变更控制 |

## 必须走变更控制的设置

### 变更控制要求
| 设置类型 | 变更控制流程 | 审批要求 |
|----------|--------------|----------|
| 安全策略 | 标准变更流程 | 安全审批 |
| 合规规则 | 标准变更流程 | 合规审批 |
| 系统参数 | 标准变更流程 | 技术审批 |
| 权限策略 | 标准变更流程 | 安全审批 |
| 架构配置 | 标准变更流程 | 架构审批 |

### 变更控制流程
```
变更申请
    ↓
影响评估
    ↓
沙盘验证（如需）
    ↓
审批流程
    ↓
变更执行
    ↓
效果验证
    ↓
变更关闭
```

## 需要二次确认的操作

### 确认级别
| 级别 | 确认方式 | 适用操作 |
|------|----------|----------|
| 简单确认 | 弹窗确认 | 一般设置修改 |
| 输入确认 | 输入特定文字 | 重要设置修改 |
| 密码确认 | 输入密码 | 敏感操作 |
| 双人确认 | 两人确认 | 高风险操作 |

### 需二次确认的操作
| 操作 | 确认方式 | 说明 |
|------|----------|------|
| 删除数据 | 输入 "DELETE" | 防误删 |
| 修改权限 | 密码确认 | 权限安全 |
| 重启服务 | 输入 "RESTART" | 防误操作 |
| 修改配额 | 弹窗确认 | 成本控制 |
| 禁用功能 | 弹窗确认 | 影响确认 |

## 必须记录审计日志的操作

### 审计级别
| 级别 | 记录内容 | 适用操作 |
|------|----------|----------|
| 基础审计 | 操作类型 + 时间 + 操作人 | 一般操作 |
| 标准审计 | + 操作详情 + 结果 | 重要操作 |
| 强审计 | + 前后差异 + 完整上下文 | 敏感操作 |
| 最强审计 | + 审批链 + 验证记录 | 高风险操作 |

### 必须审计的操作
| 操作 | 审计级别 | 说明 |
|------|----------|------|
| 配置修改 | 标准审计 | 记录修改内容 |
| 权限变更 | 强审计 | 记录前后差异 |
| 数据删除 | 最强审计 | 记录完整信息 |
| 服务操作 | 标准审计 | 记录操作详情 |
| 配额调整 | 强审计 | 记录调整原因 |
| 功能开关 | 标准审计 | 记录开关状态 |

## 控制面权限

### 按角色
| 角色 | 展示权限 | 控制权限 |
|------|----------|----------|
| 普通用户 | 授权范围展示 | 个人设置控制 |
| 租户管理员 | 租户范围展示 | 租户范围控制 |
| 运营人员 | 运营范围展示 | 运营范围控制 |
| 平台管理员 | 全平台展示 | 全平台控制 |
| 审计人员 | 审计范围展示 | 无控制权限 |
