# DEGRADATION_RULES.md - 降级规则

## 目的
定义系统降级规则，确保故障时系统仍能提供服务。

## 适用范围
系统异常时的降级处理。

## 降级级别

### 级别定义
| 级别 | 说明 | 功能限制 |
|------|------|----------|
| L0 | 正常 | 全部功能 |
| L1 | 轻度降级 | 禁用非核心技能 |
| L2 | 中度降级 | 仅核心技能 |
| L3 | 重度降级 | 仅基础功能 |
| L4 | 安全模式 | 最小功能集 |

## 降级触发

### 触发条件
| 条件 | 降级级别 |
|------|----------|
| 连续失败 ≥ 3 | L1 |
| 错误率 > 30% | L2 |
| 错误率 > 50% | L3 |
| 系统错误 | L3 |
| 安全问题 | L4 |

### 恢复条件
| 条件 | 恢复级别 |
|------|----------|
| 连续成功 ≥ 5 | 升一级 |
| 错误率 < 10% | 升一级 |
| 用户确认 | 恢复L0 |

## 降级策略

### L1: 轻度降级
| 限制 | 说明 |
|------|------|
| 禁用P3技能 | 仅保留P0-P2 |
| 禁用并行 | 顺序执行 |
| 重试减少 | 最多1次 |

### L2: 中度降级
| 限制 | 说明 |
|------|------|
| 仅P0-P1 | 核心技能 |
| 禁用外部检索 | 仅内部记忆 |
| 简化输出 | 基础格式 |

### L3: 重度降级
| 限制 | 说明 |
|------|------|
| 仅P0 | 核心技能 |
| 禁用工具 | 仅直接回答 |
| 强制确认 | 需用户确认 |

### L4: 安全模式
| 限制 | 说明 |
|------|------|
| 最小功能 | 仅基础问答 |
| 禁用所有工具 | 无工具调用 |
| 强制审计 | 记录所有操作 |

## 降级流程

```
检测到异常
    ↓
┌─────────────────────────────────────┐
│ 1. 评估异常程度                      │
│    - 错误率                          │
│    - 连续失败次数                    │
│    - 异常类型                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 确定降级级别                      │
│    - 匹配触发条件                    │
│    - 选择降级策略                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 执行降级                          │
│    - 应用限制                        │
│    - 通知用户                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 监控恢复                          │
│    - 持续监控                        │
│    - 满足条件后恢复                  │
└─────────────────────────────────────┘
```

## 降级报告

### 报告格式
```json
{
  "degradationId": "deg_20260406_001",
  "timestamp": "2026-04-06T12:13:00+08:00",
  "trigger": {
    "condition": "consecutive_failures >= 3",
    "value": 3
  },
  "fromLevel": "L0",
  "toLevel": "L1",
  "restrictions": [
    "disable_p3_skills",
    "disable_parallel",
    "reduce_retries"
  ],
  "recoveryCondition": "consecutive_success >= 5"
}
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 降级失败 | 进入L4 |
| 恢复失败 | 保持当前级别 |
| 级别冲突 | 使用更高级别 |
