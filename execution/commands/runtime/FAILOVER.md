# FAILOVER.md - 故障恢复策略

## 目的
定义工具失败、检索为空、技能失败、格式不符、上下文冲突时的恢复策略。

## 适用范围
所有执行过程中的故障和异常情况。

## 故障类型与恢复策略

### 工具失败
| 故障类型 | 恢复策略 | 说明 |
|----------|----------|------|
| 工具不存在 | 换备选工具 | 从 SKILL_ROUTER.json 获取备选 |
| 参数无效 | 修正参数重试 | 根据错误信息修正 |
| 调用超时 | 重试或降级 | 按重试策略执行 |
| 权限不足 | 请求授权或跳过 | 根据任务重要性决定 |
| 返回异常 | 解析或降级 | 尝试解析部分结果 |

### 检索为空
| 故障类型 | 恢复策略 | 说明 |
|----------|----------|------|
| 内部记忆为空 | 扩大搜索范围 | 放宽搜索条件 |
| 外部搜索为空 | 换搜索引擎 | 尝试备选搜索技能 |
| 文件搜索为空 | 确认路径 | 请求用户确认路径 |
| 全部为空 | 标注不确定性 | 明确说明信息不足 |

### 技能失败
| 故障类型 | 恢复策略 | 说明 |
|----------|----------|------|
| 技能不存在 | 换备选技能 | 从路由表获取备选 |
| 技能加载失败 | 重试或降级 | 最多重试2次 |
| 技能执行失败 | 换备选或降级 | 记录失败原因 |
| 技能超时 | 终止或降级 | 返回部分结果 |

### 格式不符
| 故障类型 | 恢复策略 | 说明 |
|----------|----------|------|
| 输出格式错误 | 格式修正 | 调用格式修正逻辑 |
| JSON解析失败 | 修复或降级 | 尝试修复JSON |
| Markdown格式错误 | 格式修正 | 自动修正格式 |
| 编码错误 | 转换编码 | 转为UTF-8 |

### 上下文冲突
| 故障类型 | 恢复策略 | 说明 |
|----------|----------|------|
| 信息冲突 | 标记冲突 | 显式标记冲突来源 |
| 规则冲突 | 按优先级处理 | 参考 POLICY_PRECEDENCE.md |
| 记忆冲突 | 按冲突规则处理 | 参考 MEMORY_CONFLICT_RULES.md |
| 用户约束冲突 | 请求用户澄清 | 询问用户选择 |

## 恢复流程

### 标准恢复流程
```
故障发生
    ↓
┌─────────────────────────────────────┐
│ 1. 识别故障类型                      │
│    - 分类故障                        │
│    - 评估严重程度                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 选择恢复策略                      │
│    - 查找对应恢复策略                │
│    - 评估恢复可行性                  │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 执行恢复                          │
│    - 执行恢复动作                    │
│    - 记录恢复日志                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 验证恢复结果                      │
│    - 检查是否恢复正常                │
│    - 决定是否继续或终止              │
└─────────────────────────────────────┘
    ↓
继续执行 / 终止解释
```

### 终止解释格式
```
[TERMINATED]
原因: {终止原因}
已尝试恢复: {恢复尝试列表}
部分结果: {已完成部分}
建议: {用户可选操作}
```

## 降级策略

### 降级级别
| 级别 | 说明 | 允许操作 |
|------|------|----------|
| L1 | 轻度降级 | 禁用非核心技能 |
| L2 | 中度降级 | 仅允许核心技能 |
| L3 | 重度降级 | 仅允许低风险操作 |
| L4 | 安全模式 | 仅允许只读操作 |

### 降级触发条件
| 条件 | 降级级别 |
|------|----------|
| 单次失败 | 不降级 |
| 连续失败2次 | L1 |
| 连续失败3次 | L2 |
| 错误率>30% | L3 |
| 错误率>50% | L4 |

```json
  },

```

## 异常处理

| 异常 | 处理 |
|------|------|
| 恢复策略不存在 | 默认终止解释 |
| 恢复失败 | 尝试下一策略或终止 |
| 降级失败 | 进入安全模式 |

## 维护方式
- 新增故障类型: 添加到故障类型表
- 新增恢复策略: 添加到恢复策略表
- 调整降级条件: 更新降级触发条件
