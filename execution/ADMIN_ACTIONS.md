# ADMIN_ACTIONS.md - 控制台管理员动作集合

## 目的
定义控制台管理员动作边界，确保管理员操作可控可审计。

## 适用范围
平台管理员在控制台内的所有操作。

## 动作执行流程

### 低风险动作
```
执行动作 → 记录日志 → 完成
```

### 中风险动作
```
执行动作 → 二次确认 → 记录日志 → 完成
```

### 高风险动作
```
执行动作 → 审批流程 → 执行 → 记录日志 → 通知相关方 → 完成
```

### 极高风险动作
```
执行动作 → 多级审批 → 沙盘验证 → 执行 → 强审计 → 通知相关方 → 复核确认 → 完成
```

## 动作限制

### 频率限制
| 动作类型 | 限制 | 说明 |
|----------|------|------|
| 查看操作 | 无限制 | 只读操作 |
| 配置修改 | 10 次/分钟 | 防误操作 |
| 批量操作 | 5 次/小时 | 防批量风险 |
| 高风险操作 | 3 次/小时 | 严格控制 |

### 时间限制
| 动作类型 | 时间限制 | 说明 |
|----------|----------|------|
| 敏感操作 | 工作时间 | 需人工监督 |
| 批量操作 | 工作时间 | 需人工监督 |
| 紧急操作 | 无限制 | 紧急情况 |

### 范围限制
| 动作类型 | 范围限制 | 说明 |
|----------|----------|------|
| 租户操作 | 授权租户 | 只能操作授权租户 |
| 配额调整 | 授权范围 | 只能在授权范围调整 |
| 功能开关 | 授权功能 | 只能操作授权功能 |

## 紧急动作

### 紧急动作定义
| 动作 | 说明 | 紧急条件 |
|------|------|----------|
| 紧急禁用租户 | 立即禁用租户 | 安全事件 |
| 紧急回滚 | 立即回滚发布 | Sev1 事故 |
| 紧急禁用功能 | 立即禁用功能 | 安全风险 |
| 紧急收紧自治 | 立即收紧自治 | 资源滥用 |

### 紧急动作流程
```
紧急情况确认
    ↓
执行紧急动作
    ↓
事后补审批
    ↓
记录完整信息
    ↓
事后复盘
```
