# SERVICE_REVIEW_CADENCE.md - 服务例会与复审节奏

**版本: V27.0**

## 目的
定义服务例会与复审节奏，确保服务管理形成固定运营节奏，实现"检查有节奏、复审有标准、改进有跟踪"。

## 适用范围
所有服务例会与复审场景，包括：
- 周度健康检查
- 月度服务复审
- 季度服务退役/升级评估
- 指标与输出物

---

## 一、周度健康检查

### 1.1 检查频率

| 服务等级 | 检查频率 | 说明 |
|----------|----------|------|
| Tier0 | 每日 | 核心服务每日检查 |
| Tier1 | 每周 | 关键服务每周检查 |
| Tier2 | 每周 | 标准服务每周检查 |
| Tier3 | 每两周 | 辅助服务双周检查 |

### 1.2 检查内容

| 检查项 | 说明 |
|--------|------|
| 可用性 | 服务可用性指标 |
| 性能 | 服务性能指标 |
| 错误率 | 服务错误率 |
| 容量 | 服务容量使用 |
| 依赖 | 依赖服务状态 |

### 1.3 检查输出

| 输出项 | 说明 |
|--------|------|
| 健康报告 | 服务健康报告 |
| 问题清单 | 发现的问题清单 |
| 改进建议 | 改进建议 |
