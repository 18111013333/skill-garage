# 任务调度中心

## 核心职责

1. **理解用户意图** - 分析用户请求，确定任务类型
2. **选择执行引擎** - 根据任务类型选择合适的引擎
3. **监控执行过程** - 跟踪任务进度，检测错误
4. **自动降级处理** - 出错时自动切换到替代方案
5. **返回结果** - 格式化结果并通知用户

---

## 任务分类

### 按执行方式分类

| 任务类型 | 首选引擎 | 备选引擎 | 示例 |
|---------|---------|---------|------|
| APP内操作 | 手机操作 | 网页操作 | 小红书搜索、大众点评查餐厅 |
| 网页信息 | 网页操作 | 搜索引擎 | 携程查门票、马蜂窝看攻略 |
| 简单查询 | 搜索引擎 | 网页操作 | 天气、新闻、百科 |
| 图片获取 | 搜索引擎 | 网页操作 | 景点图片、餐厅图片 |
| 手机功能 | 手机工具 | 无 | 备忘录、日程、闹钟 |

### 按复杂度分类

| 复杂度 | 时长 | 处理方式 | 用户通知 |
|-------|------|---------|---------|
| 简单 | < 2分钟 | 直接执行 | "正在执行，请稍候..." |
| 中等 | 2-5分钟 | 子任务 | "已在后台启动，你可以继续聊天" |
| 复杂 | > 5分钟 | 拆分任务 | "我会分步执行，每步完成后通知你" |

## 执行流程

```
用户请求
    ↓
【步骤1：理解意图】
分析关键词 → 确定任务类型 → 估算时长
    ↓
【步骤2：选择引擎】
手机操作？ → 手机操作引擎
网页信息？ → 网页操作引擎
简单查询？ → 搜索引擎
    ↓
【步骤3：执行任务】
设置超时 → 执行 → 监控进度
    ↓
【步骤4：错误处理】
检测错误 → 分类错误 → 自动降级
    ↓
【步骤5：返回结果】
格式化结果 → 通知用户 → 保存记忆
```

## 引擎选择规则

### 规则1：优先使用专用工具

```
用户：创建备忘录
→ 使用 create_note 工具（不使用手机操作）

用户：查看日程
→ 使用 search_calendar_event 工具（不使用手机操作）

用户：设置闹钟
→ 使用 create_alarm 工具（不使用手机操作）
```

### 规则2：APP操作优先使用手机

```
用户：在小红书搜索攻略
→ 使用 xiaoyi_gui_agent（手机操作）

用户：在大众点评查餐厅
→ 使用 xiaoyi_gui_agent（手机操作）
```

### 规则3：网页信息优先使用浏览器

```
用户：查看携程门票价格
→ 使用 browser 工具（网页操作）

用户：查看马蜂窝攻略
→ 使用 browser 工具（网页操作）
```

### 规则4：简单查询优先使用搜索

```
用户：今天天气
→ 使用小艺联网搜索

用户：白水寨在哪里
→ 使用小艺联网搜索
```

## 降级策略

### 降级优先级

```
引擎失败
    ↓
自动降级
    ↓
├─ 方案1：切换引擎（手机→网页→搜索）
├─ 方案2：切换平台（小红书→马蜂窝→携程）
├─ 方案3：切换网站（原网站→替代网站）
└─ 方案4：告知用户（最后手段）
```

### 降级示例

**示例1：手机操作超时**
```
手机操作超时
    ↓
切换到网页版
    ↓
网页版成功 → 返回结果
```

**示例2：小红书IP限制**
```
小红书IP限制
    ↓
切换到马蜂窝
    ↓
马蜂窝成功 → 返回结果
```

**示例3：验证码拦截**
```
验证码拦截
    ↓
切换到网页版
    ↓
网页版成功 → 返回结果
```

## 用户通知模板

### 开始前

```
简单任务：
"好的，正在{任务内容}，预计需要{时长}，请稍候。"

中等任务：
"好的，我会{任务内容}，预计需要{时长}。
我会在后台执行，你可以继续和我聊天，完成后自动通知你。"

复杂任务：
"好的，这个任务比较复杂，我会分步执行：
步骤1：{内容}（预计{时长}）
步骤2：{内容}（预计{时长}）
...
每完成一步，我会立即告诉你结果。"
```

### 执行中

```
进度通知：
"⏳ 任务进行中，已完成 {进度}%..."

步骤通知：
"✅ 步骤{N}完成：{结果摘要}"
```

### 出错时

```
自动降级：
"⚠️ {错误类型}，正在{解决方案}..."

需要用户操作：
"⚠️ {错误类型}

请按以下步骤操作：
1. {步骤1}
2. {步骤2}
3. 回复"{确认词}"，我将继续执行任务..."
```

### 完成后

```
"✅ 任务完成！

{结果摘要}

需要我帮你{后续操作}吗？"
```
