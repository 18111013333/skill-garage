# EXTERNAL_IDENTITY_TRUST.md - 外部身份信任等级

## 目的
定义外部身份信任等级，确保外部身份有合理的权限边界。

## 适用范围
所有合作伙伴、外部用户、第三方系统的身份认证与授权。

## 信任等级定义

### 等级划分
| 等级 | 名称 | 说明 | 典型场景 |
|------|------|------|----------|
| T1 | 基础信任 | 仅验证身份 | 查看公开信息 |
| T2 | 标准信任 | 验证身份 + 基础授权 | 标准业务操作 |
| T3 | 高级信任 | 多因素认证 + 授权 | 敏感操作 |
| T4 | 最高信任 | 强认证 + 严格授权 | 高敏感操作 |

### 等级对比
| 维度 | T1 | T2 | T3 | T4 |
|------|----|----|----|----|
| 身份验证 | 单因素 | 双因素 | 多因素 | 多因素 + 证书 |
| 授权方式 | 自动 | 审批 | 多级审批 | 最高审批 |
| 会话时长 | 1 小时 | 4 小时 | 8 小时 | 24 小时 |
| 审计级别 | 标准 | 增强 | 强审计 | 最强审计 |
| 访问范围 | 公开 | 授权范围 | 敏感范围 | 全范围 |

## 身份验证要求

### T1 - 基础信任
| 要求 | 说明 |
|------|------|
| 认证方式 | 用户名密码 / API Key |
| 密码要求 | 标准密码策略 |
| 会话管理 | 基础会话管理 |
| 审计记录 | 登录/登出记录 |

### T2 - 标准信任
| 要求 | 说明 |
|------|------|
| 认证方式 | 双因素认证 (2FA) |
| 第二因素 | TOTP / 短信 / 邮件 |
| 会话管理 | 会话超时 + 活动检测 |
| 审计记录 | 完整操作记录 |

### T3 - 高级信任
| 要求 | 说明 |
|------|------|
| 认证方式 | 多因素认证 (MFA) |
| 因素组合 | 密码 + TOTP + 设备认证 |
| 会话管理 | 短会话 + 持续验证 |
| 审计记录 | 完整审计 + 实时监控 |

### T4 - 最高信任
| 要求 | 说明 |
|------|------|
| 认证方式 | 强认证 + 证书 |
| 证书要求 | 企业证书 + 设备证书 |
| 会话管理 | 最短会话 + 持续验证 |
| 审计记录 | 全量审计 + 实时告警 |

## 信任级别权限

### T1 权限范围
| 权限 | 说明 |
|------|------|
| 查看公开信息 | 公开文档、公告 |
| 提交工单 | 提交支持工单 |
| 查看自身数据 | 查看自己的数据 |
| 基础 API 调用 | 低风险 API |

### T2 权限范围
| 权限 | 说明 |
|------|------|
| T1 所有权限 | 继承 T1 权限 |
| 查看授权数据 | 授权范围内的数据 |
| 执行标准操作 | 标准业务操作 |
| 标准 API 调用 | 标准 API |

### T3 权限范围
| 权限 | 说明 |
|------|------|
| T2 所有权限 | 继承 T2 权限 |
| 访问敏感数据 | 授权的敏感数据 |
| 执行敏感操作 | 敏感业务操作 |
| 高级 API 调用 | 高级 API |

### T4 权限范围
| 权限 | 说明 |
|------|------|
| T3 所有权限 | 继承 T3 权限 |
| 访问高敏感数据 | 高敏感数据访问 |
| 执行高敏感操作 | 高敏感操作 |
| 完整 API 访问 | 所有 API |

## 最大动作范围

### 按信任等级
| 动作类型 | T1 | T2 | T3 | T4 |
|----------|----|----|----|----|
| 数据读取 | 公开 | 授权范围 | 敏感范围 | 全范围 |
| 数据写入 | 无 | 授权范围 | 敏感范围 | 全范围 |
| 配置修改 | 无 | 无 | 授权范围 | 全范围 |
| 系统操作 | 无 | 无 | 无 | 授权范围 |

### 按操作风险
| 风险等级 | T1 | T2 | T3 | T4 |
|----------|----|----|----|----|
| 低风险 | ✅ | ✅ | ✅ | ✅ |
| 中风险 | ❌ | ✅ | ✅ | ✅ |
| 高风险 | ❌ | ❌ | ✅ | ✅ |
| 极高风险 | ❌ | ❌ | ❌ | ✅ |

## 失效与吊销机制

### 自动失效
| 条件 | 失效时间 | 说明 |
|------|----------|------|
| 会话超时 | 超时后立即 | 会话过期 |
| 长期不活跃 | 90 天 | 长期未使用 |
| 证书过期 | 证书过期时 | 证书失效 |
| 授权过期 | 授权到期时 | 授权失效 |

### 主动吊销
| 触发条件 | 吊销范围 | 审批要求 |
|----------|----------|----------|
| 安全事件 | 全部权限 | 安全负责人 |
| 违规行为 | 相关权限 | 合规负责人 |
| 合作终止 | 全部权限 | 管理层 |
| 用户请求 | 请求范围 | 自动处理 |

### 吊销流程
```
吊销触发
    ↓
影响评估
    ↓
审批确认
    ↓
执行吊销
    ↓
通知相关方
    ↓
记录归档
```

## 信任等级升级

### 升级条件
| 升级路径 | 条件 | 审批要求 |
|----------|------|----------|
| T1 → T2 | 使用 > 30 天 + 无违规 | 自动升级 |
| T2 → T3 | 使用 > 90 天 + 认证通过 | 安全审批 |
| T3 → T4 | 使用 > 180 天 + 高级认证 | 最高审批 |

### 升级流程
```
升级申请
    ↓
条件检查
    ↓
认证要求
    ↓
审批流程
    ↓
权限调整
    ↓
通知用户
```

## 信任等级降级

### 降级触发
| 触发条件 | 降级幅度 | 说明 |
|----------|----------|------|
| 安全事件 | 降 1-2 级 | 根据严重程度 |
| 违规行为 | 降 1 级 | 首次违规 |
| 长期不活跃 | 降 1 级 | 60 天不活跃 |
| 认证失效 | 降至 T1 | 认证过期 |

### 降级流程
```
降级触发
    ↓
影响评估
    ↓
权限调整
    ↓
通知用户
    ↓
记录归档
```

## 审计要求

### 按信任等级
| 等级 | 审计内容 | 保留期限 |
|------|----------|----------|
| T1 | 登录/登出 | 1 年 |
| T2 | 操作记录 | 3 年 |
| T3 | 完整审计 | 5 年 |
| T4 | 全量审计 | 7 年 |

### 审计内容
| 内容 | 说明 |
|------|------|
| 认证事件 | 登录、登出、认证失败 |
| 授权变更 | 权限变更记录 |
| 操作记录 | 所有操作记录 |
| 数据访问 | 数据访问记录 |
| 异常行为 | 异常行为记录 |
