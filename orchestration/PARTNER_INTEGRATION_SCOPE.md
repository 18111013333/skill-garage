# PARTNER_INTEGRATION_SCOPE.md - 合作伙伴集成范围

## 目的
定义合作伙伴可使用的集成范围，确保 partner 能接系统但不触碰过深内部边界。

## 适用范围
所有合作伙伴的外部集成、API 访问、数据交互。

## 集成范围分类

### 可开放给 Partner 的集成
| 集成类型 | 开放级别 | 说明 |
|----------|----------|------|
| 状态查询集成 | 完全开放 | 查询服务状态 |
| 监控数据集成 | 只读开放 | 读取监控数据 |
| 报表数据集成 | 只读开放 | 读取报表数据 |
| 告警通知集成 | 接收开放 | 接收告警通知 |
| 工单系统集成 | 授权开放 | 工单交互 |

### 需 Tenant Owner 批准的集成
| 集成类型 | 说明 | 审批要求 |
|----------|------|----------|
| 租户数据集成 | 访问租户数据 | Tenant Owner 批准 |
| 配置管理集成 | 修改租户配置 | Tenant Owner 批准 |
| 任务执行集成 | 执行租户任务 | Tenant Owner 批准 |
| 自动化集成 | 管理自动化任务 | Tenant Owner 批准 |

### 禁止 Partner 访问的集成
| 集成类型 | 原因 | 说明 |
|----------|------|------|
| 系统管理集成 | 平台安全 | 系统级管理 |
| 安全策略集成 | 安全边界 | 安全配置 |
| 合规规则集成 | 合规边界 | 合规配置 |
| 审计系统集成 | 审计完整性 | 审计数据 |
| 计费系统集成 | 财务安全 | 计费数据 |

## 数据字段遮蔽

### 必须遮蔽的字段
| 字段类型 | 遮蔽方式 | 说明 |
|----------|----------|------|
| 用户密码 | 完全遮蔽 | 不传输 |
| API 密钥 | 完全遮蔽 | 不传输 |
| 个人身份信息 | 脱敏处理 | 部分遮蔽 |
| 财务数据 | 脱敏处理 | 部分遮蔽 |
| 安全配置 | 完全遮蔽 | 不传输 |

### 脱敏规则
| 数据类型 | 脱敏方式 | 示例 |
|----------|----------|------|
| 邮箱地址 | 掩码 | a***@b.com |
| 手机号码 | 掩码 | 138****5678 |
| 姓名 | 泛化 | 用户A |
| 金额 | 范围化 | 1000-5000 |
| 时间 | 偏移 | 偏移后时间 |

### 遮蔽配置
```json
{
  "field_masking": {
    "always_mask": [
      "password",
      "api_key",
      "secret_key",
      "token"
    ],
    "desensitize": [
      "email",
      "phone",
      "name",
      "address"
    ],
    "mask_patterns": {
      "email": "a***@b.com",
      "phone": "138****5678",
      "name": "用户*"
    }
  }
}
```

## Approval Gateway

### 必须通过 Approval Gateway 的动作
| 动作类型 | 说明 | Gateway 要求 |
|----------|------|---------------|
| 数据导出 | 导出租户数据 | Tenant Owner 审批 |
| 配置修改 | 修改租户配置 | Tenant Owner 审批 |
| 任务执行 | 执行租户任务 | Tenant Owner 审批 |
| 权限变更 | 变更租户权限 | Tenant Owner + 平台审批 |
| 批量操作 | 批量执行操作 | Tenant Owner + 平台审批 |

### Approval Gateway 流程
```
Partner 发起请求
    ↓
Gateway 拦截
    ↓
路由到审批流程
    ↓
Tenant Owner 审批
    ↓
（如需）平台审批
    ↓
审批通过
    ↓
Gateway 放行
    ↓
执行操作
    ↓
记录审计
```

### Gateway 配置
```json
{
  "approval_gateway": {
    "enabled": true,
    "intercept_actions": [
      "data_export",
      "config_modify",
      "task_execute",
      "permission_change",
      "batch_operation"
    ],
    "approval_routing": {
      "data_export": "tenant_owner",
      "config_modify": "tenant_owner",
      "task_execute": "tenant_owner",
      "permission_change": "tenant_owner,platform",
      "batch_operation": "tenant_owner,platform"
    },
    "timeout_hours": 24,
    "auto_reject_on_timeout": true
  }
}
```

## 集成访问控制

### API 访问控制
| API 类别 | Partner 访问 | 说明 |
|----------|--------------|------|
| 公开 API | ✅ 允许 | 公开信息 API |
| 只读 API | ✅ 授权后允许 | 只读数据 API |
| 操作 API | ⚠️ 需审批 | 操作类 API |
| 管理 API | ❌ 禁止 | 管理类 API |
| 系统 API | ❌ 禁止 | 系统级 API |

### API 限流
| Partner 等级 | 限流配置 | 说明 |
|--------------|----------|------|
| Bronze | 100 次/分钟 | 基础限流 |
| Silver | 300 次/分钟 | 标准限流 |
| Gold | 1000 次/分钟 | 高级限流 |
| Platinum | 3000 次/分钟 | 最高限流 |

## 集成审计

### 审计内容
| 审计项 | 说明 | 保留期限 |
|--------|------|----------|
| API 调用 | 所有 API 调用记录 | 7 年 |
| 数据访问 | 所有数据访问记录 | 7 年 |
| 审批记录 | 所有审批记录 | 7 年 |
| 异常行为 | 异常行为记录 | 7 年 |

### 审计报告
| 报告类型 | 频率 | 接收方 |
|----------|------|--------|
| 调用日报 | 每日 | Partner + 平台 |
| 异常报告 | 实时 | 平台安全团队 |
| 合规报告 | 每月 | 合规团队 |
| 综合报告 | 每季度 | 管理层 |

## 集成监控

### 实时监控
| 监控项 | 告警条件 | 说明 |
|--------|----------|------|
| 调用频率 | > 限流阈值 | 频率异常 |
| 错误率 | > 10% | 错误异常 |
| 敏感数据访问 | 任何访问 | 敏感告警 |
| 越权尝试 | 任何尝试 | 安全告警 |
| 异常时间访问 | 非工作时间 | 行为告警 |

### 定期审查
| 审查项 | 频率 | 审查内容 |
|--------|------|----------|
| 集成权限 | 每月 | 权限是否仍必要 |
| 访问模式 | 每周 | 访问模式是否正常 |
| 审批记录 | 每周 | 审批是否合理 |
| Partner 状态 | 每季度 | 合作关系是否有效 |
