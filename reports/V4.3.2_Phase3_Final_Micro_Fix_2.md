### 问题 2: budget_exceeded 状态不一致 ✅
**现状**: `current_usage (1496) < max_tokens (9500)` 但 `budget_exceeded = True`

**原因分析**:
- `can_load()` 方法同时检查**层级预算**和**总预算**
- 层级预算 (L4 = 1500 tokens) 比总预算 (9500 tokens) 更严格
- 当层级预算超限时，`budget_exceeded` 被错误地设置为 `True`
- 但 `budget_exceeded` 应该只反映**总预算**状态

**修复方案**:
```python
# 修复前
if not self.budget_manager.can_load(layer, content):
    self._budget_exceeded = True  # 错误：层级预算超限也设置
    return None

# 修复后
if not self.budget_manager.can_load(layer, content):
    tokens = self.budget_manager.estimate_tokens(content)
    
    # 只有总预算超限才设置 budget_exceeded
    if self.budget_manager.current_usage + tokens > self.budget_manager.max_tokens:
        self._budget_exceeded = True
        return None
    
    # 层级预算超限：跳过此文件但不设置 budget_exceeded
    return None
```

**修复后效果**:
```
search("docx"):
  token_budget: 1496/9500
  budget_exceeded: False  ✅ 状态一致
```
