## 修复问题

### 问题 1: Vector 模式明确输出
**现状**: Vector 模式实测为 `embedding`，但需要明确输出是 `embedding` 还是 `degraded`

**修复**: 
- `QwenEmbeddingEngine` 已正确检测连接状态
- `_test_connection()` 方法会尝试连接 Gitee AI Embedding API
- 成功则 `_mode = "embedding"`，失败则 `_mode = "degraded"`
- `get_vector_mode()` 返回实际模式

**验证结果**: ✅ Vector 模式: `embedding`（真实 Qwen3-Embedding-8B 后端已接通）

### 问题 2: Token 预算与 LazyLoader 联动重置
**现状**: `search()` 只 reset 了 `token_manager`，没有 reset `lazy_loader`

**修复**:
```python
# 修复前
def search(self, query: str, mode: str = "balanced", limit: int = 10) -> Dict:
    start = time.time()
    # 重置 token 预算
    self.token_manager.reset()

# 修复后
def search(self, query: str, mode: str = "balanced", limit: int = 10) -> Dict:
    start = time.time()
    # 重置 token 预算和 lazy_loader 状态（联动重置）
    self.token_manager.reset()
    self.lazy_loader.reset()
```

**验证结果**: 
- ✅ Token 预算每次独立统计（不累积）
- ✅ LazyLoader 状态每次独立重置（不包含污染文件）
- ✅ Budget exceeded 每次搜索开始时重置
