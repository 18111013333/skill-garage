## 🎯 系统概述

记忆系统是终极鸽子王的知识核心，负责信息存储、检索、管理。

## 📁 模块结构

```
memory/
├── MEMORY_V7.md            # 记忆系统说明
├── MEMORY.md               # 记忆索引
├── MEMORY_INDEX.md         # 详细索引
├── scenarios/              # 场景记忆
```

---

## 🧠 记忆层级

| 层级 | 名称 | 存储 | 生命周期 | 容量 |
|------|------|------|----------|------|
| L0 | 工作记忆 | 内存 | 当前任务 | 10KB |
| L1 | 会话记忆 | session-summaries/ | 单次会话 | 100KB |
| L2 | 场景记忆 | scenarios/ | 跨会话 | 1MB |
| L3 | 长期记忆 | MEMORY.md | 永久 | 无限 |

## 🔄 记忆流程

### 写入流程
```
新信息输入
    ↓
┌─────────────────────────────────────┐
│ 1. 信息分类                          │
│    - 判断信息类型                    │
│    - 确定存储层级                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 冲突检测                          │
│    - 检查是否与现有记忆冲突          │
│    - 冲突时应用冲突规则              │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 写入存储                          │
│    - 写入对应层级                    │
│    - 更新索引                        │
│    - 记录日志                        │
└─────────────────────────────────────┘
```

### 读取流程
```
查询请求
    ↓
┌─────────────────────────────────────┐
│ 1. 向量搜索                          │
│    - 语义相似度匹配                  │
│    - 返回相关片段                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 精确获取                          │
│    - 获取完整内容                    │
│    - 验证内容有效性                  │
└─────────────────────────────────────┘
    ↓
返回结果
```

## 🔧 记忆管理

### 清理规则
| 类型 | 触发条件 | 处理 |
|------|----------|------|
| 会话归档 | 会话结束 > 7天 | 移至 archive/ |
| 场景清理 | 场景失效 | 标记废弃 |
| 长期记忆 | 用户请求 | 审计后删除 |
| 紧急清空 | 安全事件 | 备份后清空 |

### 冲突处理
| 冲突类型 | 处理规则 |
|----------|----------|
| 用户画像冲突 | 新信息覆盖旧信息 |
| 时间敏感信息 | 以最新时间为准 |
| 事实性冲突 | 标记待确认，询问用户 |
| 系统配置冲突 | 需用户确认后覆盖 |
