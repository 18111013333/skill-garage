# QUERY_REWRITE_RULES.md - 查询改写规则

## 目的
定义检索前查询改写规则，确保检索query质量可控。

## 适用范围
所有检索操作前的查询改写。

## 改写规则

### 去歧义
| 规则 | 说明 | 示例 |
|------|------|------|
| 代词替换 | 将代词替换为具体实体 | "它" → "Python" |
| 省略补全 | 补充省略的主语/宾语 | "怎么样" → "Python怎么样" |
| 指代消解 | 解析指代关系 | "那个文件" → "report.docx" |

### 补实体
| 规则 | 说明 | 示例 |
|------|------|------|
| 实体识别 | 识别并标注实体 | "苹果" → "苹果(公司)" |
| 类型推断 | 推断实体类型 | "Python" → "Python(编程语言)" |
| 关联实体 | 添加关联实体 | "马斯克" → "马斯克 特斯拉" |

### 加时间范围
| 规则 | 说明 | 示例 |
|------|------|------|
| 时效词识别 | 识别时效关键词 | "最新" → "2024" |
| 时间推断 | 推断时间范围 | "最近" → "2024年" |
| 历史限定 | 限定历史时间 | "以前" → "2023年及以前" |

### 拆分子查询
| 规则 | 说明 | 示例 |
|------|------|------|
| 复合问题拆分 | 拆分复合问题 | "A和B的区别" → ["A是什么", "B是什么", "A vs B"] |
| 多实体拆分 | 拆分多实体查询 | "北京上海天气" → ["北京天气", "上海天气"] |
| 多维度拆分 | 拆分多维度查询 | "价格和评价" → ["价格", "评价"] |

### 加同义词
| 规则 | 说明 | 示例 |
|------|------|------|
| 同义词扩展 | 添加同义词 | "手机" → "手机 智能手机" |
| 缩写展开 | 展开缩写 | "AI" → "AI 人工智能" |
| 别名添加 | 添加别名 | "微信" → "微信 WeChat" |

## 保留原词规则

### 保留用户原词
| 条件 | 说明 |
|------|------|
| 专业术语 | 用户使用的专业术语 |
| 特定名称 | 用户指定的名称 |
| 精确匹配 | 用户要求精确匹配 |
| 引号内容 | 用户用引号标注的内容 |

### 扩展查询
| 条件 | 说明 |
|------|------|
| 模糊查询 | 用户意图模糊 |
| 广泛搜索 | 用户要求广泛搜索 |
| 信息不足 | 原词信息不足 |

## 改写流程

```
原始查询
    ↓
┌─────────────────────────────────────┐
│ 1. 分析查询                          │
│    - 识别实体                        │
│    - 识别意图                        │
│    - 识别时效                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 应用改写规则                      │
│    - 去歧义                          │
│    - 补实体                          │
│    - 加时间                          │
│    - 拆分子查询                      │
│    - 加同义词                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 生成改写结果                      │
│    - 主查询                          │
│    - 备选查询                        │
│    - 子查询列表                      │
└─────────────────────────────────────┘
    ↓
执行检索
```

## 改写结果格式

```json
{
  "rewriteId": "rw_001",
  "original": "最近AI新闻",
  "rewritten": {
    "main": "AI人工智能新闻 2024",
    "alternatives": ["人工智能最新动态", "AI行业资讯"],
    "subQueries": []
  },
  "rulesApplied": [
    "时效词识别: 最近 → 2024",
    "同义词扩展: AI → AI 人工智能"
  ]
}
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 改写失败 | 使用原始查询 |
| 歧义无法消除 | 保留多个解释 |
| 实体无法识别 | 使用原始词 |

## 维护方式
- 新增改写规则: 添加到改写规则表
- 调整保留规则: 更新保留原词规则
- 新增同义词: 更新同义词库
