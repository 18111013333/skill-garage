# RETRIEVAL_POLICY.md - 检索策略

## 目的
定义何时需要检索、何时不需要检索，确保不会该查的不查，也不会不该查的乱查。

## 适用范围
所有信息检索决策。

## 检索决策矩阵

### 按任务类型决策
| 任务类型 | 是否检索 | 检索类型 | 原因 |
|----------|----------|----------|------|
| qa | 按需 | 内部记忆 + 外部 | 可能需要补充信息 |
| search | 必须 | 外部网页 | 任务本身就是搜索 |
| document_edit | 按需 | 文件检索 | 需要了解文档内容 |
| planning | 按需 | 内部记忆 | 可能需要历史决策 |
| creative_generation | 通常不检索 | - | 创作不需要检索 |
| tool_use | 不检索 | - | 直接执行工具 |
| memory_update | 不检索 | - | 直接写入记忆 |
| multi_step_agent | 按需 | 全类型 | 根据子任务决定 |
| high_risk_request | 必须校验 | 全类型 | 高风险需要验证 |

### 按信息类型决策
| 信息类型 | 是否检索 | 检索类型 | 原因 |
|----------|----------|----------|------|
| 用户画像 | 内部检索 | 记忆检索 | 了解用户背景 |
| 事实性问题 | 必须检索 | 外部 + 记忆 | 确保准确性 |
| 时效性信息 | 必须检索 | 外部网页 | 获取最新信息 |
| 创作内容 | 不检索 | - | 基于模型能力 |
| 系统配置 | 内部检索 | 文件检索 | 获取配置信息 |
| 历史决策 | 内部检索 | 记忆检索 | 了解上下文 |

## 强制检索条件

### 时效敏感任务
| 任务类型 | 时效要求 | 检索要求 |
|----------|----------|----------|
| 新闻查询 | < 24小时 | 必须外部检索 |
| 价格查询 | < 1小时 | 必须外部检索 |
| 天气查询 | < 6小时 | 必须外部检索 |
| 政策查询 | < 30天 | 必须外部检索 |
| 职位查询 | < 7天 | 必须外部检索 |
| 产品规格 | < 30天 | 必须外部检索 |

### 事实敏感任务
| 任务类型 | 检索要求 | 校验要求 |
|----------|----------|----------|
| 数据查询 | 必须检索 | 多源校验 |
| 事实陈述 | 必须检索 | 来源标注 |
| 引用请求 | 必须检索 | 原文核对 |
| 对比分析 | 必须检索 | 多源对比 |

### 高风险任务
| 任务类型 | 检索要求 | 校验要求 |
|----------|----------|----------|
| 财务决策 | 必须检索 | 多源校验 |
| 法律咨询 | 必须检索 | 官方来源 |
| 医疗建议 | 必须检索 | 权威来源 |
| 安全操作 | 必须检索 | 官方文档 |

## 禁止检索条件

### 不需要检索
| 条件 | 原因 |
|------|------|
| 用户明确说"不要搜索" | 尊重用户意愿 |
| 纯创作任务 | 不需要外部信息 |
| 简单问候 | 无需检索 |
| 已有足够上下文 | 避免冗余 |
| 用户隐私问题 | 保护隐私 |

### 不应该检索
| 条件 | 原因 |
|------|------|
| 敏感个人信息 | 隐私保护 |
| 内部机密信息 | 安全边界 |
| 未授权访问 | 权限限制 |

## 检索类型选择

### 内部记忆检索
- **触发条件**: 需要用户历史、偏好、上下文
- **检索范围**: MEMORY.md, session-summaries, scenarios
- **优先级**: 高（优先使用内部信息）

### 文件检索
- **触发条件**: 需要用户文件内容
- **检索范围**: 用户授权的文件路径
- **优先级**: 中（需要用户授权）

### 外部网页检索
- **触发条件**: 需要最新信息、外部知识
- **检索范围**: 互联网
- **优先级**: 低（作为补充）

### 无需检索直接回答
- **触发条件**: 常识问题、创作任务、简单对话
- **依据**: 模型内置知识
- **优先级**: 默认

## 检索流程

```
任务分析
    ↓
┌─────────────────────────────────────┐
│ 检索决策                             │
│ - 任务类型判断                       │
│ - 信息类型判断                       │
│ - 强制检索检查                       │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 选择检索类型                         │
│ - 内部记忆                           │
│ - 文件检索                           │
│ - 外部网页                           │
│ - 组合检索                           │
└─────────────────────────────────────┘
    ↓
执行检索 → 返回结果
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 检索失败 | 使用缓存或降级 |
| 无检索结果 | 标记并继续 |
| 检索超时 | 返回部分结果 |

## 维护方式
- 新增任务类型: 更新决策矩阵
- 调整时效要求: 更新时效敏感表
- 新增强制检索: 更新强制检索条件
