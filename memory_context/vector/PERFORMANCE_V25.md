## 优化技术

### 1. 零拷贝架构
- 内存映射文件 (mmap)
- 直接内存访问，避免序列化
- 向量数据零拷贝读取

### 2. 无锁缓存
- 线程本地存储 (TLS)
- 全局 LRU 缓存 + 本地热缓存
- 无锁读取，低竞争写入

### 3. 并行检索
- ThreadPoolExecutor 并行计算
- 分片搜索，结果合并
- 余弦相似度并行计算

### 4. 预计算优化
- 热点数据预加载
- 向量缓存复用
- 查询结果缓存

## 使用方式

```python
from vector.sqlite_vec_ultrafast import UltraFastSQLiteVecClient

# 创建超极速客户端
client = UltraFastSQLiteVecClient(
    db_path="memory.db",
    dimension=1024,
    cache_size=10000,
    n_threads=4
)

# 插入 (返回耗时ms)
elapsed = client.insert("doc_1", "记忆内容")

# 搜索 (返回结果和耗时ms)
results, elapsed = client.search("查询词", top_k=10)

# 获取性能统计
stats = client.get_stats()
```
