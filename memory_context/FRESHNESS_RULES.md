# FRESHNESS_RULES.md - 时效规则

## 目的
区分稳定信息与易变信息，确保不会把旧知识误当当前事实。

## 适用范围
所有信息的时效性判断和处理。

## 信息时效分类

### 稳定信息 (Stable)
| 类型 | 说明 | 有效期 |
|------|------|--------|
| 历史事实 | 已发生的历史事件 | 永久 |
| 科学定律 | 科学原理和定律 | 永久 |
| 地理信息 | 地理位置、地形等 | 10年+ |
| 语言规则 | 语法、词汇等 | 10年+ |
| 数学公式 | 数学定理和公式 | 永久 |

### 中期稳定信息 (Medium-term)
| 类型 | 说明 | 有效期 |
|------|------|--------|
| 技术文档 | 技术规范、API文档 | 1-3年 |
| 产品规格 | 产品参数、规格 | 1-2年 |
| 法律法规 | 法律条文、政策 | 1-5年 |
| 学术知识 | 学术理论、方法 | 3-5年 |

### 易变信息 (Volatile)
| 类型 | 说明 | 有效期 |
|------|------|--------|
| 新闻资讯 | 新闻事件、动态 | 24小时 |
| 价格信息 | 商品价格、汇率 | 1小时 |
| 天气预报 | 天气信息 | 6小时 |
| 股票行情 | 股票价格、指数 | 实时 |
| 职位信息 | 招聘职位 | 7天 |
| 活动信息 | 活动、演出信息 | 30天 |

### 高时效内容列表
| 内容类型 | 时效窗口 | 检索要求 |
|----------|----------|----------|
| 新闻 | 24小时 | 必须外部检索 |
| 价格 | 1小时 | 必须外部检索 |
| 天气 | 6小时 | 必须外部检索 |
| 政策 | 30天 | 必须外部检索 |
| 职位 | 7天 | 必须外部检索 |
| 产品规格 | 30天 | 建议外部检索 |
| 体育赛事 | 实时 | 必须外部检索 |
| 公司动态 | 7天 | 建议外部检索 |

## 时效标记格式

### 信息时效标记
```
[FRESHNESS]
- Type: {信息类型}
- Published: {发布时间}
- ValidUntil: {有效期至}
- Status: {valid/expired/unknown}
```

### 过期警告
```
[WARNING: EXPIRED]
- 信息已过期
- 原有效期: {有效期}
- 过期时间: {过期时间}
- 建议: 重新检索最新信息
```

## 时效检查流程

```
信息获取
    ↓
┌─────────────────────────────────────┐
│ 1. 识别信息类型                      │
│    - 判断时效分类                    │
│    - 确定有效期                      │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 检查时效状态                      │
│    - 获取发布时间                    │
│    - 计算是否过期                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 处理时效问题                      │
│    - 有效 → 正常使用                 │
│    - 过期 → 标注警告或重新检索       │
│    - 未知 → 标注不确定               │
└─────────────────────────────────────┘
```

## 时效处理策略

### 有效信息
- 正常使用
- 标注发布时间
- 可选标注有效期

### 即将过期
- 标注即将过期警告
- 建议用户确认
- 可选重新检索

### 已过期
- 标注过期警告
- 建议重新检索
- 不作为主要依据

### 时效未知
- 标注时效未知
- 降低可信度
- 建议用户确认

## 时效与可信度

| 时效状态 | 可信度调整 |
|----------|------------|
| 有效 | 无调整 |
| 即将过期 | -0.1 |
| 已过期 | -0.3 |
| 时效未知 | -0.2 |

## 特殊场景处理

### 无发布时间
| 处理 | 说明 |
|------|------|
| 稳定信息 | 假设有效 |
| 易变信息 | 标注时效未知 |
| 高风险场景 | 必须重新检索 |

### 无法判断时效
| 处理 | 说明 |
|------|------|
| 保守处理 | 按易变信息处理 |
| 标注不确定 | 明确告知用户 |
| 建议检索 | 建议获取最新信息 |

## 异常处理

| 异常 | 处理 |
|------|------|
| 时效判断失败 | 按易变信息处理 |
| 发布时间解析失败 | 标注时效未知 |
| 有效期计算失败 | 保守处理 |

## 维护方式
- 新增信息类型: 添加到时效分类表
- 调整有效期: 更新有效期配置
- 新增高时效内容: 更新高时效列表
