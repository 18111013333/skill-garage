# REASONING_ENGINE.md - 深度推理引擎

## 目的
实现复杂问题的多步推理与深度分析，给出可信结论。

## 核心能力

### 1. 多步推理
- 问题分解
- 逐步推理
- 结论聚合
- 回溯修正

### 2. 因果分析
- 相关性分析
- 干预分析
- 反事实推理

### 3. 假设验证
- 自动生成假设
- 证据收集
- 假设验证

### 4. 置信度评估
- 证据权重
- 逻辑一致性
- 结论可信度

## 推理示例

```
问题: 为什么用户留存率下降？

推理链:
1. 数据分析 → 发现下降趋势
2. 假设生成 → 可能原因: 性能问题、竞品、功能缺失
3. 假设验证 → 收集证据验证各假设
4. 因果分析 → 确定根本原因
5. 结论 → 性能问题是主因 (置信度: 92%)
```
