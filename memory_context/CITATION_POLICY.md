# CITATION_POLICY.md - 引用策略

## 目的
定义什么时候必须给依据，什么时候可不显式引用，确保回答的依据展示有章法。

## 适用范围
所有回答中的来源引用和依据标注。

## 引用触发条件

### 必须引用
| 条件 | 说明 |
|------|------|
| 高风险回答 | 涉及医疗、法律、财务等 |
| 事实密集回答 | 包含大量事实陈述 |
| 时效敏感回答 | 涉及时效敏感信息 |
| 外部检索得来 | 从外部来源获取的信息 |
| 用户要求引用 | 用户明确要求标注来源 |
| 数据引用 | 引用具体数据、统计 |

### 建议引用
| 条件 | 说明 |
|------|------|
| 重要事实 | 关键事实陈述 |
| 专业观点 | 专业领域观点 |
| 争议内容 | 存在争议的内容 |
| 用户可能质疑 | 用户可能质疑的内容 |

### 可不显式引用
| 条件 | 说明 |
|------|------|
| 常识知识 | 广为人知的常识 |
| 用户已知信息 | 用户已提供的信息 |
| 系统记忆 | 来自用户画像的信息 |
| 创作内容 | 创作生成的原创内容 |
| 一般对话 | 普通对话交流 |

## 引用格式

### 标准引用格式
```
{陈述内容} [来源: {来源名称}, {时间}]
```

### 多来源引用
```
{陈述内容}
[来源: 
- {来源A} ({可信度})
- {来源B} ({可信度})
]
```

### 冲突信息引用
```
关于{主题}，存在不同说法：
- 观点A: {内容A} [来源: {来源A}]
- 观点B: {内容B} [来源: {来源B}]
```

## 引用内容要求

### 必须包含
| 元素 | 说明 |
|------|------|
| 来源名称 | 信息来源的名称 |
| 时间 | 信息发布/获取时间 |
| 可信度 | 来源可信度（可选） |

### 可选包含
| 元素 | 说明 |
|------|------|
| 原文链接 | 原文URL（如有） |
| 作者 | 作者名称 |
| 发布机构 | 发布机构名称 |

## 禁止行为

### 禁止把猜测写成事实
```
❌ 错误: "明天一定会下雨"
✅ 正确: "根据天气预报，明天可能下雨 [来源: 天气预报, 2026-04-06]"
```

### 禁止编造来源
```
❌ 错误: "据专家研究..." (无实际来源)
✅ 正确: "根据我的了解..." (明确是推断)
```

### 禁止混淆事实与推断
```
❌ 错误: "他一定是..." (把推断当事实)
✅ 正确: "根据现有信息推测，他可能是..."
```

## 引用与不确定性

### 高置信引用
```
{事实陈述} [来源: {权威来源}]
```

### 中置信引用
```
{事实陈述} [来源: {一般来源}, 需进一步验证]
```

### 低置信/不确定
```
据{来源}报道，{内容}，但该信息尚未得到独立验证。
```

## 引用示例

### 新闻引用
```
根据新华社报道，2026年4月6日，{事件内容}。[来源: 新华社, 2026-04-06]
```

### 数据引用
```
2025年中国GDP增长率为5.2%。[来源: 国家统计局, 2026-01-20]
```

### 学术引用
```
研究表明，深度学习在图像识别领域的准确率已超过人类。[来源: Nature, 2025-03-15]
```

### 多来源引用
```
关于AI发展趋势，业界存在不同观点：
- 观点A: AI将在5年内实现AGI [来源: OpenAI, 2026-01]
- 观点B: AGI仍需数十年发展 [来源: 学术研究, 2025-12]
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 来源信息缺失 | 标注"来源不详" |
| 来源可信度低 | 标注可信度警告 |
| 来源冲突 | 标注冲突，列出所有来源 |

## 维护方式

- 调整触发条件: 更新触发条件表
- 新增禁止行为: 更新禁止行为表
