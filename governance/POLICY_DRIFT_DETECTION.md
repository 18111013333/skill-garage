# POLICY_DRIFT_DETECTION.md - 策略漂移检测规则

**版本: V27.0**

## 目的
定义策略漂移检测规则，确保平台知道策略有没有"说一套做一套"，实现"漂移可检测、偏差可告警、修复可追踪"。

## 适用范围
所有策略漂移检测场景，包括：
- 文档策略与执行策略偏差
- 区域偏差
- 租户偏差
- 控制台配置偏差
- 回滚后残留偏差

---

## 一、漂移类型

| 类型 | 代码 | 说明 | 风险等级 |
|------|------|------|----------|
| 文档漂移 | DOC_DRIFT | 文档与执行不一致 | 高 |
| 区域漂移 | REGION_DRIFT | 区域间策略不一致 | 中 |
| 租户漂移 | TENANT_DRIFT | 租户间策略不一致 | 中 |
| 配置漂移 | CONFIG_DRIFT | 配置与策略不一致 | 高 |
| 回滚残留 | ROLLBACK_DRIFT | 回滚后残留配置 | 高 |

---

## 二、文档策略与执行策略偏差

### 2.1 检测内容

| 检测项 | 说明 |
|--------|------|
| 规则一致性 | 文档规则与执行规则一致 |
| 参数一致性 | 文档参数与执行参数一致 |
| 版本一致性 | 文档版本与执行版本一致 |

### 2.2 检测方法

```
检测流程:
1. 解析文档策略
2. 获取执行策略
3. 对比差异
4. 生成漂移报告
```

### 2.3 偏差处理

| 偏差类型 | 处理方式 |
|----------|----------|
| 文档过时 | 更新文档 |
| 执行异常 | 修复执行 |
| 不一致 | 统一处理 |

---

## 三、区域偏差

### 3.1 检测内容

| 检测项 | 说明 |
|--------|------|
| 策略版本 | 各区域策略版本一致 |
| 策略配置 | 各区域策略配置一致 |
| 策略状态 | 各区域策略状态一致 |

### 3.2 检测方法

```
检测流程:
1. 获取各区域策略
2. 对比区域间差异
3. 识别偏差区域
4. 生成漂移报告
```

### 3.3 偏差处理

| 偏差类型 | 处理方式 |
|----------|----------|
| 版本不一致 | 同步版本 |
| 配置不一致 | 同步配置 |
| 状态不一致 | 同步状态 |

---

## 四、租户偏差

### 4.1 检测内容

| 检测项 | 说明 |
|--------|------|
| 租户策略 | 租户策略与模板一致 |
| 租户配置 | 租户配置与模板一致 |
| 租户例外 | 租户例外合规 |

### 4.2 检测方法

```
检测流程:
1. 获取租户策略
2. 对比模板策略
3. 识别偏差租户
4. 生成漂移报告
```

### 4.3 偏差处理

| 偏差类型 | 处理方式 |
|----------|----------|
| 策略偏差 | 同步策略 |
| 配置偏差 | 同步配置 |
| 例外过期 | 清理例外 |

---

## 五、控制台配置偏差

### 5.1 检测内容

| 检测项 | 说明 |
|--------|------|
| 控制台配置 | 控制台配置与策略一致 |
| 手动修改 | 检测手动修改 |
| 未授权修改 | 检测未授权修改 |

### 5.2 检测方法

```
检测流程:
1. 获取控制台配置
2. 获取策略配置
3. 对比差异
4. 识别未授权修改
```

### 5.3 偏差处理

| 偏差类型 | 处理方式 |
|----------|----------|
| 配置漂移 | 恢复配置 |
| 未授权修改 | 告警 + 恢复 |
| 手动修改 | 评估 + 处理 |

---

## 六、回滚后残留偏差

### 6.1 检测内容

| 检测项 | 说明 |
|--------|------|
| 回滚完整性 | 回滚后配置完整 |
| 残留配置 | 检测残留配置 |
| 依赖恢复 | 依赖关系恢复 |

### 6.2 检测方法

```
检测流程:
1. 获取回滚前配置
2. 获取回滚后配置
3. 对比差异
4. 识别残留配置
```

### 6.3 偏差处理

| 偏差类型 | 处理方式 |
|----------|----------|
| 回滚不完整 | 完成回滚 |
| 残留配置 | 清理残留 |
| 依赖异常 | 修复依赖 |

---

## 七、漂移告警

### 7.1 告警级别

| 级别 | 条件 | 通知方式 |
|------|------|----------|
| 严重 | 关键策略漂移 | 电话 + 短信 |
| 高 | 重要策略漂移 | 短信 + IM |
| 中 | 一般策略漂移 | IM + 邮件 |
| 低 | 轻微策略漂移 | 邮件 |

### 7.2 告警内容

```
【策略漂移告警】
漂移类型: {drift_type}
策略ID: {policy_id}
偏差详情: {drift_detail}
影响范围: {impact}
建议处理: {suggestion}
```
