# MARKETPLACE_PAYOUTS.md - 终极鸽子王市场资产分账与打款规则

## 目的
定义市场资产销售的分账规则和打款流程。

## 适用范围
所有市场资产销售、模板销售、连接器销售的结算。

---

## 一、收益产生

### 1.1 收益来源
| 来源类型 | 说明 | 分账比例 |
|----------|------|----------|
| 资产销售 | 模板/连接器/技能销售 | 70%开发者 / 30%平台 |
| 订阅分成 | 资产订阅收入 | 75%开发者 / 25%平台 |
| 服务收入 | 定制服务收入 | 80%开发者 / 20%平台 |

### 1.2 收益计算
```
收益计算公式:
开发者收益 = 销售金额 × 开发者分成比例 - 平台费用 - 退款扣除
```

---

## 二、分账比例

### 2.1 标准分账
| 资产类型 | 开发者 | 平台 |
|----------|--------|------|
| 标准模板 | 70% | 30% |
| 高级模板 | 75% | 25% |
| 认证资产 | 80% | 20% |
| 官方资产 | 50% | 50% |

### 2.2 特殊分账
| 场景 | 分账调整 |
|------|----------|
| 首月促销 | 开发者让利10% |
| 批量采购 | 开发者让利5% |
| 战略合作 | 协商分账 |

---

## 三、最低结算门槛

### 3.1 门槛设置
| 结算方式 | 最低门槛 |
|----------|----------|
| 银行转账 | 1000元 |
| 信用余额 | 100元 |
| PayPal | 100美元 |

### 3.2 未达门槛处理
| 处理方式 | 说明 |
|----------|------|
| 累计结算 | 累计到达门槛后结算 |
| 按需结算 | 申请后可提前结算（扣除手续费） |

---

## 四、冻结期

### 4.1 冻结规则
| 资产类型 | 冻结期 | 说明 |
|----------|--------|------|
| 标准资产 | 30天 | 防止退款风险 |
| 高级资产 | 45天 | 更长保护期 |
| 服务类 | 15天 | 服务完成后 |

### 4.2 冻结解除
```
冻结解除流程
    │
    ├── 正常解除
    │   └── 冻结期满自动解除
    │
    ├── 提前解除
    │   ├── 无退款记录
    │   └── 申请审批通过
    │
    └── 延长冻结
        ├── 有退款申请
        └── 有争议处理中
```

---

## 五、风险保留

### 5.1 风险保留金
| 风险类型 | 保留比例 | 保留期限 |
|----------|----------|----------|
| 新开发者 | 10% | 前3个月 |
| 高风险资产 | 15% | 持续 |
| 争议资产 | 20% | 争议解决后 |

### 5.2 风险释放
| 条件 | 处理 |
|------|------|
| 无退款记录 | 释放保留金 |
| 无投诉记录 | 释放保留金 |
| 争议解决 | 释放相关保留金 |

---

## 六、违规追回

### 6.1 违规类型
| 违规类型 | 处理 |
|----------|------|
| 资产侵权 | 扣除全部收益 |
| 恶意刷单 | 扣除违规收益 |
| 虚假宣传 | 扣除相关收益 |
| 安全问题 | 扣除相关收益 |

### 6.2 追回流程
```
违规追回流程
    ↓
┌─────────────────────────────────────┐
│ 1. 违规认定                          │
│    - 调查核实                        │
│    - 违规等级评估                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 追回决定                          │
│    - 确定追回金额                    │
│    - 通知开发者                      │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 追回执行                          │
│    - 从待结算扣除                    │
│    - 或从信用余额扣除                │
└─────────────────────────────────────┘
```

---

## 七、打款流程

### 7.1 打款周期
| 周期 | 打款日 | 说明 |
|------|--------|------|
| 月结 | 每月15日 | 上月可结算收益 |
| 周结 | 每周五 | 上周可结算收益 |

### 7.2 打款流程
```
打款流程
    ↓
┌─────────────────────────────────────┐
│ 1. 收益汇总                          │
│    - 汇总可结算收益                  │
│    - 生成结算单                      │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 开发者确认                        │
│    - 发送结算单                      │
│    - 开发者确认                      │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 打款执行                          │
│    - 执行打款                        │
│    - 发送打款通知                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 打款确认                          │
│    - 开发者确认收款                  │
│    - 完成结算                        │
└─────────────────────────────────────┘
```
