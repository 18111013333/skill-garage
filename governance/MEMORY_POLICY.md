# MEMORY_POLICY.md - 终极鸽子王记忆治理策略

## 目的
定义记忆的生命周期管理、访问控制、清理规则。

## 适用范围
所有记忆层级的创建、更新、删除、归档操作。

## 记忆生命周期

### 生命周期阶段
```
创建 → 活跃 → 衰退 → 归档 → 清理
```

### 各阶段定义
| 阶段 | 条件 | 动作 |
|------|------|------|
| 创建 | 新信息写入 | 建立索引 |
| 活跃 | 30天内被访问 | 保持热存储 |
| 衰退 | 30天未访问 | 移至冷存储 |
| 归档 | 90天未访问 | 压缩归档 |
| 清理 | 365天未访问 | 标记删除 |

## 写入规则

### 允许写入长期记忆
| 信息类型 | 条件 | 说明 |
|----------|------|------|
| 用户画像 | 用户明确陈述 | 长期稳定偏好 |
| 系统配置 | 系统生成 | 系统运行配置 |
| 关键决策 | 用户确认 | 重要决策记录 |
| 学习成果 | 已验证 | 系统学习结果 |

### 仅留在会话层
| 信息类型 | 条件 | 说明 |
|----------|------|------|
| 临时偏好 | 当前会话有效 | 短期偏好 |
| 上下文信息 | 仅当前任务 | 任务相关 |
| 草稿内容 | 未确认 | 临时内容 |
| 错误信息 | 待修复 | 错误记录 |

### 禁止写入
| 信息类型 | 原因 |
|----------|------|
| 密码 | 安全风险 |
| API密钥 | 安全风险 |
| 敏感隐私 | 隐私保护 |
| 临时Token | 时效性 |

## 用户显式操作

### 保存请求
```
用户: "记住我喜欢简洁的回答"
动作: 写入长期记忆，标记为用户显式偏好
```

### 忘记请求
```
用户: "忘掉我之前说的那个"
动作: 标记对应记忆为 stale，不立即删除
```

### 查看请求
```
用户: "你知道关于我的什么"
动作: 返回用户画像摘要
```

## 最小化记忆原则

### 原则定义
1. 只存储必要信息
2. 优先存储结构化数据
3. 避免存储可推导信息
4. 定期清理过期信息

### 实施规则
| 规则 | 说明 |
|------|------|
| 去重 | 相同信息只存一份 |
| 压缩 | 长内容压缩存储 |
| 归档 | 旧信息定期归档 |
| 清理 | 过期信息自动清理 |

## 访问控制

### 访问权限
| 角色 | L1会话 | L2场景 | L3长期 |
|------|--------|--------|--------|
| 用户 | 读写 | 读写 | 读写 |
| 系统 | 只读 | 读写 | 读写 |
| 技能 | 只读 | 只读 | 只读 |

### 敏感信息处理
| 信息类型 | 处理方式 |
|----------|----------|
| 密码 | 禁止存储 |
| API密钥 | 加密存储 |
| 个人隐私 | 标记敏感，限制访问 |
| 财务信息 | 加密存储 |

## 删除规则

### 删除条件
| 条件 | 动作 |
|------|------|
| 用户请求 | 审计后删除 |
| 过期清理 | 自动归档后删除 |
| 安全事件 | 紧急清空 |
| 冲突无法解决 | 标记待确认 |

### 删除流程
```
删除请求
    ↓
权限检查 → 无权限 → 拒绝
    ↓
审计记录
    ↓
备份创建
    ↓
删除执行
    ↓
索引更新
```

## 归档规则

### 归档触发
| 触发条件 | 归档类型 |
|----------|----------|
| 会话结束 > 7天 | 会话归档 |
| 场景失效 | 场景归档 |
| 90天未访问 | 自动归档 |

### 归档格式
```
archive/
├── sessions/
│   └── session-YYYY-MM-DD.tar.gz
├── scenarios/
│   └── scenario-YYYY-MM-DD.tar.gz
└── index/
    └── archive-index.json
```

## 清理规则

### 自动清理
| 清理项 | 条件 | 周期 |
|--------|------|------|
| 过期会话 | > 7天 | 每日 |
| 冷数据 | > 90天 | 每周 |
| 归档数据 | > 365天 | 每月 |

### 清理日志
```json
{
  "cleanupId": "cleanup_001",
  "timestamp": "2026-04-06T10:32:00+08:00",
  "type": "expired_sessions",
  "count": 10,
  "freedSpace": 1048576
}
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 写入失败 | 重试 + 降级到临时存储 |
| 读取失败 | 返回缓存 + 标记修复 |
| 删除失败 | 回滚 + 记录错误 |
| 索引损坏 | 重建索引 |

## 维护方式
- 调整生命周期: 修改各阶段条件
- 调整访问控制: 更新权限表
- 新增清理规则: 更新清理规则表
