# ERROR_TAXONOMY.md - 错误分类体系

## 目的
定义统一的错误分类体系，确保错误可追踪、可分析、可修复。

## 适用范围
所有系统错误、异常、失败事件。

## 错误分类

### 一级分类
| 代码 | 名称 | 说明 |
|------|------|------|
| E0xx | 系统错误 | 系统级故障 |
| E1xx | 工具错误 | 工具调用失败 |
| E2xx | 技能错误 | 技能执行失败 |
| E3xx | 数据错误 | 数据处理失败 |
| E4xx | 网络错误 | 网络通信失败 |
| E5xx | 权限错误 | 权限验证失败 |
| E6xx | 配置错误 | 配置问题 |
| E7xx | 业务错误 | 业务逻辑错误 |
| E8xx | 用户错误 | 用户输入问题 |
| E9xx | 未知错误 | 未分类错误 |

## 错误严重级别

| 级别 | 说明 | 处理 |
|------|------|------|
| debug | 调试信息 | 记录日志 |
| info | 提示信息 | 记录日志 |
| warning | 警告 | 记录日志 + 提示 |
| error | 错误 | 记录日志 + 处理 |
| critical | 严重 | 记录日志 + 通知 + 降级 |

## 异常处理

| 异常 | 处理 |
|------|------|
| 错误分类失败 | 归类为 E901 |
| 错误记录失败 | 缓存到内存 |
| 统计失败 | 使用缓存数据 |

## 维护方式
- 新增错误类型: 添加到对应分类
- 调整严重级别: 修改级别定义
- 新增统计维度: 更新统计维度表
