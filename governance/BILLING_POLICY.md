# BILLING_POLICY.md - 计费与结算规则

## 目的
定义计费与结算规则，建立正式计费口径。

## 适用范围
所有租户计费、成本结算、额度管理。

## 计费周期

### 周期类型
| 周期类型 | 说明 | 结算时间 |
|----------|------|----------|
| 月度 | 自然月计费 | 次月 1-5 日 |
| 季度 | 自然季度计费 | 次季度首月 1-10 日 |
| 年度 | 自然年计费 | 次年 1 月 1-15 日 |

### 周期配置
```json
{
  "billing_cycles": {
    "monthly": {
      "start_day": 1,
      "end_day": "last_day_of_month",
      "settlement_days": "1-5",
      "invoice_day": 10
    },
    "quarterly": {
      "months": [1, 4, 7, 10],
      "settlement_days": "1-10",
      "invoice_day": 15
    },
    "annual": {
      "start_month": 1,
      "settlement_days": "1-15",
      "invoice_day": 20
    }
  }
}
```

| 层级 | 计费方式 | 说明 |

## 退款与补偿

### 退款条件
| 条件 | 退款比例 | 说明 |
|------|----------|------|
| 服务不可用 > 4 小时 | 100% 月费 | 全额退款 |
| 服务不可用 1-4 小时 | 50% 月费 | 部分退款 |
| SLA 未达成 | 按比例 | 按未达成比例 |
| 计费错误 | 全额 | 错误金额全额退 |

### 补偿方式
| 方式 | 说明 | 适用场景 |
|------|------|----------|
| 现金退款 | 退到支付账户 | 用户要求 |
| 额度补偿 | 增加免费额度 | 常规补偿 |
| 服务延期 | 延长服务期限 | 订阅服务 |

### 退款流程
```
申请提交 → 审核确认 → 计算金额 → 审批 → 执行退款 → 记录归档
```

## 账单与发票

### 账单内容
| 项目 | 说明 |
|------|------|
| 账单周期 | 计费周期 |
| 计费对象 | 租户/团队/项目 |
| 使用明细 | 各项使用量 |
| 费用明细 | 各项费用 |
| 免费额度 | 已用免费额度 |
| 超额费用 | 超额部分费用 |
| 税费 | 适用税费 |
| 应付总额 | 最终应付金额 |

### 发票开具
| 类型 | 说明 | 开具时间 |
|------|------|----------|
| 电子发票 | 标准发票 | 账单确认后 3 日内 |
| 纸质发票 | 特殊需求 | 申请后 7 日内 |
| 增值税专用发票 | 企业客户 | 资质审核后开具 |

## 结算流程

### 标准流程
```
周期结束 → 使用量统计 → 费用计算 → 账单生成 → 账单确认 → 支付 → 发票开具
```

### 结算时限
| 阶段 | 时限 | 说明 |
|------|------|------|
| 使用量统计 | 周期后 1 日 | 统计使用量 |
| 费用计算 | 周期后 2 日 | 计算费用 |
| 账单生成 | 周期后 3 日 | 生成账单 |
| 账单确认 | 生成后 3 日 | 用户确认 |
| 支付期限 | 确认后 15 日 | 支付截止 |
