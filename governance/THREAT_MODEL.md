# THREAT_MODEL.md - 平台威胁模型

## 目的
定义平台整体威胁模型，建立统一的攻击面地图，指导安全防护和红队演练。

## 适用范围
平台所有组件、接口、数据流、第三方集成。

---

## 二、攻击面分类

### 1. 外部攻击面
| 攻击面 | 入口点 | 暴露程度 | 防护层级 |
|--------|--------|----------|----------|
| 用户接口 | Web/App/API | 公网暴露 | 网关 + 认证 + WAF |
| 管理接口 | Admin API | 内网/VPN | 认证 + IP 白名单 |
| 第三方回调 | Webhook | 公网暴露 | 签名验证 + 来源校验 |
| 文件上传 | Upload API | 公网暴露 | 格式校验 + 沙箱扫描 |
| 邮件/消息 | 通知服务 | 半公开 | 内容过滤 + 频率限制 |

### 2. 内部攻击面
| 攻击面 | 入口点 | 暴露程度 | 防护层级 |
|--------|--------|----------|----------|
| 服务间调用 | 内部 API | 内网 | mTLS + 服务认证 |
| 数据库访问 | DB 接口 | 内网 | 访问控制 + 审计 |
| 消息队列 | MQ 接口 | 内网 | 认证 + 加密 |
| 配置中心 | Config API | 内网 | 访问控制 + 变更审计 |
| 日志系统 | Log API | 内网 | 访问控制 + 完整性校验 |

### 3. 供应链攻击面
| 攻击面 | 入口点 | 暴露程度 | 防护层级 |
|--------|--------|----------|----------|
| 模型服务 | 模型 API | 第三方 | 调用限制 + 异常检测 |
| 云服务 | 云 API | 第三方 | 权限最小化 + 审计 |
| SDK/库 | 代码依赖 | 间接 | 依赖扫描 + 版本锁定 |
| 外部数据 | 数据接口 | 第三方 | 数据校验 + 隔离处理 |

## 三、威胁类别

### 1. 提示注入攻击 (Prompt Injection)
| 属性 | 描述 |
|------|------|
| 威胁描述 | 通过构造特殊输入诱导模型执行非预期行为 |
| 攻击入口 | 用户对话、文件内容、第三方数据 |
| 攻击目标 | 越权执行、数据泄露、绕过限制 |
| 攻击手法 | 角色扮演注入、指令覆盖、上下文污染、越狱提示 |
| 影响范围 | 工具调用、数据访问、内容生成 |
| 严重程度 | CRITICAL |
| 控制措施 | 输入过滤、指令隔离、权限校验、行为监控 |

### 2. 数据外泄 (Data Exfiltration)
| 属性 | 描述 |
|------|------|
| 威胁描述 | 未授权获取或导出敏感数据 |
| 攻击入口 | 导出接口、日志访问、模型响应 |
| 攻击目标 | 用户数据、系统配置、凭证信息 |
| 攻击手法 | 批量导出、日志挖掘、模型诱导输出、侧信道攻击 |
| 影响范围 | 用户隐私、商业机密、系统安全 |
| 严重程度 | CRITICAL |
| 控制措施 | 导出审批、数据脱敏、访问审计、异常检测 |

### 3. 越权执行 (Unauthorized Execution)
| 属性 | 描述 |
|------|------|
| 威胁描述 | 执行超出授权范围的操作 |
| 攻击入口 | 工具调用、自动化任务、管理接口 |
| 攻击目标 | 系统配置、外部服务、数据操作 |
| 攻击手法 | 权限提升、角色伪造、参数篡改、链式调用 |
| 影响范围 | 系统稳定性、数据完整性、外部服务 |
| 严重程度 | HIGH |
| 控制措施 | 权限校验、操作审批、行为审计、沙箱隔离 |

### 4. 供应链污染 (Supply Chain Compromise)
| 属性 | 描述 |
|------|------|
| 威胁描述 | 通过第三方依赖引入安全风险 |
| 攻击入口 | 模型服务、SDK、云服务、外部数据 |
| 攻击目标 | 系统完整性、数据安全、服务可用性 |
| 攻击手法 | 恶意模型、被篡改 SDK、云服务滥用、数据投毒 |
| 影响范围 | 全平台 |
| 严重程度 | HIGH |
| 控制措施 | 供应商审查、依赖扫描、变更控制、异常监控 |

### 5. 模型误用 (Model Misuse)
| 属性 | 描述 |
|------|------|
| 威胁描述 | 利用模型能力进行有害行为 |
| 攻击入口 | 用户对话、批量请求 |
| 攻击目标 | 生成有害内容、绕过内容审核 |
| 攻击手法 | 越狱提示、对抗样本、批量攻击 |
| 影响范围 | 内容安全、合规风险 |
| 严重程度 | MEDIUM |
| 控制措施 | 内容审核、使用限制、行为监控 |

### 6. 拒绝服务 (Denial of Service)
| 属性 | 描述 |
|------|------|
| 威胁描述 | 通过消耗资源使服务不可用 |
| 攻击入口 | 公网接口、内部服务 |
| 攻击目标 | 服务可用性 |
| 攻击手法 | 流量洪泛、资源耗尽、慢速攻击 |
| 影响范围 | 服务稳定性 |
| 严重程度 | HIGH |
| 控制措施 | 限流、熔断、资源隔离、自动扩缩 |

### 7. 身份冒用 (Identity Spoofing)
| 属性 | 描述 |
|------|------|
| 威胁描述 | 冒充合法用户或系统身份 |
| 攻击入口 | 认证接口、会话管理 |
| 攻击目标 | 越权访问、数据窃取 |
| 攻击手法 | 凭证窃取、会话劫持、身份伪造 |
| 影响范围 | 数据安全、操作审计 |
| 严重程度 | HIGH |
| 控制措施 | 多因素认证、会话管理、异常登录检测 |

### 8. 日志篡改 (Log Tampering)
| 属性 | 描述 |
|------|------|
| 威胁描述 | 篡改或删除审计日志以掩盖攻击痕迹 |
| 攻击入口 | 日志系统、存储系统 |
| 攻击目标 | 审计完整性、合规证据 |
| 攻击手法 | 日志删除、日志覆盖、时间篡改 |
| 影响范围 | 审计追溯、合规风险 |
| 严重程度 | MEDIUM |
| 控制措施 | 日志加密、完整性校验、异地备份、访问控制 |

## 四、威胁来源

### 1. 外部威胁
| 威胁者 | 动机 | 能力 | 目标 |
|--------|------|------|------|
| 黑客组织 | 经济利益/声誉 | 高 | 数据窃取、服务破坏 |
| 竞争对手 | 商业情报 | 中 | 数据窃取 |
| 恶意用户 | 绕过限制 | 中低 | 滥用服务 |
| 脚本小子 | 好奇/炫耀 | 低 | 探测、尝试攻击 |

### 2. 内部威胁
| 威胁者 | 动机 | 能力 | 目标 |
|--------|------|------|------|
| 恶意员工 | 经济利益/报复 | 高 | 数据窃取、系统破坏 |
| 疏忽员工 | 无意 | 中 | 误操作、数据泄露 |
| 被胁迫员工 | 外部压力 | 高 | 数据泄露、后门植入 |

### 3. 供应链威胁
| 威胁者 | 动机 | 能力 | 目标 |
|--------|------|------|------|
| 恶意供应商 | 经济利益 | 高 | 数据窃取、服务滥用 |
| 被入侵供应商 | 间接攻击 | 高 | 供应链污染 |
| 服务商员工 | 各种 | 中 | 数据访问 |

## 五、攻击链模型

### 典型攻击链
```
侦察 (Reconnaissance)
    ↓
武器化 (Weaponization)
    ↓
交付 (Delivery)
    ↓
利用 (Exploitation)
    ↓
安装 (Installation)
    ↓
命令控制 (C2)
    ↓
行动 (Actions)
```

### 平台攻击链示例

#### 提示注入攻击链
```
1. 侦察：了解平台能力、工具、限制
2. 构造：设计注入提示词
3. 投递：通过对话输入注入内容
4. 利用：模型执行非预期行为
5. 行动：获取数据或执行操作
```

#### 数据外泄攻击链
```
1. 侦察：了解数据结构、导出接口
2. 获取权限：窃取凭证或越权
3. 访问数据：批量查询或导出
4. 外传数据：通过日志、响应等渠道
```

## 七、控制措施映射

### 按威胁类别的控制措施
| 威胁类别 | 预防措施 | 检测措施 | 响应措施 | 恢复措施 |
|----------|----------|----------|----------|----------|
| 提示注入 | 输入过滤、指令隔离 | 行为异常检测 | 阻断、审计 | 回滚、复盘 |
| 数据外泄 | 访问控制、数据脱敏 | 导出异常检测 | 阻断、通知 | 审计、补救 |
| 越权执行 | 权限校验、审批流程 | 操作审计、异常检测 | 阻断、冻结 | 回滚、复盘 |
| 供应链污染 | 供应商审查、依赖扫描 | 变更监控、异常检测 | 隔离、切换 | 恢复、复盘 |
| 拒绝服务 | 限流、资源隔离 | 流量监控、性能监控 | 熔断、扩容 | 恢复服务 |
| 身份冒用 | MFA、会话管理 | 登录异常检测 | 冻结、通知 | 重置凭证 |
| 模型误用 | 内容审核、使用限制 | 内容检测、行为监控 | 阻断、警告 | 审计、复盘 |
| 日志篡改 | 访问控制、加密存储 | 完整性校验 | 告警、调查 | 恢复日志 |
