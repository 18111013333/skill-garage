# RECORD_RETENTION.md - 合规记录保留规则

## 目的
定义合规相关记录的保留规则，确保合规留痕有统一保留制度。

## 适用范围
所有审批记录、关键输出、审计日志、例外批准记录。

## 保留期限总览

### 按记录类型
| 记录类型 | 保留期限 | 存储要求 | 销毁方式 |
|----------|----------|----------|----------|
| 审批记录 | 7 年 | 归档存储 | 安全删除 |
| 关键输出 | 7 年 | 归档存储 | 安全删除 |
| 审计日志 | 7 年 | 归档存储 | 安全删除 |
| 例外批准 | 7 年 | 归档存储 | 安全删除 |
| 合规报告 | 7 年 | 归档存储 | 安全删除 |
| 事故记录 | 5 年 | 归档存储 | 安全删除 |
| 操作日志 | 3 年 | 标准存储 | 安全删除 |

### 按租户/领域
| 租户类型 | 基础期限 | 延长要求 |
|----------|----------|----------|
| 标准租户 | 标准期限 | 无 |
| 金融租户 | 标准期限 × 2 | 监管要求 |
| 医疗租户 | 标准期限 × 2 | 监管要求 |
| 政府租户 | 标准期限 × 2 | 法规要求 |
| 儿童相关 | 标准期限 × 2 | 法规要求 |

## 销毁规则

### 销毁条件
| 条件 | 说明 |
|------|------|
| 保留期满 | 保留期限已过 |
| 无法律保留 | 无法律诉讼保留 |
| 无审计需求 | 无进行中审计 |
| 审批通过 | 销毁申请已批准 |

### 销毁流程
```
1. 识别到期记录
   ↓
2. 检查保留条件
   ↓
3. 提交销毁申请
   ↓
4. 审批销毁申请
   ↓
5. 执行安全销毁
   ↓
6. 记录销毁证明
```

### 销毁方式
| 方式 | 适用场景 | 安全级别 |
|------|----------|----------|
| 安全删除 | 电子记录 | 高 |
| 物理销毁 | 物理介质 | 极高 |
| 加密擦除 | 加密存储 | 高 |
