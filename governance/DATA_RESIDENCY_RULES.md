# DATA_RESIDENCY_RULES.md - 数据驻留规则

## 目的
定义数据驻留规则，确保数据驻留要求有清晰制度，不会和高可用目标冲突。

## 适用范围
平台所有数据的存储、复制、传输、处理。

---

## 一、数据驻留原则

| 原则 | 说明 | 执行要求 |
|------|------|----------|
| 本地优先 | 数据优先存储在产生地 | 默认本地存储 |
| 最小跨境 | 最小化跨境数据传输 | 审批后跨境 |
| 合规优先 | 合规要求高于可用性 | 合规强制执行 |
| 分类管理 | 按数据类型分类管理 | 分类驻留规则 |
| 透明可追溯 | 数据位置可追溯 | 完整记录 |

---

## 二、数据分类

### 2.1 按驻留要求分类
| 分类 | 驻留要求 | 跨境限制 | 示例数据 |
|------|----------|----------|----------|
| 强制驻留 | 必须本地存储 | 禁止跨境 | 身份信息、金融数据 |
| 条件驻留 | 本地存储，条件跨境 | 审批后跨境 | 对话数据、行为数据 |
| 自由流动 | 无驻留要求 | 自由跨境 | 公开数据、脱敏数据 |
| 禁止存储 | 禁止存储 | 禁止存储 | 违规内容、临时数据 |

### 2.2 按敏感级别分类
| 级别 | 敏感程度 | 驻留要求 | 加密要求 |
|------|----------|----------|----------|
| L1 | 公开 | 无要求 | 传输加密 |
| L2 | 内部 | 建议本地 | 传输 + 存储加密 |
| L3 | 敏感 | 强制本地 | 传输 + 存储加密 |
| L4 | 高敏感 | 强制本地 + 审计 | 全程加密 + 审计 |

### 2.3 数据类型映射
| 数据类型 | 驻留分类 | 敏感级别 | 说明 |
|----------|----------|----------|------|
| 用户身份信息 | 强制驻留 | L4 | 身份证、姓名、地址 |
| 用户认证信息 | 强制驻留 | L4 | 密码、密钥、令牌 |
| 用户对话数据 | 条件驻留 | L3 | 对话内容、上下文 |
| 用户记忆数据 | 条件驻留 | L3 | 向量数据、记忆 |
| 用户行为数据 | 条件驻留 | L2 | 操作日志、使用统计 |
| 系统配置数据 | 条件驻留 | L2 | 配置参数、策略 |
| 审计日志数据 | 强制驻留 | L3 | 操作审计、安全日志 |
| 公开数据 | 自由流动 | L1 | 公开文档、公告 |

## 三、区域驻留规则

| 数据类型 | 驻留要求 | 跨境规则 | 法规依据 |

| 一般数据 | 本地存储 | 合规后可跨境 | 网络安全法 |

| 非个人数据 | 无强制要求 | 自由流动 | - |

| 健康数据 | 强制本地 | 严格限制 | HIPAA |
| 金融数据 | 强制本地 | 合规后可跨境 | GLBA |

### 3.4 其他区域
| 区域 | 驻留要求 | 说明 |

## 四、数据存储规则

### 4.1 本地存储规则
| 规则 | 说明 | 执行要求 |
|------|------|----------|
| 主存储本地 | 主数据存储在本地区域 | 强制执行 |
| 副本本地 | 副本优先存储在本地 | 优先执行 |
| 加密存储 | 敏感数据加密存储 | 强制执行 |
| 访问控制 | 本地访问控制策略 | 强制执行 |
| 审计记录 | 本地审计记录 | 强制执行 |

### 4.2 跨区复制规则
| 数据类型 | 跨区复制 | 复制条件 | 复制范围 |
|----------|----------|----------|----------|
| 强制驻留数据 | 禁止 | - | - |
| 条件驻留数据 | 允许 | 审批后 | 同合规区域 |
| 自由流动数据 | 允许 | 无需审批 | 任意区域 |

### 4.3 同合规区域复制
| 区域组 | 包含区域 | 复制规则 |
|--------|----------|----------|
| 中国区 | CN-NORTH, CN-SOUTH | 允许复制 |
| 欧盟区 | EU-WEST, EU-EAST | 允许复制 |
| 美国区 | US-EAST, US-WEST | 允许复制 |
| 亚太区 | AP-SOUTHEAST, CN-HK | 条件复制 |

## 五、数据传输规则

### 5.1 区域内传输
| 传输类型 | 要求 | 说明 |
|----------|------|------|
| 同区域内传输 | 加密传输 | TLS 1.3 |
| 同区域内同步 | 加密同步 | 实时或定时 |
| 同区域内备份 | 加密备份 | 异地备份 |

### 5.2 跨区域传输
| 传输类型 | 数据类型 | 审批要求 | 技术要求 |
|----------|----------|----------|----------|
| 跨区查询 | 自由流动 | 无需审批 | 加密传输 |
| 跨区同步 | 条件驻留 | 技术负责人审批 | 加密 + 审计 |
| 跨区备份 | 条件驻留 | 安全负责人审批 | 加密 + 访问控制 |
| 跨区迁移 | 强制驻留 | 高管审批 + 合规评估 | 全程加密 + 审计 |

### 5.3 跨境传输
| 传输类型 | 数据类型 | 审批要求 | 合规要求 |
|----------|----------|----------|----------|
| 跨境查询 | 自由流动 | 无需审批 | 脱敏处理 |
| 跨境同步 | 条件驻留 | CISO 审批 | 保护措施 |
| 跨境备份 | 条件驻留 | CISO + 法务审批 | 充分性评估 |
| 跨境迁移 | 强制驻留 | 禁止 | - |

## 六、数据处理规则

|------|------|----------|

| 本地工具 | 使用本地工具服务 | 优先执行 |

| 远程计算 | 条件驻留 | 技术负责人审批 | 结果本地存储 |
| 远程分析 | 条件驻留 | 安全负责人审批 | 脱敏后可传输 |
| 远程存储 | 强制驻留 | 禁止 | - |

| 泛化处理 | 数值数据 | 范围泛化 |
| 合成数据 | 测试环境 | 生成替代数据 |
