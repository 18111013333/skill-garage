# REGULATED_ACTIONS.md - 受监管动作清单

## 目的
列出所有受监管动作，明确审批、日志、引用、复核要求。

## 适用范围
所有高风险动作、敏感操作、外部交互。

## 受监管动作分类

### 高风险外部写入
| 动作类型 | 风险等级 | 审批要求 | 日志要求 |
|----------|----------|----------|----------|
| 发送邮件 | 高 | 需审批 | 完整日志 |
| 发送短信 | 高 | 需审批 | 完整日志 |
| 发布内容 | 高 | 需审批 | 完整日志 |
| 修改外部系统 | 极高 | 双重审批 | 完整日志 + 截图 |
| 调用支付接口 | 极高 | 双重审批 | 完整日志 + 审计 |

**审批要求**:
- 审批级别: 部门负责人
- 审批时限: 4 小时
- 有效期: 单次

**日志要求**:
- 记录完整请求内容
- 记录响应结果
- 记录审批链
- 保留期限: 7 年

## 审批要求详解

### 审批级别
| 级别 | 审批人 | 适用场景 |
|------|--------|----------|
| 自动审批 | 系统自动 | 低风险、标准场景 |
| 单级审批 | 团队负责人 | 中等风险 |
| 双重审批 | 部门负责人 + 相关方 | 高风险 |
| 三重审批 | VP + 合规 + 安全 | 极高风险 |

### 审批流程
```
动作触发 → 风险评估 → 匹配审批级别 → 发起审批 → 审批决策 → 执行/阻断 → 记录归档
```

### 审批时限
| 风险等级 | 审批时限 | 超时处理 |
|----------|----------|----------|
| 中 | 2 小时 | 自动拒绝 |
| 高 | 4 小时 | 自动拒绝 |
| 极高 | 24 小时 | 自动拒绝 |

## 日志要求详解

### 日志内容
| 字段 | 说明 | 必需 |
|------|------|------|
| action_id | 动作标识 | 是 |
| action_type | 动作类型 | 是 |
| timestamp | 时间戳 | 是 |
| actor | 执行者 | 是 |
| target | 目标对象 | 是 |
| content | 动作内容 | 是 |
| result | 执行结果 | 是 |
| approval_chain | 审批链 | 按需 |
| risk_level | 风险等级 | 是 |

### 日志保留
| 风险等级 | 保留期限 | 存储要求 |
|----------|----------|----------|
| 中 | 3 年 | 标准存储 |
| 高 | 5 年 | 归档存储 |
| 极高 | 7 年 | 归档存储 + 备份 |

## 引用要求

### 引用内容
| 要求 | 说明 |
|------|------|
| 来源标识 | 标注数据/信息来源 |
| 版本标识 | 标注使用的版本 |
| 时间标识 | 标注获取时间 |
| 可追溯性 | 可追溯到原始来源 |

### 引用格式
```
[来源: <source_id>] [版本: <version>] [时间: <timestamp>]
```

## 复核要求

### 复核场景
| 场景 | 复核要求 | 复核人 |
|------|----------|--------|
| 高风险执行后 | 24 小时内复核 | 团队负责人 |
| 批量操作后 | 立即复核 | 数据负责人 |
| 跨租户操作后 | 48 小时内复核 | 合规负责人 |
| 安全相关后 | 立即复核 | 安全负责人 |

### 复核内容
- [ ] 执行结果符合预期
- [ ] 无异常副作用
- [ ] 日志记录完整
- [ ] 审批链完整
