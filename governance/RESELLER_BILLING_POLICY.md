# RESELLER_BILLING_POLICY.md - 转售计费规则

## 目的
定义代理商转售模式下的计费与结算规则。

---

## 计费关系

```
┌─────────────────────────────────────┐
│ 最终客户                             │
│ - 向代理商付费                       │
│ - 获取服务                           │
└─────────────────────────────────────┘
            ↓ 付费
┌─────────────────────────────────────┐
│ 代理商                               │
│ - 向客户收费                         │
│ - 向厂商结算                         │
│ - 赚取差价                           │
└─────────────────────────────────────┘
            ↓ 结算
┌─────────────────────────────────────┐
│ 厂商（平台）                         │
│ - 向代理商提供产品                   │
│ - 收取结算款                         │
└─────────────────────────────────────┘
```

---

## 价格体系

### 厂商价
| 产品 | 厂商价 |
|------|--------|
| 基础版 | ¥1000/月 |
| 专业版 | ¥3000/月 |
| 企业版 | ¥10000/月 |

### 转售价
| 等级 | 折扣权限 |
|------|----------|
| 注册级 | 无折扣 |
| 认证级 | 9折 |
| 高级级 | 8折 |
| 战略级 | 7折 |

---

## 结算周期

| 结算类型 | 周期 |
|----------|------|
| 首单结算 | T+30 |
| 续费结算 | 月度 |
| 服务结算 | 季度 |
| 退款结算 | 实时 |

---

## 对账方式

### 对账周期
| 周期 | 说明 |
|------|------|
| 日对账 | 交易明细 |
| 周对账 | 异常处理 |
| 月对账 | 正式结算 |

### 对账维度
| 维度 | 说明 |
|------|------|
| 按客户 | 客户维度 |
| 按产品 | 产品维度 |
| 按代理商 | 代理商维度 |
| 按时间 | 时间维度 |

---

## 欠费处理

| 阶段 | 处理 |
|------|------|
| 逾期7天 | 提醒通知 |
| 逾期14天 | 服务降级 |
| 逾期30天 | 服务暂停 |
| 逾期60天 | 合同终止 |

---

## 退款链路

```
客户退款申请
    ↓
代理商审核
    ↓
厂商确认
    ↓
┌─────────────────────────────────────┐
│ 退款计算                             │
│ - 客户退款金额                       │
│ - 代理商退回分成                     │
│ - 厂商退回结算款                     │
└─────────────────────────────┘
    ↓
退款执行
```
