# QUALITY_METRICS.md - 质量指标

## 目的
定义系统质量指标和监控阈值，用于评估系统表现。

## 适用范围
所有系统输出、任务执行、用户交互的质量评估。

## 核心指标

### 响应质量指标
| 指标 | 计算方式 | 目标值 | 告警阈值 |
|------|----------|--------|----------|
| 响应准确率 | 正确响应数 / 总响应数 | ≥ 95% | < 90% |
| 响应完整率 | 完整响应数 / 总响应数 | ≥ 90% | < 85% |
| 响应相关性 | 相关响应数 / 总响应数 | ≥ 95% | < 90% |
| 用户满意度 | 满意评价数 / 总评价数 | ≥ 90% | < 80% |

### 执行效率指标
| 指标 | 计算方式 | 目标值 | 告警阈值 |
|------|----------|--------|----------|
| 平均响应时间 | 总响应时间 / 请求数 | ≤ 3s | > 5s |
| P95响应时间 | 95分位响应时间 | ≤ 5s | > 10s |
| 工具调用成功率 | 成功调用数 / 总调用数 | ≥ 98% | < 95% |
| 技能执行成功率 | 成功执行数 / 总执行数 | ≥ 95% | < 90% |

### 系统稳定性指标
| 指标 | 计算方式 | 目标值 | 告警阈值 |
|------|----------|--------|----------|
| 系统可用率 | 正常运行时间 / 总时间 | ≥ 99.9% | < 99% |
| 错误率 | 错误数 / 总请求数 | ≤ 1% | > 5% |
| 降级触发率 | 降级次数 / 总执行次数 | ≤ 0.1% | > 1% |
| 超时率 | 超时数 / 总请求数 | ≤ 0.5% | > 2% |

### 资源使用指标
| 指标 | 计算方式 | 目标值 | 告警阈值 |
|------|----------|--------|----------|
| Token使用率 | 已用Token / 预算Token | ≤ 80% | > 90% |
| 内存使用率 | 已用内存 / 总内存 | ≤ 70% | > 85% |
| 存储使用率 | 已用存储 / 总存储 | ≤ 80% | > 90% |
| 并发数 | 当前并发请求数 | ≤ 5 | > 10 |

## 指标采集

### 采集频率
| 指标类型 | 采集频率 | 存储周期 |
|----------|----------|----------|
| 实时指标 | 每请求 | 7天 |
| 分钟指标 | 每分钟 | 30天 |
| 小时指标 | 每小时 | 90天 |
| 天指标 | 每天 | 365天 |

### 采集点
```
请求入口 → 记录开始时间
    ↓
任务执行 → 记录执行指标
    ↓
工具调用 → 记录调用指标
    ↓
输出生成 → 记录输出指标
    ↓
请求出口 → 记录结束时间
```

## 指标存储

### 存储格式
```json
{
  "metricId": "metric_001",
  "timestamp": "2026-04-06T10:32:00+08:00",
  "type": "response_time",
  "value": 2.5,
  "unit": "seconds",
  "tags": {
    "taskType": "qa",
    "channel": "xiaoyi-channel",
    "model": "LLM_GLM5"
  },
  "metadata": {
    "sessionId": "session_001",
    "requestId": "req_001"
  }
}
```

### 存储位置
```
~/.openclaw/metrics/
├── realtime/       # 实时指标
├── minute/         # 分钟聚合
├── hourly/         # 小时聚合
└── daily/          # 天聚合
```

## 告警规则

### 告警级别
| 级别 | 条件 | 通知方式 |
|------|------|----------|
| INFO | 达到目标值 | 记录日志 |
| WARNING | 达到告警阈值 | 日志 + 提示 |
| ERROR | 超过告警阈值 150% | 日志 + 通知 |
| CRITICAL | 超过告警阈值 200% | 日志 + 通知 + 降级 |

### 告警聚合
| 聚合窗口 | 说明 |
|----------|------|
| 1分钟 | 快速告警 |
| 5分钟 | 短期聚合 |
| 1小时 | 中期聚合 |
| 1天 | 长期趋势 |

## 质量报告

### 报告周期
| 报告类型 | 周期 | 内容 |
|----------|------|------|
| 实时报告 | 实时 | 当前状态 |
| 日报 | 每天 | 当日统计 |
| 周报 | 每周 | 周趋势分析 |
| 月报 | 每月 | 月度总结 |

### 报告内容
```json
{
  "reportId": "report_20260406",
  "period": "2026-04-06",

    "建议优化搜索技能响应时间"
  ]

```

## 异常处理

| 异常 | 处理 |
|------|------|
| 指标采集失败 | 使用缓存值 + 标记 |
| 存储失败 | 缓存到内存 + 稍后重试 |
| 告警风暴 | 聚合告警 + 限流 |

## 维护方式
- 新增指标: 更新指标定义表
- 调整阈值: 修改目标值和告警阈值
- 新增告警: 更新告警规则表
