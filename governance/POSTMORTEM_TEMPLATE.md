# POSTMORTEM_TEMPLATE.md - 事故复盘模板

**版本: V27.0**

## 目的
建立事故复盘模板，确保事故能变成长期治理资产，实现"复盘有标准、根因有分析、改进有跟踪"。

## 适用范围
所有事故复盘场景，包括：
- SEV0/SEV1 事故复盘
- 安全事件复盘
- 重大故障复盘
- 客户影响事件复盘

---

## 一、复盘模板结构

| 章节 | 必填 | 说明 |
|------|------|------|
| 摘要 | 是 | 事故概述 |
| 时间线 | 是 | 事故时间线 |
| 影响范围 | 是 | 影响范围说明 |
| 根因分析 | 是 | 根本原因分析 |
| 触发因素 | 是 | 触发因素分析 |
| 处置动作 | 是 | 处置动作记录 |
| 修复动作 | 是 | 修复动作记录 |
| 防再发动作 | 是 | 防止再发措施 |
| 经验教训 | 是 | 经验教训总结 |
| 附录 | 否 | 相关附件 |
