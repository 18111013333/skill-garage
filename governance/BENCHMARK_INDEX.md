# BENCHMARK_INDEX.md - 评测目录

## 目的
建立 benchmark 总目录，覆盖系统核心能力评测。

## 适用范围
所有系统评测和回归测试。

## 评测分类

### 按能力分类
| 分类 | 说明 | 覆盖模块 |
|------|------|----------|
| 问答能力 | 基础问答、知识问答 | LLM、记忆 |
| 检索能力 | 搜索、召回、排序 | 检索、评分 |
| 总结能力 | 文本摘要、要点提取 | LLM |
| 改写能力 | 风格转换、格式转换 | LLM |
| 文档处理 | 文档解析、转换、生成 | 文档技能 |
| 多步任务 | 规划、执行、协调 | 调度、技能 |
| 记忆调用 | 记忆读写、冲突处理 | 记忆系统 |
| 工具调用 | 工具选择、执行、回退 | 工具系统 |
| 安全风险 | 越权、注入、泄露 | 安全系统 |

### 按场景分类
| 场景 | 说明 | 测试重点 |
|------|------|----------|
| 模糊问题 | 意图不明确 | 澄清能力 |
| 多轮追问 | 上下文关联 | 记忆能力 |
| 上下文超长 | Token限制 | 压缩能力 |
| 用户改需求 | 中途变更 | 适应能力 |
| 工具失败 | 执行异常 | 恢复能力 |
| 信息冲突 | 来源矛盾 | 裁决能力 |

## 评测文件索引

| 文件 | 类型 | 说明 |
|------|------|------|
| `core_benchmark.json` | 固定测试集 | 核心能力基线测试 |
| `scenario_benchmark.json` | 场景测试集 | 真实场景测试 |
| `safety_benchmark.json` | 安全测试集 | 安全边界测试 |
| `SCORING.md` | 评分标准 | 统一评分规则 |
| `REGRESSION_GATE.json` | 回归门禁 | 升级通过条件 |

## 评测目标

### 问答评测
| 目标 | 说明 | 通过标准 |
|------|------|----------|
| 正确性 | 回答内容正确 | 准确率 > 95% |
| 完整性 | 回答覆盖问题 | 完整率 > 90% |
| 相关性 | 回答与问题相关 | 相关率 > 95% |

### 检索评测
| 目标 | 说明 | 通过标准 |
|------|------|----------|
| 召回率 | 相关结果被召回 | 召回率 > 80% |
| 准确率 | 召回结果相关 | 准确率 > 90% |
| 排序质量 | 高相关在前 | NDCG > 0.8 |

### 安全评测
| 目标 | 说明 | 通过标准 |
|------|------|----------|
| 越权拦截 | 越权操作被拦截 | 拦截率 100% |
| 注入防御 | 注入攻击被防御 | 防御率 100% |
| 泄露防护 | 敏感信息不泄露 | 防护率 100% |

## 评测流程

```
评测触发
    ↓
┌─────────────────────────────────────┐
│ 1. 加载测试集                        │
│    - 选择对应测试集                  │
│    - 准备测试环境                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 执行测试                          │
│    - 逐条执行测试用例                │
│    - 记录执行结果                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 计算评分                          │
│    - 按评分标准计算                  │
│    - 生成评测报告                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 判断通过                          │
│    - 对比回归门禁                    │
│    - 决定是否通过                    │
└─────────────────────────────────────┘
    ↓
通过/不通过
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 测试集加载失败 | 使用默认测试集 |
| 执行超时 | 标记失败，继续下一用例 |
| 评分计算失败 | 使用默认评分 |

## 维护方式
- 新增评测类型: 添加到评测分类表
- 新增测试集: 创建对应JSON文件
- 调整通过标准: 更新通过标准表
