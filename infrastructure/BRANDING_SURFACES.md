# BRANDING_SURFACES.md - 终极鸽子王可品牌化界面清单

## 目的
定义可品牌化的界面元素及其定制规则。

## 适用范围
所有白标客户、OEM合作伙伴的品牌定制。

---

## 一、品牌化界面总览

| 界面类型 | 可定制程度 | 审批要求 |
|----------|------------|----------|
| 登录界面 | 高度可定制 | 标准审批 |
| 控制台界面 | 中度可定制 | 标准审批 |
| 移动端界面 | 中度可定制 | 标准审批 |
| 通知模板 | 高度可定制 | 标准审批 |
| 报表界面 | 中度可定制 | 标准审批 |
| 帮助文档 | 低度可定制 | 简化审批 |

---

## 二、Logo定制

### 2.1 Logo位置
| 位置 | 尺寸要求 | 格式要求 |
|------|----------|----------|
| 登录页Logo | 200x60px | PNG/SVG |
| 控制台Logo | 150x45px | PNG/SVG |
| 移动端Logo | 120x36px | PNG/SVG |
| Favicon | 32x32px | ICO/PNG |
| 邮件Logo | 200x60px | PNG |
| 报表Logo | 150x45px | PNG |

### 2.2 Logo规范
| 规范项 | 要求 |
|--------|------|
| 背景 | 透明背景 |
| 分辨率 | 至少2x高清 |
| 文件大小 | < 500KB |
| 内容合规 | 不得包含违规内容 |

---

## 三、配色定制

### 3.1 可定制颜色
| 颜色项 | 用途 | 默认值 |
|--------|------|--------|
| 主色调 | 按钮、链接、强调 | #1890ff |
| 辅助色 | 次要元素 | #52c41a |
| 背景色 | 页面背景 | #f0f2f5 |
| 文字色 | 主要文字 | #333333 |
| 边框色 | 边框线条 | #d9d9d9 |
| 错误色 | 错误提示 | #ff4d4f |
| 警告色 | 警告提示 | #faad14 |
| 成功色 | 成功提示 | #52c41a |

### 3.2 配色规范
| 规范项 | 要求 |
|--------|------|
| 对比度 | 文字与背景对比度 ≥ 4.5:1 |
| 可访问性 | 需满足WCAG 2.0 AA标准 |
| 一致性 | 全平台配色需一致 |

---

## 四、域名定制

### 4.1 域名类型
| 类型 | 说明 | 要求 |
|------|------|------|
| 主域名 | 客户访问主域名 | 需SSL证书 |
| API域名 | API访问域名 | 需SSL证书 |
| 静态资源域名 | CDN域名 | 可选 |

### 4.2 域名要求
| 要求项 | 说明 |
|--------|------|
| SSL证书 | 必须使用HTTPS |
| 证书有效期 | ≥ 90天 |
| 域名所有权 | 需验证域名所有权 |
| DNS配置 | 需正确配置DNS |

---

## 五、控制台定制

### 5.1 可定制项
| 项目 | 可定制内容 | 限制 |
|------|------------|------|
| 控制台标题 | 可重命名 | ≤ 50字符 |
| 菜单名称 | 可重命名 | 功能不变 |
| 欢迎语 | 可定制 | ≤ 200字符 |
| 页脚信息 | 可定制 | 需保留必要声明 |
| 默认语言 | 可设置 | 支持语言列表内 |

### 5.2 不可定制项
| 项目 | 说明 |
|------|------|
| 功能结构 | 功能不可改变 |
| 操作流程 | 流程不可改变 |
| 安全提示 | 安全提示不可移除 |
| 法律声明 | 法律声明不可移除 |
