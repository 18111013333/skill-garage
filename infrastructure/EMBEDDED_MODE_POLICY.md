# EMBEDDED_MODE_POLICY.md - 嵌入式集成模式

## 目的
定义平台嵌入第三方产品的规则与边界。

---

## 嵌入模式

### 1. iframe 嵌入模式
| 特性 | 说明 |
|------|------|
| **集成方式** | iframe嵌入宿主页面 |
| **权限边界** | 受宿主页面限制 |
| **数据边界** | 独立域，数据隔离 |
| **审计边界** | 独立审计日志 |
| **适用场景** | 轻量集成、快速上线 |

### 2. SDK 嵌入模式
| 特性 | 说明 |
|------|------|
| **集成方式** | SDK集成到宿主应用 |
| **权限边界** | 受SDK权限控制 |
| **数据边界** | 按SDK配置隔离 |
| **审计边界** | SDK内置审计 |
| **适用场景** | 深度集成、移动端 |

### 3. API-First 模式
| 特性 | 说明 |
|------|------|
| **集成方式** | 纯API调用 |
| **权限边界** | API密钥控制 |
| **数据边界** | 按API权限隔离 |
| **审计边界** | API调用日志 |
| **适用场景** | 后端集成、自动化 |

### 4. Embedded Workflow 模式
| 特性 | 说明 |
|------|------|
| **集成方式** | 工作流嵌入宿主流程 |
| **权限边界** | 流程权限控制 |
| **数据边界** | 流程数据隔离 |
| **审计边界** | 流程执行日志 |
| **适用场景** | 业务流程集成 |

---

## 权限边界

### 宿主系统权限
| 权限 | 说明 |
|------|------|
| 界面控制 | 宿主控制UI展示 |
| 用户认证 | 宿主管理用户 |
| 会话管理 | 宿主管理会话 |

### 平台权限
| 权限 | 说明 |
|------|------|
| 数据处理 | 平台处理数据 |
| AI能力 | 平台提供AI |
| 审计追踪 | 平台记录审计 |

---

## 数据边界

### 数据归属
| 数据类型 | 归属 |
|----------|------|
| 用户数据 | 宿主系统 |
| 业务数据 | 共同归属 |
| AI生成数据 | 平台归属 |
| 审计数据 | 平台归属 |

### 数据隔离
| 隔离级别 | 说明 |
|----------|------|
| 租户隔离 | 不同租户数据隔离 |
| 宿主隔离 | 不同宿主数据隔离 |
| 环境隔离 | 生产/测试隔离 |

---

## 审计边界

### 宿主审计
- 用户操作日志
- 界面访问日志
- 会话管理日志

### 平台审计
- API调用日志
- 数据处理日志
- AI执行日志
- 权限变更日志

---

## 宿主系统责任

| 责任项 | 说明 |
|--------|------|
| 用户管理 | 用户创建、认证、授权 |
| 界面展示 | UI渲染、交互处理 |
| 会话安全 | 会话管理、安全防护 |
| 合规披露 | 向终端用户披露数据处理 |
