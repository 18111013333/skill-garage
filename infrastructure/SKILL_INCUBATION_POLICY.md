# SKILL_INCUBATION_POLICY.md - 技能孵化策略

**版本: V28.0**

## 目的
定义未成熟技能的孵化流程，确保新技能有明确的成长路径和保护机制。

## 适用范围
所有处于 incubating、candidate、beta 阶段的技能。

---

## 一、孵化原则

| 原则 | 说明 |
|------|------|
| 保护优先 | 孵化期技能不参加普通低频删除 |
| 验证驱动 | 必须通过验证才能晋升 |
| Owner负责 | 每个孵化技能必须有明确Owner |
| 黄金路径绑定 | 必须有候选黄金路径 |
| 定期评审 | 定期评审孵化进展 |

---

## 二、孵化阶段

### 2.1 incubating 阶段

| 项目 | 要求 |
|------|------|
| 保护期 | 30天 |
| 目标 | 完成可用性验证 |
| 验证方式 | shadow mode / 离线测试 |
| 路由权限 | 禁止自动路由 |
| 评审频率 | 每周 |

### 2.2 candidate 阶段

| 项目 | 要求 |
|------|------|
| 保护期 | 30天 |
| 目标 | 完成受控试用和对照实验 |
| 验证方式 | 定向beta路由 / 人工指定调用 |
| 路由权限 | 仅指定场景 |
| 评审频率 | 每周 |

### 2.3 beta 阶段

| 项目 | 要求 |
|------|------|
| 保护期 | 45天 |
| 目标 | 完成真实低风险任务验证 |
| 验证方式 | 低风险场景真实调用 |
| 路由权限 | 低风险路由 |
| 评审频率 | 每两周 |

## 四、验证方式

### 4.1 Shadow Mode

| 项目 | 说明 |
|------|------|
| 定义 | 影子模式，不实际执行，只记录 |
| 适用 | incubating 阶段 |
| 方式 | 并行运行，记录结果但不返回 |
| 评估 | 对比影子结果与实际结果 |

### 4.2 定向 Beta 路由

| 项目 | 说明 |
|------|------|
| 定义 | 仅对特定用户/场景开放 |
| 适用 | candidate 阶段 |
| 方式 | 配置特定路由规则 |
| 评估 | 收集定向用户反馈 |

### 4.3 人工指定调用

| 项目 | 说明 |
|------|------|
| 定义 | 由人工显式调用 |
| 适用 | incubating/candidate 阶段 |
| 方式 | 通过显式命令调用 |
| 评估 | 人工评估结果质量 |

## 五、孵化评审流程

```
评审触发
    ↓
┌─────────────────────────────────────┐
│ 1. 数据收集                          │
│    - 收集验证数据                    │
│    - 收集反馈数据                    │
│    - 收集问题记录                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 成果评估                          │
│    - 评估是否达成目标                │
│    - 检查成功标准                    │
│    - 识别问题和风险                  │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 决策                              │
│    - 晋升：进入下一阶段              │
│    - 延长：继续当前阶段              │
│    - 降级：返回上一阶段              │
│    - 废弃：进入废弃流程              │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 状态更新                          │
│    - 更新孵化状态                    │
│    - 通知相关方                      │
│    - 调整验证方式                    │
└─────────────────────────────────────┘
```

## 六、孵化技能删除条件

孵化技能只有在以下条件全部满足时才能删除：

| 条件 | 说明 |
|------|------|
| 完整周期 | 已完成至少一个完整孵化周期 |
| 受控评测 | 已做过至少一次受控评测 |
| 无增益 | 没有显示出任何明确增益 |
| 无Owner | 没有 Owner 愿意继续负责 |
| 无场景 | 没有明确接入场景 |
| 有替代 | 有替代能力可以覆盖其作用 |
