# CHANGELOG_CONSUMER_POLICY.md - 变更通知规则

## 目的
定义面向外部开发者的变更通知规则，确保开发生态跟上平台变化。

## 适用范围
所有影响外部开发者的平台变更。

---

## 一、变更分类

### 1.1 变更类型
| 类型 | 说明 | 通知要求 |
|------|------|----------|
| 新功能 | 新增功能 | 公告 |
| 功能增强 | 功能改进 | 公告 |
| Bug 修复 | 问题修复 | 可选通知 |
| API 变更 | 接口变更 | 强制通知 |
| 废弃公告 | 功能废弃 | 强制通知 |
| 下线公告 | 功能下线 | 强制通知 |
| 安全更新 | 安全修复 | 强制通知 |

### 1.2 影响级别
| 级别 | 说明 | 通知提前量 |
|------|------|------------|
| 低 | 不影响现有使用 | 无需提前 |
| 中 | 需要调整配置 | 1 周 |
| 高 | 需要修改代码 | 1 个月 |
| 紧急 | 安全/合规问题 | 即时 |

---

## 二、必须公告的变更

| 变更 | 公告要求 |

| 版本废弃 | 提前 3 个月 |
| 版本停止维护 | 提前 6 个月 |

### 2.3 连接器变更
| 变更 | 公告要求 |
|------|----------|
| 新连接器 | 发布时公告 |
| 连接器更新 | 发布时公告 |
| 连接器废弃 | 提前 3 个月 |
| 连接器下线 | 提前 6 个月 |

### 2.4 模板/资产变更
| 变更 | 公告要求 |
|------|----------|
| 新资产上架 | 发布时公告 |
| 资产更新 | 发布时公告 |
| 资产下架 | 提前 1 个月 |

## 三、公告提前量

### 3.1 提前时间
| 变更类型 | Stable | Beta | Experimental |
|----------|--------|------|--------------|
| 新功能 | 发布时 | 发布时 | 无需 |
| 功能增强 | 发布时 | 发布时 | 无需 |
| 不兼容变更 | 1 个月 | 2 周 | 无需 |
| 废弃公告 | 6 个月 | 3 个月 | 1 个月 |
| 下线公告 | 12 个月 | 6 个月 | 1 个月 |
| 安全更新 | 即时 | 即时 | 即时 |

### 3.2 提前量计算
```
公告时间 = 执行时间 - 提前量
```

### 3.3 提前量调整
| 情况 | 调整 |
|------|------|
| 安全紧急 | 可缩短 |
| 用户请求 | 可延长 |
| 合规要求 | 按法规 |

---

## 四、通知渠道

### 4.1 渠道优先级
| 渠道 | 优先级 | 用途 |
|------|--------|------|
| 开发者邮件 | 高 | 直达通知 |
| 控制台通知 | 高 | 登录提示 |
| 文档更新 | 高 | 详细说明 |
| Changelog | 中 | 变更记录 |
| 社区公告 | 中 | 社区通知 |
| 社交媒体 | 低 | 广泛传播 |

### 4.2 渠道选择
| 变更级别 | 渠道 |
|----------|------|
| 紧急 | 邮件 + 控制台 + 文档 |
| 高 | 邮件 + 控制台 + 文档 + Changelog |
| 中 | 控制台 + 文档 + Changelog |
| 低 | 文档 + Changelog |

### 4.3 通知频率
| 阶段 | 频率 |
|------|------|
| 首次公告 | 1 次 |
| 提醒期 | 每月 1 次 |
| 最终警告 | 每周 1 次 |

---

## 五、文档同步

### 5.1 文档更新要求
| 文档类型 | 更新要求 |
|----------|----------|
| API 文档 | 同步更新 |
| SDK 文档 | 同步更新 |
| 迁移指南 | 废弃时提供 |
| 示例代码 | 同步更新 |

### 5.2 文档更新流程
```
变更决定 → 更新文档 → 审核文档 → 发布文档 → 公告通知
```

### 5.3 文档版本
```
保留历史版本
标记废弃版本
提供版本切换
```

## 六、变更联动

### 6.1 联动关系
| 变更 | 联动更新 |
|------|----------|
| API 变更 | SDK + 文档 + 示例 |
| SDK 变更 | 文档 + 示例 |
| 连接器变更 | 文档 + 配置模板 |
| 模板变更 | 文档 + 示例 |

### 6.2 联动检查
```
变更发布 → 检查联动项 → 更新联动内容 → 验证完整性
```

### 6.3 联动时间
| 联动项 | 更新时间 |
|--------|----------|
| SDK | 与 API 同步 |
| 文档 | 与变更同步 |
| 示例 | 与 SDK 同步 |

---

## 七、通知内容

### 7.1 通知内容要求
| 内容 | 说明 |
|------|------|
| 变更摘要 | 简要说明变更 |
| 影响范围 | 影响的用户/接口 |
| 时间安排 | 变更时间安排 |
| 行动建议 | 用户需要做什么 |
| 相关链接 | 文档/迁移指南链接 |

### 7.2 通知模板
```markdown
# [变更类型] 变更通知

## 变更摘要
简要说明变更内容

## 影响范围
- 影响的 API: xxx
- 影响的 SDK: xxx
- 影响的用户: xxx

## 时间安排
- 公告时间: 2026-04-07
- 生效时间: 2026-07-07
- 过渡期: 3 个月

## 行动建议
1. 检查是否使用受影响功能
2. 按迁移指南更新代码
3. 测试验证

## 相关链接
- 迁移指南: https://...
- 文档更新: https://...
- 技术支持: support@openclaw.ai
```
