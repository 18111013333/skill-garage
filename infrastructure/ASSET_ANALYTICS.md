# ASSET_ANALYTICS.md - 资产使用分析规则

## 目的
定义资产使用分析规则，了解资产价值与风险。

## 适用范围
所有资产市场的资产分析。

---

## 一、分析指标

### 1.1 核心指标
| 指标 | 说明 | 计算方式 |
|------|------|----------|
| 安装率 | 安装/展示 | 安装数 / 展示数 |
| 启用率 | 启用/安装 | 启用数 / 安装数 |
| 成功率 | 成功/使用 | 成功数 / 使用数 |
| 回滚率 | 回滚/安装 | 回滚数 / 安装数 |
| 卸载率 | 卸载/安装 | 卸载数 / 安装数 |

### 1.2 使用指标
| 指标 | 说明 |
|------|------|
| 日活用户 | 每日使用用户数 |

| 月活用户 | 每月使用用户数 |
| 使用频次 | 平均使用次数 |
| 使用时长 | 平均使用时长 |

### 1.3 质量指标
| 指标 | 说明 |
|------|------|
| 错误率 | 使用错误率 |
| 崩溃率 | 崩溃率 |

## 三、高风险资产监控

### 3.1 高风险判定
| 条件 | 说明 |
|------|------|
| 错误率高 | 错误率 > 10% |
| 回滚率高 | 回滚率 > 20% |
| 卸载率高 | 卸载率 > 30% |
| 评分低 | 评分 < 3.0 |
| 投诉多 | 投诉 > 10/月 |

### 3.2 监控指标
| 指标 | 告警阈值 |
|------|----------|
| 错误率 | > 10% |
| 回滚率 | > 20% |
| 卸载率 | > 30% |
| 投诉数 | > 10/月 |

### 3.3 高风险处理
```
检测高风险 → 评估风险 → 通知作者 → 限期整改 → 下架处理
    ↓            ↓           ↓           ↓           ↓
  系统         审核        作者        作者        系统
```

## 五、价值评估

### 5.1 价值指标
| 指标 | 权重 | 说明 |
|------|------|------|
| 安装量 | 20% | 安装数量 |
| 使用量 | 30% | 使用频次 |
| 评分 | 20% | 用户评分 |
| 留存率 | 20% | 用户留存 |
| 收入 | 10% | 付费收入 |

### 5.2 价值评分
```
价值分 = 安装量分 × 0.2 + 使用量分 × 0.3 + 评分分 × 0.2 + 留存率分 × 0.2 + 收入分 × 0.1
```

### 5.3 价值等级
| 等级 | 分数 | 说明 |
|------|------|------|
| 金牌 | > 90 | 高价值资产 |
| 银牌 | 70-90 | 中高价值 |
| 铜牌 | 50-70 | 中等价值 |
| 普通 | < 50 | 一般价值 |
