# CUSTOM_DOMAIN_POLICY.md - 终极鸽子王自定义域名规则

## 目的
定义自定义域名的绑定、证书、管理规则。

## 适用范围
所有白标客户、OEM合作伙伴的自定义域名配置。

---

## 一、域名绑定

### 1.1 绑定流程
```
域名绑定流程
    ↓
┌─────────────────────────────────────┐
│ 1. 域名验证                          │
│    - 提交域名                        │
│    - 添加验证记录                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. DNS配置                          │
│    - 配置CNAME记录                  │
│    - 等待DNS生效                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 证书配置                          │
│    - 上传SSL证书                    │
│    - 或使用平台证书                  │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 绑定生效                          │
│    - 测试访问                        │
│    - 正式生效                        │
└─────────────────────────────────────┘
```

### 1.2 DNS配置要求
| 记录类型 | 配置内容 | 说明 |
|----------|----------|------|
| CNAME | 指向平台域名 | 主域名配置 |
| TXT | 验证记录 | 域名所有权验证 |
| CAA | 证书颁发机构 | 可选 |

---

## 二、证书要求

### 2.1 证书类型
| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 平台托管证书 | 平台自动管理 | 标准场景 |
| 自定义证书 | 客户自行管理 | 高安全要求 |
| 通配符证书 | 覆盖子域名 | 多子域名场景 |

### 2.2 证书要求
| 要求项 | 说明 |
|--------|------|
| 加密类型 | TLS 1.2+ |
| 证书有效期 | ≥ 90天 |
| 证书链完整 | 需包含完整证书链 |
| 域名匹配 | 证书域名需与绑定域名一致 |

### 2.3 证书更新
| 场景 | 处理方式 |
|------|----------|
| 平台托管 | 自动更新 |
| 自定义证书 | 需手动更新 |
| 即将过期 | 提前30天提醒 |

## 三、区域限制

### 3.1 区域配置
| 区域 | 域名要求 | 说明 |
|------|----------|------|
| 中国大陆 | 需ICP备案 | 合规要求 |
| 其他区域 | 无特殊要求 | 标准配置 |

### 3.2 备案要求
| 要求项 | 说明 |
|--------|------|
| ICP备案 | 中国大陆域名需备案 |
| 备案主体 | 需与租户主体一致 |
| 备案号展示 | 需在页面展示备案号 |

---

## 四、审计记录

### 4.1 审计内容
| 审计项 | 记录内容 |
|--------|----------|
| 域名绑定 | 绑定时间、操作人 |
| 证书变更 | 变更时间、证书信息 |
| DNS变更 | 变更时间、变更内容 |
| 访问日志 | 访问时间、来源IP |

### 4.2 审计保留
| 记录类型 | 保留期限 |
|----------|----------|
| 绑定记录 | 永久 |
| 证书记录 | 证书过期后1年 |
| 访问日志 | 90天 |

---

## 五、撤销与迁移

### 5.1 撤销条件
| 条件 | 处理 |
|------|------|
| 客户申请 | 按流程撤销 |
| 合同终止 | 自动撤销 |
| 安全风险 | 立即撤销 |

### 5.2 撤销流程
```
域名撤销流程
    ↓
┌─────────────────────────────────────┐
│ 1. 撤销申请                          │
│    - 提交撤销申请                    │
│    - 说明撤销原因                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 数据处理                          │
│    - 导出必要数据                    │
│    - 通知相关用户                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 域名解绑                          │
│    - 解除域名绑定                    │
│    - 撤销证书                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 记录归档                          │
│    - 归档相关记录                    │
│    - 通知完成                        │
└─────────────────────────────────────┘
```

### 5.3 迁移规则
| 迁移场景 | 处理方式 |
|----------|----------|
| 域名更换 | 先绑定新域名，再解绑旧域名 |
| 租户迁移 | 需重新绑定域名 |
| 平台迁移 | 需重新配置DNS |
