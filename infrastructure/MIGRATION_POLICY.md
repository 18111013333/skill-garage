# MIGRATION_POLICY.md - 数据迁移策略

## 目的
定义配置结构、记忆schema、技能目录、工作流注册表升级时的数据迁移规则。

## 适用范围
所有系统演进时的数据迁移。

## 迁移场景

### 配置结构变更
| 场景 | 说明 |
|------|------|
| 字段新增 | 新增配置字段 |
| 字段删除 | 删除配置字段 |
| 字段重命名 | 重命名字段 |
| 字段类型变更 | 改变字段类型 |

### 记忆Schema变更
| 场景 | 说明 |
|------|------|
| Schema升级 | 记忆结构升级 |
| 字段迁移 | 字段迁移转换 |
| 数据格式转换 | 数据格式转换 |

### 技能目录变更
| 场景 | 说明 |
|------|------|
| 技能重命名 | 技能名称变更 |
| 技能合并 | 多技能合并 |
| 技能拆分 | 技能拆分 |

### 工作流注册表变更
| 场景 | 说明 |
|------|------|

| 工作流合并 | 多工作流合并 |

## 兼容策略

### 向后兼容
| 策略 | 说明 |
|------|------|
| 保留旧字段 | 旧字段保留，标记废弃 |
| 默认值填充 | 新字段使用默认值 |
| 双写支持 | 同时支持新旧格式 |

### 向前兼容
| 策略 | 说明 |
|------|------|
| 忽略未知字段 | 忽略新版本字段 |
| 降级处理 | 降级到兼容模式 |
| 版本协商 | 版本协商处理 |

### 不兼容变更
| 策略 | 说明 |
|------|------|
| 强制升级 | 强制升级数据 |
| 数据转换 | 转换数据格式 |
| 清空重建 | 清空后重建 |

## 废弃字段处理

### 废弃流程
```
字段废弃
    ↓
┌─────────────────────────────────────┐
│ 1. 标记废弃                          │
│    - 添加deprecated标记              │
│    - 记录废弃版本                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 兼容期维护                        │
│    - 保持向后兼容                    │
│    - 输出废弃警告                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 移除字段                          │
│    - 兼容期结束后移除                │
│    - 清理相关代码                    │
└─────────────────────────────────────┘
```

### 废弃周期
| 阶段 | 周期 | 说明 |
|------|------|------|
| 标记废弃 | 立即 | 添加废弃标记 |
| 兼容期 | 3个月 | 保持兼容 |
| 移除 | 3个月后 | 移除字段 |

## 迁移流程

### 迁移前准备
```
1. 备份数据
2. 验证迁移脚本
3. 准备回滚方案
4. 通知相关人员
```

### 迁移执行
```
┌─────────────────────────────────────┐
│ 1. 创建备份                          │
│    - 全量备份                        │
│    - 增量备份                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 执行迁移                          │
│    - 数据转换                        │
│    - 数据验证                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 验证结果                          │
│    - 数据完整性检查                  │
│    - 功能验证                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 清理备份                          │
│    - 保留期后清理                    │
│    - 归档备份                        │
└─────────────────────────────────────┘
```

### 迁移失败回滚
```
迁移失败
    ↓
┌─────────────────────────────────────┐
│ 1. 停止迁移                          │
│    - 暂停所有操作                    │
│    - 记录失败点                      │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 恢复备份                          │
│    - 恢复数据备份                    │
│    - 恢复配置备份                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 验证恢复                          │
│    - 验证数据完整性                  │
│    - 验证系统功能                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 记录失败原因                      │
│    - 分析失败原因                    │
│    - 制定修复方案                    │
└─────────────────────────────────────┘
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 迁移失败 | 回滚到备份 |
| 数据丢失 | 从备份恢复 |
| 格式错误 | 转换或跳过 |
