# SANDBOX_PROVISIONING.md - 沙箱开通流程

## 目的
定义客户隔离测试环境的标准化开通流程。

---

## 开通触发条件

| 触发场景 | 沙箱类型 | 自动/手动 |
|----------|----------|----------|
| 项目启动 | 开发沙箱 | 自动 |
| UAT阶段 | UAT沙箱 | 手动审批 |
| 上线前 | 预演沙箱 | 手动审批 |
| 问题复现 | 调试沙箱 | 按需 |

---

## 默认配置

### 开发沙箱默认配置
| 配置项 | 默认值 |
|--------|--------|
| 计算资源 | 2核4G |
| 存储空间 | 20GB |
| 并发用户 | 5 |
| 数据模式 | 合成数据 |
| 连接器模式 | 模拟 |
| 有效期 | 30天 |

### UAT沙箱默认配置
| 配置项 | 默认值 |
|--------|--------|
| 计算资源 | 4核8G |
| 存储空间 | 50GB |
| 并发用户 | 20 |
| 数据模式 | 脱敏数据 |
| 连接器模式 | 测试 |
| 有效期 | 60天 |

### 预演沙箱默认配置
| 配置项 | 默认值 |
|--------|--------|
| 计算资源 | 与生产一致 |
| 存储空间 | 与生产一致 |
| 并发用户 | 与生产一致 |
| 数据模式 | 生产副本 |
| 连接器模式 | 生产镜像 |
| 有效期 | 14天 |

---

## 默认角色

| 角色 | 权限 |
|------|------|
| 沙箱管理员 | 全部权限 |
| 测试用户 | 测试权限 |
| 观察者 | 只读权限 |

## 开通流程

```
沙箱申请
    ↓
┌─────────────────────────────────────┐
│ 资源评估                             │
│ - 计算资源需求                       │
│ - 存储需求                           │
│ - 网络需求                           │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 环境创建                             │
│ - 租户隔离                           │
│ - 网络隔离                           │
│ - 资源分配                           │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 配置部署                             │
│ - 基础配置                           │
│ - 连接器配置                         │
│ - 数据初始化                         │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 验证测试                             │
│ - 连接性测试                         │
│ - 功能验证                           │
│ - 性能基线                           │
└─────────────────────────────┘
    ↓
沙箱交付
```

## 数据初始化

### 合成数据模式
| 数据类型 | 生成规则 |
|----------|----------|
| 用户数据 | 随机生成 |
| 业务数据 | 模板生成 |
| 配置数据 | 标准模板 |

### 脱敏数据模式
| 数据类型 | 脱敏规则 |
|----------|----------|
| PII数据 | 掩码处理 |
| 敏感数据 | 替换处理 |
| 业务数据 | 保留结构 |

### 生产副本模式
| 数据类型 | 处理规则 |
|----------|----------|
| 全量数据 | 时间点快照 |
| 敏感字段 | 自动脱敏 |
| 访问日志 | 独立记录 |
