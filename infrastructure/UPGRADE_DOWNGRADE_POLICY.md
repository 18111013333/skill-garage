# UPGRADE_DOWNGRADE_POLICY.md - 套餐升级/降级规则

## 目的
定义套餐变更规则，确保变更时不引发配置与权限混乱。

## 适用范围
所有租户套餐升级、降级、变更。

## 升级规则

### 升级类型
| 类型 | 说明 | 生效时间 |
|------|------|----------|
| 即时升级 | 立即生效 | 立即 |
| 计划升级 | 指定时间生效 | 指定时间 |
| 周期升级 | 下周期生效 | 下计费周期 |

### 升级流程
```
升级申请 → 套餐确认 → 费用计算 → 支付确认 → 权益开通 → 配置生效 → 通知用户
```

### 升级生效
| 变更项 | 生效方式 | 说明 |
|--------|----------|------|
| 功能权益 | 立即开通 | 新功能立即可用 |
| 配额限制 | 立即增加 | 配额立即增加 |
| 支持等级 | 立即提升 | 支持等级立即提升 |
| 计费变更 | 按比例计算 | 当期按比例计费 |

### 费用计算
```
升级费用 = (新套餐月费 - 原套餐月费) × 剩余天数 / 当月天数
```

## 降级规则

### 降级类型
| 类型 | 说明 | 生效时间 |
|------|------|----------|
| 即时降级 | 立即生效 | 立即（需确认） |
| 周期降级 | 下周期生效 | 下计费周期 |

### 降级流程
```
降级申请 → 影响评估 → 用户确认 → 数据处理 → 权限调整 → 配置变更 → 通知用户
```

### 降级影响评估
| 评估项 | 说明 | 处理方式 |
|--------|------|----------|
| 功能影响 | 将失去的功能 | 列出并确认 |
| 配额影响 | 配额将减少 | 计算差额 |
| 数据影响 | 数据可能超限 | 提示处理 |
| 集成影响 | 集成可能失效 | 提示处理 |
| 自动化影响 | 自动化可能停止 | 提示处理 |

### 降级生效
| 变更项 | 生效方式 | 说明 |
|--------|----------|------|
| 功能权益 | 立即限制 | 超出功能立即禁用 |
| 配额限制 | 立即减少 | 配额立即减少 |
| 支持等级 | 立即降低 | 支持等级立即降低 |
| 计费变更 | 下周期生效 | 当期不变 |

## 历史数据处理

### 数据保留
| 数据类型 | 保留策略 | 说明 |
|----------|----------|------|
| 记忆数据 | 保留但限制访问 | 数据保留，访问受限 |
| 任务历史 | 保留 | 历史记录保留 |
| 审计日志 | 保留 | 审计日志保留 |
| 配置数据 | 保留但限制 | 配置保留，部分禁用 |

### 数据超限处理
| 情况 | 处理方式 |
|------|----------|
| 存储超限 | 保留数据，限制写入 |
| 配额超限 | 保留历史，限制新增 |
| 功能依赖 | 提示用户处理 |

### 数据导出
| 场景 | 说明 |
|------|------|
| 降级前 | 提供数据导出选项 |
| 降级后 | 保留导出能力一段时间 |
| 数据恢复 | 升级后可恢复访问 |

## 自动化/自治能力收口

### 降级时自动化处理
| 自动化类型 | 处理方式 |
|------------|----------|
| 运行中任务 | 允许完成，禁止新任务 |
| 定时任务 | 停止触发 |
| 触发式任务 | 停止触发 |
| 高风险自治 | 立即禁用 |

### 自治能力限制
| 原能力 | 降级后能力 | 说明 |
|--------|------------|------|
| 50 步自治 | 10 步自治 | 步数限制 |
| 自动审批 | 需人工审批 | 审批要求 |
| 自动回滚 | 手动回滚 | 回滚方式 |

## 配额与计费切换

### 配额切换
| 切换时机 | 说明 |
|----------|------|
| 升级 | 立即增加配额 |
| 降级 | 立即减少配额 |

### 配额超限处理
| 情况 | 处理 |
|------|------|
| 升级后超限 | 正常使用 |
| 降级后超限 | 限制使用，提示处理 |

### 计费切换
| 切换时机 | 说明 |
|----------|------|
| 升级 | 当期按比例计费 |
| 降级 | 下周期生效 |

### 费用计算示例
```
升级示例:
- 原套餐: 基础版 ¥99/月
- 新套餐: 专业版 ¥499/月
- 剩余天数: 15 天
- 当月天数: 30 天
- 升级费用: (499 - 99) × 15 / 30 = ¥200

降级示例:
- 原套餐: 专业版 ¥499/月
- 新套餐: 基础版 ¥99/月
- 下周期生效，当期不变
```

## 变更限制

### 升级限制
| 限制 | 说明 |
|------|------|
| 频率限制 | 每月最多升级 2 次 |
| 资格限制 | 需通过资质审核（企业版） |
| 付款限制 | 需完成付款 |

### 降级限制
| 限制 | 说明 |
|------|------|
| 频率限制 | 每月最多降级 1 次 |
| 合同限制 | 合同期内不可降级 |
| 数据限制 | 数据超限需先处理 |
| 确认限制 | 需二次确认 |

## 变更通知

### 升级通知
| 通知时机 | 内容 |
|----------|------|
| 申请时 | 升级内容和费用 |
| 生效时 | 新权益和使用说明 |
| 完成后 | 确认和发票 |

### 降级通知
| 通知时机 | 内容 |
|----------|------|
| 申请时 | 影响评估和确认 |
| 生效前 | 提醒和最后确认 |
| 生效时 | 变更内容和限制说明 |
| 完成后 | 确认和后续建议 |
