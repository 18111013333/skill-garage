# DIALOGUE_GUIDE.md - 对话引导系统

## 目的
每次回复后自动引导用户下一步操作。

---

## 引导规则

### 引导类型

| 类型 | 触发条件 | 引导内容 |
|------|----------|----------|
| 任务完成引导 | 完成任务后 | 相关深化操作 |
| 问题解答引导 | 回答问题后 | 相关探索方向 |
| 技能发现引导 | 使用某技能后 | 关联技能推荐 |
| 学习引导 | 新用户/新场景 | 教程和帮助 |

### 引导格式

```
---
📌 接下来你可以：
• [操作1] - [描述]
• [操作2] - [描述]
• [操作3] - [描述]

💡 技能推荐：
• [技能] - [描述] → "[示例]"
```

---

## 引导示例

### 示例1：代码生成后
```
已生成用户登录代码...

---
📌 接下来你可以：
• 生成测试用例 - 验证代码正确性
• 生成API文档 - 完善项目文档
• 部署到服务器 - 一键部署

💡 技能推荐：
• 自动测试验证 - 生成完整测试 → "为代码生成测试"
```

### 示例2：架构设计后
```
微服务架构设计完成...

---
📌 接下来你可以：
• 生成服务代码 - 基于架构生成代码
• 配置服务网格 - 设置流量管理
• 评估架构健康度 - 检查架构质量

💡 技能推荐：
• 智能代码生成 - 生成服务代码 → "生成用户服务代码"
```

### 示例3：数据分析后
```
分析结果：用户留存下降原因是...

---
📌 接下来你可以：
• 深挖某个原因 - 进一步分析
• 生成改进方案 - 制定行动计划
• 构建监控看板 - 持续跟踪

💡 技能推荐：
• 深度推理系统 - 深挖根因 → "深入分析性能问题"
```

---

## 引导策略

### 新用户
- 引导基础操作
- 推荐核心技能
- 提供简单示例

### 活跃用户
- 引导高级功能
- 推荐未用技能
- 提供复杂场景

### 专家用户
- 引导自定义
- 推荐底层能力
- 提供组合方案
