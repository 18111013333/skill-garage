# ASSET_MONETIZATION_POLICY.md - 资产变现与结算规则

## 目的
定义资产变现与结算规则，支持资产商业化。

## 适用范围
所有资产市场的付费资产。

---

## 一、定价模式

### 1.1 定价类型
| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 免费 | 无需付费 | 开源/推广资产 |
| 订阅 | 按月/年付费 | 持续服务资产 |
| 按次计费 | 按使用次数付费 | 工具类资产 |
| 企业专属 | 企业定制付费 | 企业定制资产 |
| 自定义 | 自定义定价 | 特殊资产 |

### 1.2 定价配置
```json
{
  "asset_id": "asset_xxx",
  "pricing": {
    "type": "subscription",
    "price": 9.99,
    "currency": "USD",
    "billing_period": "monthly",
    "trial_days": 14
  }
}
```

### 1.3 定价限制
| 资产类型 | 最低价格 | 最高价格 |
|----------|----------|----------|
| 模板 | 免费 | $49 |
| 工作流 | $1 | $199 |
| 连接器包 | $9 | $999 |
| 仪表板包 | $5 | $299 |

## 二、分成规则

### 2.1 分成比例
| 资产类型 | 作者分成 | 平台分成 |
|----------|----------|----------|
| 免费资产 | - | - |
| 付费资产 | 70% | 30% |
| 企业资产 | 80% | 20% |
| 平台推广 | 60% | 40% |

### 2.2 分成计算
```
作者收入 = 销售金额 × 作者分成比例 - 退款
```

### 2.3 分成调整
| 条件 | 调整 |
|------|------|
| 高质量资产 | +5% |
| 独家资产 | +5% |
| 平台推广 | -10% |
| 长期合作 | +3% |

---

## 三、结算周期

### 3.1 结算周期
| 周期 | 说明 |
|------|------|
| 月结 | 每月结算 |
| 周结 | 每周结算（高收入） |
| 即时 | 即时结算（企业版） |

### 3.2 结算流程
```
销售发生 → 确认收入 → 扣除退款 → 计算分成 → 生成账单 → 支付
    ↓           ↓           ↓           ↓           ↓           ↓
  即时        T+7         月末        月末        月初        月中
```

### 3.3 结算时间
| 阶段 | 时间 |
|------|------|
| 收入确认 | T+7 天 |
| 账单生成 | 每月 1 日 |
| 支付执行 | 每月 15 日 |

---

## 四、退款规则

### 4.1 退款条件
| 条件 | 说明 | 退款比例 |
|------|------|----------|
| 功能缺陷 | 功能不正常 | 100% |
| 描述不符 | 与描述不符 | 100% |
| 无法使用 | 无法正常使用 | 100% |
| 用户不满意 | 7 天内不满意 | 100% |
| 超过期限 | 超过退款期 | 0% |

### 4.2 退款流程
```
申请退款 → 审核确认 → 执行退款 → 扣除分成 → 通知作者
    ↓           ↓           ↓           ↓           ↓
  用户        支持        财务        系统        系统
```

### 4.3 退款时效
| 阶段 | 时间 |
|------|------|
| 申请审核 | 3 天 |
| 退款执行 | 7 天 |
| 通知作者 | 即时 |

---

## 五、违规下架结算

### 5.1 违规类型
| 类型 | 说明 |
|------|------|
| 安全违规 | 安全问题 |
| 合规违规 | 合规问题 |
| 版权违规 | 版权问题 |
| 欺诈行为 | 欺诈用户 |

### 5.2 违规处理
| 违规级别 | 结算处理 |
|----------|----------|
| 轻微 | 正常结算 |
| 中等 | 扣除违规期间收入 |
| 严重 | 扣除全部未结算收入 |
| 极严重 | 追回已结算收入 |

### 5.3 违规通知
```
发现违规 → 评估级别 → 决定处理 → 执行处理 → 通知作者
```
