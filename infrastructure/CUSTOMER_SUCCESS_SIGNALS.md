# CUSTOMER_SUCCESS_SIGNALS.md - 客户成功信号

**版本: V27.0**

## 目的
定义客户成功信号，确保平台可以主动经营客户，而不是被动响应，实现"信号可识别、风险可预警、介入有依据"。

## 适用范围
所有客户成功场景，包括：
- Adoption 信号识别
- 停滞信号识别
- Churn 风险识别
- 主动介入触发

---

## 一、信号分类

| 分类 | 代码 | 说明 | 优先级 |
|------|------|------|--------|
| Adoption 顺利 | ADOPTION_POSITIVE | 采用顺利的信号 | 低 |
| Adoption 停滞 | ADOPTION_STALLED | 采用停滞的信号 | 中 |
| Churn 风险 | CHURN_RISK | 流失风险信号 | 高 |
| 扩展机会 | EXPANSION_OPPORTUNITY | 扩展机会信号 | 中 |
| 问题信号 | ISSUE_SIGNAL | 问题信号 | 高 |

---

## 二、Adoption 顺利信号

### 2.1 信号定义

| 信号 | 说明 | 判定条件 |
|------|------|----------|
| 活跃增长 | 用户活跃度持续增长 | DAU 周环比增长 > 10% |
| 功能扩展 | 功能使用范围扩大 | 新功能采用 > 50% |
| 深度使用 | 使用深度增加 | 高级功能使用 > 30% |
| 团队扩展 | 使用团队扩大 | 活跃用户数增长 > 20% |
| 集成增加 | 集成数量增加 | 新增集成 > 1个/月 |

### 2.2 信号指标

| 指标 | 说明 | 目标 |
|------|------|------|
| DAU/MAU | 日活/月活比 | > 30% |
| 功能采用率 | 功能采用比例 | > 50% |
| 高级功能使用 | 高级功能使用率 | > 20% |
| 团队覆盖率 | 团队成员覆盖率 | > 80% |
| 集成数量 | 集成使用数量 | > 2个 |

### 2.3 信号处理

| 信号 | 处理方式 |
|------|----------|
| 活跃增长 | 记录 + 定期回访 |
| 功能扩展 | 记录 + 推荐相关功能 |
| 深度使用 | 记录 + 推荐最佳实践 |
| 团队扩展 | 记录 + 提供培训支持 |
| 集成增加 | 记录 + 提供集成支持 |

---

## 三、Adoption 停滞信号

### 3.1 信号定义

| 信号 | 说明 | 判定条件 |
|------|------|----------|
| 活跃下降 | 用户活跃度下降 | DAU 周环比下降 > 10% |
| 功能收缩 | 功能使用范围缩小 | 功能采用率下降 > 20% |
| 使用浅化 | 使用深度降低 | 高级功能使用下降 > 30% |
| 团队收缩 | 使用团队缩小 | 活跃用户数下降 > 20% |
| 集成减少 | 集成使用减少 | 集成使用下降 > 50% |

### 3.2 信号指标

| 指标 | 说明 | 警戒值 |
|------|------|--------|
| DAU/MAU | 日活/月活比 | < 15% |
| 功能采用率 | 功能采用比例 | < 30% |
| 高级功能使用 | 高级功能使用率 | < 10% |
| 团队覆盖率 | 团队成员覆盖率 | < 50% |
| 集成数量 | 集成使用数量 | < 1个 |

### 3.3 信号处理

| 信号 | 处理方式 |
|------|----------|
| 活跃下降 | 分析原因 + 主动联系 |

| 团队收缩 | 了解原因 + 提供支持 |
| 集成减少 | 了解原因 + 提供替代方案 |

## 四、Churn 风险信号

### 4.1 信号定义

| 信号 | 说明 | 判定条件 |
|------|------|----------|
| 长期不活跃 | 长期未使用 | 超过 14 天未活跃 |
| 大量投诉 | 投诉数量增加 | 月投诉 > 3次 |
| 负面反馈 | 满意度下降 | 满意度 < 60% |
| 付费问题 | 付费状态异常 | 逾期或欠费 |
| 竞品调研 | 竞品调研行为 | 检测到竞品调研 |
| 合同到期 | 合同即将到期 | 距到期 < 30天 |
| 关键人离职 | 关键联系人离职 | 关键人状态变更 |

### 4.2 风险等级

| 等级 | 说明 | 条件 | 介入要求 |
|------|------|------|----------|
| 低风险 | 轻微风险 | 1个风险信号 | 定期关注 |
| 中风险 | 中等风险 | 2-3个风险信号 | 主动联系 |
| 高风险 | 高风险 | 4-5个风险信号 | 紧急介入 |
| 极高风险 | 极高风险 | >5个风险信号 | 立即介入 |

### 4.3 信号处理

| 信号 | 处理方式 |
|------|----------|
| 长期不活跃 | 立即联系 + 了解原因 |
| 大量投诉 | 升级处理 + 专项跟进 |
| 负面反馈 | 专项改进 + 定期回访 |
| 付费问题 | 财务沟通 + 解决方案 |
| 竞品调研 | 了解需求 + 价值强化 |
| 合同到期 | 提前续约沟通 |
| 关键人离职 | 建立新联系人 + 关系维护 |
