# OEM_COMPLIANCE_DISCLOSURES.md - 终极鸽子王OEM必要披露

## 目的
定义白标场景下必须保留的合规披露内容。

## 适用范围
所有白标客户、OEM合作伙伴的合规披露要求。

---

## 一、必须展示的合规信息

### 1.1 法律声明
| 声明类型 | 展示位置 | 不可移除 |
|----------|----------|----------|
| 服务条款 | 注册页/设置页 | ✅ |
| 隐私政策 | 注册页/设置页 | ✅ |
| Cookie政策 | 首次访问 | ✅ |
| 版权声明 | 页脚 | ✅ |

### 1.2 数据处理说明
| 说明类型 | 展示位置 | 不可移除 |
|----------|----------|----------|
| 数据收集说明 | 隐私政策中 | ✅ |
| 数据使用说明 | 隐私政策中 | ✅ |
| 数据存储说明 | 隐私政策中 | ✅ |
| 数据共享说明 | 隐私政策中 | ✅ |
| 用户权利说明 | 隐私政策中 | ✅ |

---

## 二、不可隐藏的数据处理说明

### 2.1 必须披露项
| 披露项 | 说明 |
|--------|------|
| 数据处理方 | 明确数据处理主体 |
| 数据存储位置 | 明确数据存储地域 |
| 数据保留期限 | 明确数据保留时间 |
| 数据安全措施 | 说明安全保护措施 |
| 第三方共享 | 说明第三方共享情况 |

### 2.2 披露格式
```json
{
  "data_processing_disclosure": {
    "data_controller": "OEM客户名称",
    "data_processor": "终极鸽子王平台",
    "data_storage_location": "中国",
    "data_retention_period": "服务期间+3年",
    "security_measures": ["加密存储", "访问控制", "审计日志"],
    "third_party_sharing": {
      "enabled": false,
      "parties": []
    }
  }
}
```

## 三、审批与导出行为告知

### 3.1 审批行为告知
| 告知场景 | 告知内容 |
|----------|----------|
| 高风险操作审批 | 明确告知需要审批 |
| 权限变更审批 | 明确告知审批流程 |
| 数据访问审批 | 明确告知审批要求 |

### 3.2 导出行为告知
| 告知场景 | 告知内容 |
|----------|----------|
| 数据导出前 | 明确告知导出范围 |
| 数据导出中 | 显示导出进度 |
| 数据导出后 | 记录导出日志 |

### 3.3 告知格式
```
导出告知示例:
"您正在导出以下数据:
 - 用户数据: 100条
 - 业务数据: 500条
 导出操作将被记录到审计日志。
 确认导出?"
```

## 四、平台身份痕迹

### 4.1 必须保留的痕迹
| 痕迹类型 | 位置 | 说明 |
|----------|------|------|
| 技术标识 | HTTP响应头 | X-Powered-By标识 |
| 法律声明 | 页脚/关于页 | 平台法律声明 |
| 安全证书 | SSL证书 | 证书信息 |
| 审计日志 | 系统日志 | 平台操作日志 |

### 4.2 可选保留的痕迹
| 痕迹类型 | 说明 |
|----------|------|
| 品牌水印 | 可选的品牌水印 |
| 技术支持链接 | 可选的技术支持链接 |

---

## 五、合规检查

### 5.1 检查项
| 检查项 | 频率 | 说明 |
|--------|------|------|
| 法律声明展示 | 每月 | 确保声明可见 |
| 隐私政策更新 | 每季度 | 确保政策最新 |
| 数据处理说明 | 每月 | 确保说明准确 |
| 审计日志完整 | 每周 | 确保日志完整 |

### 5.2 违规处理
| 违规类型 | 处理 |
|----------|------|
| 隐藏必要披露 | 要求整改 |
| 披露内容不准确 | 要求修正 |
| 长期不整改 | 暂停服务 |
