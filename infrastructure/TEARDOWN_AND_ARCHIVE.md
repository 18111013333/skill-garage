# TEARDOWN_AND_ARCHIVE.md - 终极鸽子王测试环境回收与归档

## 目的
定义测试环境的回收流程和数据归档规则。

## 适用范围
所有沙箱环境、测试环境的生命周期结束处理。

---

## 一、回收时机

### 1.1 自动回收
| 触发条件 | 说明 |
|----------|------|
| 到期未续期 | 保留期满自动回收 |
| 项目结束 | 项目正式结束后回收 |
| 合同终止 | 合同终止后回收 |

### 1.2 手动回收
| 触发条件 | 说明 |
|----------|------|
| 客户申请 | 客户主动申请回收 |
| 安全事件 | 安全事件后强制回收 |
| 违规处理 | 违规后强制回收 |

---

## 二、回收前导出

### 2.1 导出内容
| 导出项 | 格式 | 说明 |
|--------|------|------|
| 配置数据 | JSON | 系统配置导出 |
| 测试数据 | CSV/JSON | 测试数据导出 |
| 日志数据 | JSON | 操作日志导出 |
| 报告文档 | PDF | 测试报告导出 |

### 2.2 导出流程
```
导出流程
    ↓
┌─────────────────────────────────────┐
│ 1. 导出申请                          │
│    - 确认导出范围                    │
│    - 审批导出申请                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 数据导出                          │
│    - 执行数据导出                    │
│    - 验证导出完整性                  │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 数据交付                          │
│    - 安全传输数据                    │
│    - 确认接收                        │
└─────────────────────────────────────┘
```

---

## 三、日志/审计保存

### 3.1 保存内容
| 保存项 | 保存期限 | 说明 |
|--------|----------|------|
| 操作日志 | 1年 | 所有操作记录 |
| 审计日志 | 3年 | 审计相关记录 |
| 安全日志 | 3年 | 安全相关记录 |
| 访问日志 | 1年 | 访问记录 |

### 3.2 保存方式
| 方式 | 说明 |
|------|------|
| 加密存储 | 日志加密存储 |
| 只读访问 | 日志只读不可修改 |
| 独立存储 | 与生产环境隔离 |

---

## 四、数据销毁

### 4.1 销毁范围
| 数据类型 | 销毁方式 | 说明 |
|----------|----------|------|
| 用户数据 | 安全擦除 | 多次覆写 |
| 业务数据 | 安全擦除 | 多次覆写 |
| 配置数据 | 安全擦除 | 多次覆写 |
| 临时数据 | 直接删除 | 标准删除 |

### 4.2 销毁验证
```
销毁验证流程
    ↓
┌─────────────────────────────────────┐
│ 1. 销毁执行                          │
│    - 执行数据销毁                    │
│    - 记录销毁操作                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 销毁验证                          │
│    - 验证数据不可恢复                │
│    - 生成销毁报告                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 销毁确认                          │
│    - 签字确认                        │
│    - 归档销毁记录                    │
└─────────────────────────────────────┘
```

## 五、资产迁移

### 5.1 可迁移资产
| 资产类型 | 迁移方式 | 说明 |
|----------|----------|------|
| 配置模板 | 导出导入 | 配置可复用 |
| 知识文档 | 导出导入 | 文档可复用 |
| 测试用例 | 导出导入 | 用例可复用 |

### 5.2 迁移流程
```
资产迁移流程
    ↓
┌─────────────────────────────────────┐
│ 1. 资产识别                          │
│    - 识别可迁移资产                  │
│    - 评估迁移价值                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 资产导出                          │
│    - 导出资产数据                    │
│    - 验证导出完整性                  │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 资产导入                          │
│    - 导入到目标环境                  │
│    - 验证导入成功                    │
└─────────────────────────────────────┘
```
