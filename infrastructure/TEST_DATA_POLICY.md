# TEST_DATA_POLICY.md - 终极鸽子王测试数据规则

## 目的
定义测试环境的数据使用规则，确保数据安全。

## 适用范围
所有沙箱环境、测试环境的数据管理。

---

## 一、数据模式

### 1.1 样例数据
| 数据类型 | 内容 | 用途 |
|----------|------|------|
| 用户数据 | 虚构用户信息 | 功能演示 |
| 业务数据 | 虚构业务记录 | 功能测试 |
| 配置数据 | 标准配置模板 | 配置参考 |

### 1.2 脱敏数据
| 原始数据 | 脱敏规则 | 示例 |
|----------|----------|------|
| 姓名 | 保留姓氏，名字替换 | 张** |
| 手机号 | 中间4位替换 | 138****1234 |
| 邮箱 | @前替换 | ***@example.com |
| 身份证 | 保留前3后4 | 110***********1234 |
| 地址 | 仅保留省市 | 北京市**区 |

### 1.3 合成数据
| 数据类型 | 生成方式 | 用途 |
|----------|----------|------|
| 用户数据 | AI生成 | 性能测试 |
| 业务数据 | 规则生成 | 压力测试 |
| 日志数据 | 模拟生成 | 审计演示 |

## 二、真实数据禁止范围

### 2.1 禁止导入
| 数据类型 | 禁止原因 |
|----------|----------|
| 生产用户数据 | 隐私保护 |
| 生产业务数据 | 数据安全 |
| 敏感配置数据 | 安全风险 |
| 认证凭据数据 | 安全风险 |

### 2.2 例外审批
| 场景 | 审批要求 | 处理方式 |
|------|----------|----------|
| UAT需要真实数据 | 安全团队审批 | 必须脱敏处理 |
| 性能测试需要大量数据 | 技术团队审批 | 使用合成数据 |

---

## 三、共享测试数据限制

### 3.1 共享规则
| 规则 | 说明 |
|------|------|
| 禁止跨租户共享 | 测试数据不可跨租户 |
| 禁止导出共享 | 测试数据不可导出共享 |
| 禁止包含真实信息 | 共享数据不得包含真实信息 |

### 3.2 数据隔离
| 隔离项 | 说明 |
|--------|------|
| 存储隔离 | 各沙箱数据独立存储 |
| 访问隔离 | 各沙箱数据独立访问 |
| 网络隔离 | 各沙箱网络隔离 |

---

## 四、数据生命周期

### 4.1 数据导入
```
数据导入流程
    ↓
┌─────────────────────────────────────┐
│ 1. 数据检查                          │
│    - 检查数据类型                    │
│    - 检查数据格式                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 安全扫描                          │
│    - 扫描敏感信息                    │
│    - 检测真实数据                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 脱敏处理                          │
│    - 自动脱敏PII                    │
│    - 替换敏感数据                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 数据导入                          │
│    - 导入到沙箱                      │
│    - 记录导入日志                    │
└─────────────────────────────────────┘
```

### 4.2 数据清理
| 清理时机 | 清理内容 |
|----------|----------|
| 沙箱到期 | 全部数据清理 |
| 项目结束 | 项目相关数据清理 |
| 定期清理 | 过期临时数据清理 |

## 五、数据安全要求

### 5.1 安全措施
| 措施 | 说明 |
|------|------|
| 加密存储 | 测试数据加密存储 |
| 访问控制 | 严格的访问权限控制 |
| 审计日志 | 完整的数据操作日志 |
| 安全销毁 | 到期数据安全销毁 |

### 5.2 安全检查
| 检查项 | 频率 | 说明 |
|--------|------|------|
| 敏感数据扫描 | 每次导入 | 检测真实数据 |
| 权限检查 | 每周 | 验证访问控制 |
| 日志审计 | 每月 | 审查数据操作 |
