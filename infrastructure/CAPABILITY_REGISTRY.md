# CAPABILITY_REGISTRY.md - 能力注册中心

## 目的
统一管理所有能力模块的注册与发现。

## 能力层级统计

| 层级 | 能力数 | 核心能力 |
|------|--------|----------|
| L0 核心层 | 5 | 身份、人格、引导 |
| L1 运行时层 | 35 | 执行、安全、工具 |
| L2 智能层 | 4 | 意图、推理、知识、决策 |
| L3 多模态层 | 3 | 感知、情感、创意 |
| L4 协作层 | 3 | 多智能体、克隆、同步 |
| L5 架构层 | 7 | 微服务、DDD、整洁架构 |
| L6 生成层 | 3 | 代码、测试、文档 |
| L7 统一层 | 8 | 八大统一中心 |
| L8 引导层 | 4 | 新手、教程、帮助、反馈 |
| L9 进化层 | 4 | 进化、学习、升级 |
| L10 极致层 | 3 | 缓存、指标、优化 |
| L11 治理层 | 4 | 治理、合规、审计 |
| **总计** | **80** | **完整融合** |

## 能力发现

```bash
# 查看所有能力
openclaw capabilities list

# 按层级查看
openclaw capabilities list --layer L5

# 搜索能力
openclaw capabilities search "微服务"

# 查看能力详情
openclaw capabilities show --name "microservice"
```
