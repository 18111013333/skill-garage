# 自进化系统完整配置

## 自动触发机制

### 1. 新技能安装后
```bash
# 自动运行
bash skills/auto-skill-upgrade/scripts/skill-integration-hook.sh <skill_name>
```

### 2. 每次会话启动
```bash
# 在 AGENTS.md 中配置
bash skills/auto-skill-upgrade/scripts/auto-evolve.sh
```

### 3. 定期检查（心跳）
```bash
# 在 HEARTBEAT.md 中配置
- 运行自进化检查
- 检测冗余技能
- 更新知识图谱
```

---

## 自进化脚本清单

| 脚本 | 功能 | 触发时机 |
|------|------|----------|
| `auto-evolve.sh` | 完整自进化流程 | 新技能安装/会话启动 |
| `skill-integration-hook.sh` | 技能安装后钩子 | 技能安装完成 |
| `detect-redundancy.sh` | 冗余检测 | 自进化流程中 |
| `learning-capture.sh` | 学习/错误捕获 | 任意时刻 |
| `upgrade-skills.sh` | 技能库升级 | 自进化流程中 |
| `scan-skills.sh` | 技能扫描 | 升级流程中 |
| `detect-conflicts.sh` | 冲突检测 | 升级流程中 |
| `merge-skills.sh` | 技能整合 | 升级流程中 |

## 冗余防护机制

### 功能重叠处理

| 类别 | 统一入口 | 其他技能处理 |
|------|----------|--------------|
| 图像处理 | xiaoyi-image-understanding | 作为备选，建立优先级链 |
| 文档转换 | xiaoyi-doc-convert | 作为底层实现，不直接调用 |
| 搜索调研 | deep-search-and-insight-synthesize | 快速搜索用 xiaoyi-web-search |
| 自进化 | self-improving-agent | 其他作为组件调用 |

### 依赖冲突处理

1. 检测重复依赖包
2. 统一版本要求
3. 隔离冲突技能

### 低使用率处理

1. 记录使用频率
2. 标记低使用技能
3. 建议禁用或删除

## 学习捕获接口

### 记录学习
```bash
bash skills/auto-skill-upgrade/scripts/learning-capture.sh learning "标题" "详细内容" "high"
```

### 记录错误
```bash
bash skills/auto-skill-upgrade/scripts/learning-capture.sh error "错误标题" "错误详情" "critical"
```

### 记录功能请求
```bash
bash skills/auto-skill-upgrade/scripts/learning-capture.sh feature "功能标题" "功能描述" "medium"
```
