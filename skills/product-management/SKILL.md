## 选品方法论

### 选品四象限

| | 高利润 | 低利润 |
|------|--------|--------|
| **高销量** | 明星产品 | 引流产品 |
| **低销量** | 利润产品 | 淘汰产品 |

### 选品公式
```
选品得分 = 市场需求(30%) × 利润空间(25%) × 竞争程度(20%) × 供应链(15%) × 差异化(10%)
```

## 定价策略

| 策略 | 适用场景 | 示例 |
|------|----------|------|
| 成本加成 | 标准品 | 成本×1.3 |
| 竞品跟随 | 红海市场 | 比竞品低5% |
| 价值定价 | 差异化产品 | 按价值定价 |
| 心理定价 | 促销场景 | 99元、199元 |
| 阶梯定价 | 批量销售 | 买越多越便宜 |

## 标题优化公式

```
标题 = 品牌词 + 核心关键词 + 属性词 + 场景词 + 促销词

示例：XX品牌 新鲜草莓 丹东99 红颜奶油香 顺丰包邮 当日达
```

## 库存管理

### 安全库存计算
```
安全库存 = (最大日销量 × 最长补货周期) - (平均日销量 × 平均补货周期)
```

### 补货预警
```
补货点 = 平均日销量 × 补货周期 + 安全库存
```

## 爆款打造流程

1. **选品** → 市场分析、竞品调研
2. **测款** → 小批量测试、数据验证
3. **打爆** → 流量倾斜、活动配合
4. **稳销** → 库存保障、服务跟上
5. **迭代** → 持续优化、延长周期
