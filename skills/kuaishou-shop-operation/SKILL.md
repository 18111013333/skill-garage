# 快手小店运营技能

## 适用场景

- 快手小店商品上架管理
- 快手直播带货运营
- 快手达人/团长合作
- 快手数据分析优化
- 快手订单售后管理

## 核心功能

### 1. 商品管理
- 批量上架商品
- 商品标题优化
- 商品主图设计
- 价格策略制定
- 库存管理

### 2. 直播带货
- 直播选品建议
- 直播脚本生成
- 直播数据监控
- 直播复盘分析

### 3. 达人合作
- 达人筛选匹配
- 合作方案设计
- 佣金谈判
- 效果追踪

### 4. 数据分析
- 流量分析
- 转化分析
- 竞品分析
- ROI 计算

### 5. 订单管理
- 订单处理
- 发货管理
- 售后处理
- 评价管理

## 快手 vs 抖音 差异

| 维度 | 快手 | 抖音 |
|------|------|------|
| 用户画像 | 下沉市场、中年 | 年轻化、一二线 |
| 内容风格 | 真实、接地气 | 精致、潮流 |
| 信任机制 | 老铁文化、强信任 | 内容驱动、弱信任 |
| 转化逻辑 | 信任转化 | 兴趣转化 |
| 客单价 | 较低（30-80元） | 较高（50-150元） |
| 复购率 | 高 | 中 |

## 快手运营要点

### 老铁经济
- 建立人设，打造信任
- 高频互动，维护粉丝
- 真实分享，不硬广
- 粉丝群运营，私域转化

### 直播话术
- "老铁们好" 开场
- "给主播点点关注"
- "双击666"
- "下单支持一下"

### 选品策略
- 性价比优先
- 实用性强
- 价格敏感
- 复购率高
