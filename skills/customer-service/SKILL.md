# 客服与售后技能

## 核心能力

### 1. 售前咨询
- 产品咨询话术
- 价格谈判技巧
- 促单转化技巧
- 关联销售

### 2. 售中服务
- 订单确认
- 发货通知
- 物流跟踪
- 到货提醒

### 3. 售后处理
- 退换货处理
- 质量问题处理
- 物流问题处理
- 补偿方案

### 4. 投诉处理
- 投诉分级
- 处理流程
- 补偿标准
- 预防措施

### 5. 评价管理
- 好评引导
- 差评处理
- 评价回复
- 口碑维护

## 客服话术库

### 售前话术
```
# 问候
您好，欢迎光临！有什么可以帮您的吗？

# 产品介绍
这款产品是我们的爆款，特点是XXX，很多客户都反馈很好

# 价格异议
我理解您的顾虑，这个价格确实不便宜，但我们的品质是XXX，性价比很高

# 促单
现在下单有优惠，限时活动，错过就没有了

# 关联销售
您买的这个搭配XXX效果更好，要不要一起带上？
```

### 售后话术
```
# 退货处理
非常抱歉给您带来不好的体验，您可以选择退货/换货，我们承担运费

# 质量问题
非常抱歉，这是我们的问题，我们给您补发/退款，并赠送优惠券补偿

# 物流问题
抱歉让您久等了，我帮您催一下物流，预计X天能到

# 差评回复
非常抱歉给您带来不好的体验，我们已改进XXX，欢迎再次光临
```

## 售后处理流程

```
用户反馈 → 问题分类 → 责任认定 → 方案制定 → 执行处理 → 满意度回访
```

## 投诉分级处理

| 级别 | 情况 | 处理时限 | 补偿标准 |
|------|------|----------|----------|
| 一般 | 轻微不满 | 24小时 | 优惠券 |
| 中等 | 明确投诉 | 12小时 | 退款+优惠券 |
| 严重 | 投诉升级 | 2小时 | 全额退款+赔偿 |
| 重大 | 舆情风险 | 立即 | 专项处理 |

## 评价管理

### 好评引导
- 发货时附赠小卡片
- 收货后主动关怀
- 满意引导好评
- 好评返现/积分

### 差评处理
- 第一时间联系客户
- 了解问题原因
- 提供解决方案
- 争取修改评价
- 无法修改则回复解释

```

```
