# 经销商-团长合作技能

## 适用场景

- 经销商寻找团长合作卖货
- 社区团购团长筛选
- 分销渠道拓展
- 佣金体系设计
- 订单与结算管理

## 核心功能

### 1. 团长筛选
- 按品类匹配团长
- 按地域筛选团长
- 按粉丝量/活跃度筛选
- 按历史销量筛选
- 团长信誉评估

### 2. 合作洽谈
- 合作方案生成
- 利益分配设计
- 合作协议模板
- 沟通话术建议

### 3. 佣金设计
- 阶梯佣金设计
- 品类佣金配置
- 促销佣金规则
- 结算周期设置

### 4. 订单管理
- 订单追踪
- 发货管理
- 售后处理
- 数据统计

### 5. 结算对账
- 自动对账
- 结算单生成
- 发票管理
- 财务报表

## 使用示例

```
# 找团长
帮我找北京地区做生鲜的团长，粉丝量1万以上

# 设计佣金
帮我设计一个生鲜品类的佣金方案，毛利30%

# 洽谈合作
帮我生成一个团长合作方案，包含佣金、结算、售后条款

# 订单管理
帮我统计本月团长订单数据
```

## 数据模型

### 团长信息
```json
{
  "leader_id": "团长ID",
  "name": "团长姓名",
  "region": "所在地区",
  "category": "主营品类",
  "followers": "粉丝数",
  "monthly_sales": "月销售额",
  "rating": "信誉评分",
  "contact": "联系方式"
}
```

### 合作协议
```json
{
  "agreement_id": "协议ID",
  "dealer_id": "经销商ID",
  "leader_id": "团长ID",
  "commission_rate": "佣金比例",
  "settlement_cycle": "结算周期",
  "valid_from": "生效日期",
  "valid_to": "到期日期"
}
```

## 工作流程

1. **需求分析** → 明确品类、预算、目标
2. **团长筛选** → 多维度筛选匹配
3. **资质审核** → 验证团长资质
4. **合作洽谈** → 方案设计、谈判
5. **签约上线** → 签订协议、系统对接
6. **运营支持** → 培训、素材、活动
7. **数据监控** → 销量、佣金、结算
