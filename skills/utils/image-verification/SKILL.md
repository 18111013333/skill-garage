# 图片验证技能

## 问题背景

在发送景点、餐厅、商品等介绍时，经常出现图片和介绍不匹配的问题：
- 下载的图片是瀑布，但介绍是石阶
- 下载的图片是栈道，但介绍是天池
- 用户看到错误的图片，体验很差

## 解决方案

### 1. 图片验证流程

```
搜索图片 → 下载图片 → 验证图片内容 → 匹配介绍 → 发送给用户
                ↓
            不匹配 → 重新搜索
```

### 2. 验证规则

**必须验证的场景**：
- 景点介绍（瀑布、石阶、栈道、天池等）
- 餐厅介绍（菜品、环境、门面等）
- 商品介绍（产品图、细节图等）
- 人物介绍（肖像、工作照等）

**验证方法**：
1. 使用 `image_reading` 工具识别图片内容
2. 对比图片内容和介绍文字
3. 如果不匹配，重新搜索

### 3. 验证检查清单

**景点验证**：
- [ ] 瀑布图片：是否有明显的瀑布水流？
- [ ] 石阶图片：是否有连续的石阶？
- [ ] 栈道图片：是否有木质或石质栈道？
- [ ] 天池图片：是否有湖泊或水潭？
- [ ] 入口图片：是否有大门、停车场、指示牌？

**餐厅验证**：
- [ ] 菜品图片：是否有食物？
- [ ] 环境图片：是否有餐厅内部？
- [ ] 门面图片：是否有餐厅招牌？

### 4. 搜索关键词优化

**问题**：搜索"白水寨"会返回各种图片，不一定是目标景点

**解决方案**：使用更精确的搜索关键词

| 景点 | 错误关键词 | 正确关键词 |
|-----|----------|----------|
| 白水仙瀑 | 白水寨 | 白水寨白水仙瀑 428.5米瀑布 |
| 天南第一梯 | 白水寨 | 白水寨9999级石阶 天南第一梯 |
| 亲水栈道 | 白水寨 | 白水寨海船木栈道 亲水栈道 |
| 仙姑天池 | 白水寨 | 白水寨仙姑天池 山顶湖泊 |
| 景区入口 | 白水寨 | 白水寨景区大门 停车场 |

### 5. 图片来源优先级

1. **百度百科** - 最准确，有官方认证
2. **携程/马蜂窝** - 游客实拍，较真实
3. **百度图片** - 数量多，需验证
4. **小红书** - 真实但需筛选

### 6. 自动化验证脚本

```python
# 伪代码
def verify_image(image_path, expected_content):
    """
    验证图片内容是否匹配预期
    
    Args:
        image_path: 图片路径
        expected_content: 预期内容（如"瀑布"、"石阶"）
    
    Returns:
        bool: 是否匹配
    """
    result = image_reading(image_path, prompt=f"这张图片是{expected_content}吗？")
    return expected_content in result

def search_and_verify(attraction_name, keyword, source="baike"):
    """
    搜索并验证图片
    
    Args:
        attraction_name: 景点名称
        keyword: 搜索关键词
        source: 图片来源（baike/ctrip/baidu）
    
    Returns:
        str: 验证通过的图片路径
    """
    max_retries = 3
    for i in range(max_retries):
        # 1. 搜索图片
        image_url = search_image(keyword, source)
        
        # 2. 下载图片
        image_path = download_image(image_url)
        
        # 3. 验证图片
        if verify_image(image_path, attraction_name):
            return image_path
        
        # 4. 验证失败，换关键词重试
        keyword = refine_keyword(keyword)
    
    return None  # 验证失败
```

### 7. 错误处理

**如果验证失败**：
1. 记录失败原因
2. 重新搜索更精确的关键词
3. 尝试其他图片来源
4. 如果多次失败，告知用户

**失败原因及解决方案**：

| 失败原因 | 解决方案 |
|---------|---------|
| 关键词太宽泛 | 使用更精确的关键词 |
| 图片来源不可靠 | 切换到百度百科 |
| 图片内容模糊 | 下载更高清的图片 |
| 多次验证失败 | 告知用户，提供文字介绍 |

### 8. 示例流程

**搜索白水仙瀑图片**：

```
步骤1：精确搜索
  关键词："白水寨白水仙瀑 428.5米"
  来源：百度百科

步骤2：下载图片
  URL：https://bkimg.cdn.bcebos.com/pic/xxx.jpg
  保存：/tmp/waterfall.jpg

步骤3：AI验证
  提示词："这张图片是瀑布吗？瀑布的规模如何？落差大约多少米？"
  结果："是大型瀑布，落差大，气势磅礴"

步骤4：匹配判断
  结果包含"瀑布" + "落差大" → 匹配 ✅

步骤5：发送给用户
  先发图片，再发介绍
```
