## 供应商评估模型

| 维度 | 权重 | 评估内容 |
|------|------|----------|
| 价格 | 30% | 价格竞争力、付款条件 |
| 质量 | 25% | 产品质量、合格率 |
| 交期 | 20% | 交货准时率、响应速度 |
| 服务 | 15% | 售后服务、配合度 |
| 风险 | 10% | 供应稳定性、替代性 |

## 采购流程

```
需求确认 → 供应商询价 → 比价议价 → 下单采购 → 到货验收 → 入库上架 → 付款结算
```

## 库存管理

### ABC分类
| 类别 | 占比 | 管理策略 |
|------|------|----------|
| A类 | 20%SKU，80%金额 | 重点管理，低库存 |
| B类 | 30%SKU，15%金额 | 常规管理 |
| C类 | 50%SKU，5%金额 | 简化管理，高库存 |

### 库存周转
```
库存周转率 = 销售成本 / 平均库存
库存周转天数 = 365 / 库存周转率
```

## 物流选择

| 物流类型 | 时效 | 成本 | 适用场景 |
|----------|------|------|----------|
| 顺丰 | 1-2天 | 高 | 高价值、急件 |
| 京东 | 1-2天 | 中高 | 品质要求高 |
| 中通/圆通 | 2-4天 | 低 | 常规商品 |
| 邮政 | 3-7天 | 最低 | 偏远地区 |
| 同城配送 | 当日/次日 | 中 | 生鲜、本地 |

## 成本控制要点

1. **采购成本** → 货比三家、批量采购、长期协议
2. **仓储成本** → 合理库存、减少积压、优化空间
3. **物流成本** → 选择合适物流、优化包装、合并发货
4. **损耗成本** → 减少破损、控制退货、降低损耗
