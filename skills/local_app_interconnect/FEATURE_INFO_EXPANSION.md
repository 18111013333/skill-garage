# 信息扩展功能

## 功能概述

当用户请求获取某类信息时，自动识别并扩展获取相关联的信息，提供更完整的答案。

## 支持的扩展类型

| 类型 | 触发关键词 | 自动扩展内容 |
|------|------------|--------------|
| 房源 | 房源、租房、商铺、门面、公寓 | 照片、位置、联系方式、价格、周边配套 |
| 商家 | 商家、店铺、餐厅、酒店、理发店 | 照片、位置、联系方式、营业时间、评价 |
| 商品 | 商品、产品、价格、比价 | 照片、价格对比、参数、评价 |
| 服务 | 服务、预约、预订 | 照片、位置、联系方式、价格 |

## 使用方式

### 方式1: 自动识别
```
用户: 帮我找淄博商铺
系统: 自动识别为"房源"类型
      → 扩展获取: 照片 + 位置 + 联系方式 + 周边配套
```

### 方式2: 明确指定
```
用户: 帮我找房源，要照片和位置
系统: 按指定项扩展获取
```

## 输出格式

### 房源完整输出
```
🏠 房源信息

【基本信息】
名称: XXX
面积: XX㎡ | 月租: XXX元
地址: XXX

【房间照片】
[图片1: 外观]
[图片2: 内部]

【地理位置】
📍 经纬度: XX.XXXX, XXX.XXXX
🗺️ [地图截图]

【联系方式】
📞 电话: 138****1234

【周边配套】
便利店: 50米 | 医院: 500米 | 公交: 200米
```

## 配置

| 选项 | 默认值 | 说明 |
|------|--------|------|
| enable_expansion | true | 启用扩展 |
| max_photos | 5 | 最大照片数 |
| include_map | true | 包含地图 |
| include_nearby | true | 包含周边 |
