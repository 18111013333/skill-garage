# 消息搜索扩展功能

## 功能概述

当用户搜索消息时，自动识别扩展需求，获取相关联的信息。

## 扩展场景

| 搜索类型 | 触发关键词 | 自动扩展 |
|----------|------------|----------|
| 联系人消息 | "张三的消息"、"和XX的聊天" | 联系人信息、最近消息、通话记录 |
| 主题消息 | "关于XX的消息"、"XX相关的短信" | 相关消息列表、发送者信息 |
| 时间消息 | "昨天的消息"、"上周的短信" | 时间范围消息、消息统计 |
| 类型消息 | "验证码"、"银行短信" | 分类消息、关键信息提取 |

## 使用方式

```
用户: 搜索张三的消息
系统: 
  1. 搜索张三的短信
  2. 扩展: 获取张三联系人信息
  3. 扩展: 获取最近通话记录
  4. 输出完整结果
```

## 输出格式

```
📱 消息搜索结果

【联系人信息】
姓名: 张三
电话: 138****1234
最近联系: 2小时前

【消息列表】
[今天 14:30] 张三: 明天见面...
[昨天 18:20] 张三: 好的...

【通话记录】
最近通话: 今天 10:15 (呼入 2分钟)
```
