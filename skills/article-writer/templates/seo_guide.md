# SEO优化建议模板

## 一、标题优化

### 标题公式
```
[数字] + [动词] + [结果] + [时间/成本]
例：10分钟完成全平台发布，效率提升94%
```

### 标题检查清单
- [ ] 长度控制在20-30字（中文）
- [ ] 包含核心关键词
- [ ] 有数字（具体化）
- [ ] 有情绪触发点（反差/冲突/好奇）
- [ ] 避免标题党过度

### 标题模板库

**教程类：**
- 如何在[时间]内完成[目标]
- [数字]步教你[达成结果]
- 从[起点]到[终点]的完整指南

**案例类：**
- 我用[方法]实现了[结果]
- [数字]天，从[起点]到[终点]
- 为什么[反差]，原因竟然是...

**工具类：**
- [数字]个工具帮你[达成目标]
- 最好用的[数字]个[类别]工具
- [工具名]使用指南

## 二、关键词布局

### 关键词密度
- 核心关键词：2-3%
- 长尾关键词：自然分布
- 避免堆砌

### 布局位置
1. **标题** - 必须包含核心关键词
2. **首段** - 前100字出现核心关键词
3. **小标题** - 自然融入相关关键词
4. **结尾** - 再次点题

### 关键词类型

| 类型 | 示例 | 用途 |
|------|------|------|
| 核心词 | AI写作 | 标题+首尾 |
| 长尾词 | AI写作工具推荐 | 小标题 |
| 问题词 | AI写作怎么用 | FAQ部分 |
| 相关词 | 效率提升、内容创作 | 正文自然分布 |

## 三、内容结构优化

### 理想结构
```
1. 开头（100-150字）
   - 痛点共鸣
   - 核心观点
   - 阅读收益

2. 正文（模块化）
   01 模块一
   02 模块二
   03 模块三

3. 结尾（100-150字）
   - 总结要点
   - 行动指引
   - 互动引导
```

### 段落规则
- 每段不超过150字
- 每500字一个金句
- 每300字一张配图

## 四、内链外链

### 内链策略
- 相关文章互链
- 关键词锚文本
- 每篇3-5个内链

### 外链策略
- 权威来源引用
- 数据来源标注
- 工具官网链接

## 五、元数据优化

### Meta Description
```
长度：120-160字符
公式：[问题] + [解决方案] + [结果]
例：还在为写文章发愁？本文教你用AI工具10分钟完成高质量文章，效率提升94%。
```

### Meta Keywords
```
核心词, 长尾词1, 长尾词2, 相关词1, 相关词2
例：AI写作, AI写作工具, 文章生成, 内容创作, 效率提升
```

## 六、结构化数据

### 文章Schema
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "文章标题",
  "author": {
    "@type": "Person",
    "name": "作者名"
  },
  "datePublished": "2026-04-09",
  "dateModified": "2026-04-09",
  "description": "文章摘要"
}
```

### FAQ Schema
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "问题1",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "答案1"
      }
    }
  ]
}
```

## 七、移动端优化

- 字体不小于16px
- 行间距1.5-1.8
- 段落间距充足
- 图片自适应
- 避免弹窗遮挡

## 八、SEO检查清单

### 发布前检查
- [ ] 标题包含核心关键词
- [ ] URL简洁有意义
- [ ] Meta Description完整
- [ ] 首段包含核心关键词
- [ ] 图片有alt属性
- [ ] 内链外链合理
- [ ] 结构化数据正确
- [ ] 移动端显示正常

### 发布后监控
- [ ] 收录情况（site:域名）
- [ ] 关键词排名
- [ ] 点击率（CTR）
- [ ] 停留时间
- [ ] 跳出率
