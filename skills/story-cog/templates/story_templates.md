# 故事生成模板库

## 一、故事结构模板

### 1. 三幕式结构（经典）
```
第一幕：铺垫（25%）
- 日常世界
- 激励事件
- 拒绝召唤
- 跨越门槛

第二幕：对抗（50%）
- 试炼与盟友
- 接近核心
- 危机时刻
- 一无所有

第三幕：解决（25%）
- 终极之战
- 回归与改变
```

### 2. 英雄之旅（神话）
```
1. 平凡世界
2. 冒险召唤
3. 拒绝召唤
4. 遇见导师
5. 跨越第一道门槛
6. 试炼、盟友、敌人
7. 接近最深的洞穴
8. 磨难
9. 奖赏
10. 返回之路
11. 复活
12. 携带仙丹回归
```

### 3. 起承转合（东方）
```
起：开篇点题，引出人物
承：发展情节，铺垫冲突
转：转折突变，推向高潮
合：收束全文，点明主旨
```

## 二、类型模板

### 悬疑/推理
```
【开篇】发现尸体/案件
【调查】收集线索，排除嫌疑
【误导】引入红鲱鱼（假线索）
【转折】推翻之前的推理
【真相】揭示凶手和手法
【动机】解释为何犯罪
```

### 科幻
```
【设定】未来世界/科技背景
【冲突】人与技术/社会的矛盾
【探索】发现新事物/新问题
【危机】威胁到生存/价值观
【选择】主角做出关键决定
【反思】对科技/人性的思考
```

### 爱情
```
【相遇】男女主初遇
【吸引】产生好感
【障碍】误会/家庭/第三者
【分离】感情受挫
【觉醒】意识到真爱
【重逢】克服障碍，在一起
```

### 恐怖
```
【日常】平静的开端
【异变】诡异事件开始
【否认】主角试图解释
【升级】恐怖加剧
【发现】揭示真相
【对抗】生死存亡
```

## 四、场景模板

### 追逐场景
```
【起点】发现危险
【障碍】遇到阻碍
【转折】险些被抓
【逃脱】成功脱身
【代价】付出代价
```

### 对话场景
```
【目的】这场对话要推进什么
【冲突】双方立场对立
【潜台词】表面说什么，实际想什么
【转折】对话中的意外
【结果】对话后的变化
```

### 战斗场景
```
【前奏】对峙，气氛紧张
【交锋】你来我往
【劣势】主角陷入困境
【转机】找到突破口
【决胜】最后一击
【余波】战斗后的状态
```

## 五、开头模板

### 悬念式
> "那天早上，我发现自己的墓碑立在院子里。"

### 对话式
> "你确定要这么做？"她问，"没有回头路。"

### 动作式
> 子弹擦过耳边，我扑向掩体。

### 描写式
> 这座城市已经死了，只是它自己还不知道。

### 倒叙式
> 后来我常想，如果那天我没有上那趟地铁，一切会不会不同。

## 六、结尾模板

### 开放式
> 他走向远方，身后是燃烧的城市。明天会怎样，没人知道。

### 循环式
> 又是同样的早晨，同样的阳光，同样的敲门声。

### 反转式
> "恭喜你，"面试官微笑着说，"你通过了最后一关。"

### 留白式
> 她没有回答，只是望向窗外。雪开始下了。

### 金句式
> 有些路，注定要一个人走。但至少，他知道方向是对的。
