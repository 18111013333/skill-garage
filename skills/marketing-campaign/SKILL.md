# 营销活动技能

## 核心能力

### 1. 活动策划
- 活动主题设计
- 活动流程规划
- 资源需求评估
- 预算分配

### 2. 促销设计
- 优惠券策略
- 满减活动
- 限时秒杀
- 拼团活动

### 3. 内容营销
- 短视频策划

- 图文内容

### 4. 用户运营

### 日常活动
| 类型 | 频率 | 目的 |
|------|------|------|
| 限时秒杀 | 每日 | 引流、促活 |
| 新人专享 | 常驻 | 新客转化 |
| 会员日 | 每周/月 | 复购、忠诚 |
| 拼团活动 | 不定期 | 拉新、裂变 |

## 促销策略

### 优惠券设计
| 类型 | 面额 | 门槛 | 适用场景 |
|------|------|------|----------|
| 无门槛券 | 5-10元 | 无 | 新客转化 |
| 满减券 | 20-50元 | 满100+ | 提客单价 |
| 折扣券 | 8-9折 | 无 | 清库存 |
| 运费券 | 包邮 | 无 | 降低门槛 |

### 活动节奏
```
预热期(3-7天) → 爆发期(1-3天) → 返场期(1-2天) → 复盘期
```
