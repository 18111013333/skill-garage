# 抖音小店运营技能

## 适用场景

- 抖音小店商品上架管理
- 抖音直播带货运营
- 抖音达人/团长合作
- 抖音数据分析优化
- 抖音订单售后管理

### 1. 商品管理
- 批量上架商品
- 商品标题优化
- 商品主图设计建议
- 价格策略制定
- 库存管理

### 2. 直播带货
- 直播选品建议
- 直播脚本生成
- 直播数据监控
- 直播复盘分析

### 3. 达人合作
- 达人筛选匹配
- 合作方案设计
- 佣金谈判
- 效果追踪

### 4. 数据分析
- 流量分析
- 转化分析
- 竞品分析
- ROI 计算

### 5. 订单管理
- 订单处理
- 发货管理
- 售后处理
- 评价管理

## 抖音小店关键指标

| 指标 | 说明 | 目标值 |
|------|------|--------|
| 曝光量 | 商品展示次数 | > 10000/天 |
| 点击率 | 点击/曝光 | > 3% |
| 转化率 | 订单/点击 | > 5% |
| 客单价 | 订单金额/订单数 | > 50元 |
| 退款率 | 退款/订单 | < 5% |
| 好评率 | 好评/总评 | > 95% |

## 直播带货流程

1. **选品** → 确定主推款、引流款、利润款
2. **脚本** → 开场、产品介绍、互动、逼单
3. **预热** → 短视频预热、粉丝通知
4. **直播** → 控场、互动、促单
5. **复盘** → 数据分析、优化改进

## 达人合作模式

| 模式 | 说明 | 佣金范围 |
|------|------|----------|
| 纯佣 | 按销量结算佣金 | 15-30% |
| 混合 | 坑位费+佣金 | 5-15% + 500-5000元 |
| 年框 | 长期合作优惠 | 协商 |

## 抖音算法要点

- **完播率** > **互动率** > **转发率** > **点赞率**
- 前3秒决定用户是否继续看
- 互动（评论、分享）权重高
- 直播间停留时长影响推流

## 注意事项

1. 遵守抖音电商规则
2. 不发布违禁词
3. 保证商品质量
4. 及时处理售后
5. 维护店铺评分
