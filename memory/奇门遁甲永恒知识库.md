## 二、排盘方法

### 2.1 时家奇门排盘步骤

```
排盘六步法：
┌─────────────────────────────────────────┐
│ 第一步：确定局数                         │
│    根据节气和日干支确定阴遁/阳遁局数     │
├─────────────────────────────────────────┤
│ 第二步：地盘三奇六仪                     │
│    根据局数排布地盘三奇六仪             │
├─────────────────────────────────────────┤
│ 第三步：天盘三奇六仪                     │
│    根据时干确定值符，转动天盘           │
├─────────────────────────────────────────┤
│ 第四步：排八门                          │
│    根据值使确定八门位置                 │
├─────────────────────────────────────────┤
│ 第五步：排九星                          │
│    根据值符确定九星位置                 │
├─────────────────────────────────────────┤
│ 第六步：排八神                          │
│    根据阳遁/阴遁确定八神位置            │
└─────────────────────────────────────────┘
```

### 2.2 阴遁阳遁

|------|----------|------|
