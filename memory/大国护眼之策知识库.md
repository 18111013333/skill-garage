## 五、常见眼病防治

### 1. 干眼症

| 项目 | 内容 |
|------|------|
| 症状 | 眼干、眼涩、异物感、视力波动 |
| 原因 | 泪液分泌不足、蒸发过快 |
| 治疗 | 人工泪液、热敷、眨眼训练 |
| 预防 | 多眨眼、避免长时间用眼、保持湿度 |

### 2. 结膜炎

| 项目 | 内容 |
|------|------|
| 症状 | 眼红、分泌物增多、异物感 |
| 原因 | 细菌、病毒、过敏 |
| 治疗 | 抗生素眼药水、抗病毒眼药水 |
| 预防 | 注意卫生、避免揉眼、不共用毛巾 |

### 3. 白内障

| 项目 | 内容 |
|------|------|
| 症状 | 视力模糊、畏光、色觉改变 |
| 原因 | 晶状体混浊、年龄、糖尿病、外伤 |
| 治疗 | 手术是唯一有效方法 |
| 预防 | 避免紫外线、控制血糖、定期检查 |

### 4. 青光眼

| 项目 | 内容 |
|------|------|
| 症状 | 眼胀、头痛、视力下降、视野缺损 |
| 原因 | 眼压升高、视神经损伤 |
| 治疗 | 降眼压药物、激光、手术 |
| 预防 | 定期检查、早发现早治疗 |
