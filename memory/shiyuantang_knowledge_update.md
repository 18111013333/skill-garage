# 一、眼镜行业深度知识

## 1. 行业成本结构分析

### 镜片成本构成
- **镜片单位成本**：约7.31元/片（明月镜片2021年数据）
- **成镜单位成本**：约57.03元/副
- **最高售价**：可达1898元/副

### 为什么毛利高但从业者赚不到钱？

#### 原因一：前期投入高
- 验光设备：电脑验光仪、综合验光仪、验光组合台
- 加工设备：镜片磨边机、中心仪
- 备货成本：镜框、镜片、镜盒库存
- 人员成本：验光师、加工师、销售人员工资

#### 原因二：消费频次低
- 一副眼镜至少戴1年
- 一家门店每天5副左右消费已算不错
- 同行竞争激烈

#### 原因三：国产品牌竞争力不足
- 依视路、蔡司销售额远超国产品牌
- 依视路大肆并购国内企业：万新光学、雅瑞光学（暴龙、陌森）

# 二、国家政策与规范

## 1. 眼镜行业国家标准

### 强制性标准（GB）
- **GB 10810**：眼镜镜片国家标准
- **GB 13511**：配装眼镜国家标准
- **GB/T 14214**：眼镜架国家标准

### 推荐性标准（GB/T）
- **GB/T 17341**：眼镜验光配镜技术规范
- **GB/T 26397**：眼科光学验光仪

## 2. 验光师资质要求

# 三、视力训练新知识

## 1. 科学训练方法

### 训练原理
- 改善睫状肌调节能力
- 增强眼外肌协调性
- 提高视觉信息处理能力

### 训练设备
- **眼动力训练仪**：通过视觉追踪训练
- **红光治疗仪**：改善眼底血液循环
- **双眼视功能训练仪**：改善双眼协调

### 训练效果
- 假性近视：可恢复正常视力
- 真性近视：控制度数增长
- 弱视：提高矫正视力

## 2. 训练注意事项

### 适用人群
- 6-18岁青少年
- 假性近视患者
- 弱视患者
- 视疲劳人群

### 训练频率
- 每周2-3次
- 每次20-30分钟
- 坚持3-6个月见效
