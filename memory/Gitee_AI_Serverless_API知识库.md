## 十、API 调用示例

```python
import requests

API_URL = "https://ai.gitee.com/v1/chat/completions"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "model": "GLM-5",
    "messages": [
        {"role": "user", "content": "你好"}
    ]
}
response = requests.post(API_URL, headers=headers, json=data)
print(response.json())
```
