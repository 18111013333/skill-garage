# 一、爆款图文底层逻辑

## 1. 自我表达与用户需求的交叉点

### 三大核心原则
1. **与用户有关联** - 不是高高在上的说教者，而是站在同一阵营的朋友
2. **站在用户角度** - 从用户需求出发，思考他们真正关心的问题
3. **对用户有价值** - 能够解决实际问题或带来新的启发

### 案例
- ❌ 错误：单纯讲述自己如何坚持健身
- ✅ 正确：从用户减肥、增肌需求出发，分享适合不同基础的健身计划

---

## 2. 爆款都是重复的

### 标题相似法则
1. **先找词，再模仿**
   - 考拉新媒体导航
   - 新媒派
   - 内容创作平台"就看"

2. **两种必火选题模式**
   - **聚合类**：知识密度高，一站式信息获取
     - "想靠写作赚钱，推荐这5本书"
     - "职场必备的10个高效办公软件"
     - "旅游必看的15个世界文化遗产"
   
   - **同款类**：借助明星效应和热门影视综艺
     - "明星同款穿搭大揭秘"
     - "《XX》剧中女主的同款口红"
     - "综艺《XX》中嘉宾的同款健身器材"

### 内容相似法则
1. **封面制作法则**
   - 美图 + 文字 + 排版
   - 信息层级分类
   - 风格一致性

2. **竖屏封面规范**
   - 比例：3:4
   - 主体占比：60%-70%
   - 图片清晰度越高越好
   - 精修图片，去除瑕疵

3. **封面优化技巧**
   - 突出重点，善用关键词
   - 添加数字："5个技巧""10个方法"
   - 贴近用户，引发共鸣
   - 设置悬念，引起好奇心

---

## 3. 始终保持专一性

### 名字和内容保持一致
- 名字是用户对创作者的第一印象
- 通过名字快速了解主要内容
- 内容围绕名字展开，保持连贯性

### 专一领域深耕
- 选择熟悉且感兴趣的领域
- 持续输出高质量内容
- 树立权威性和专业性
- 关注最新动态和发展趋势

# 二、爆款图文设计规范

## 1. 尺寸规范

| 平台 | 尺寸 | 比例 |
|------|------|------|
| 抖音 | 1080×1920 | 9:16 |
| 小红书 | 1080×1440 | 3:4 |
| 微信公众号 | 900×500 | 16:9 |
| 朋友圈 | 1080×1080 | 1:1 |

## 2. 文字规范

### 标题字号
- 主标题：80-100px
- 副标题：50-60px
- 正文：30-40px

### 文字颜色
- 主标题：品牌色或高对比色
- 副标题：比主标题浅一级
- 正文：深灰色（#333333）

### 文字排版
- 居中对齐
- 行间距：1.5-2倍
- 字间距：适当加宽

## 3. 图片规范

### 图片质量
- 分辨率：至少1080px宽
- 格式：JPG/PNG
- 大小：不超过5MB

### 图片处理
- 调整色彩和光影
- 去除瑕疵
- 突出主体
- 保持清晰度

# 三、爆款图文内容结构

## 1. 标题公式

### 数字冲击型
- "一台设备30万，只为测准你的度数"
- "服务南充10万+顾客"
- "5个技巧让孩子远离近视"

### 痛点直击型
- "孩子近视不用慌，科学训练有方法"
- "配眼镜为什么要做视功能检查？"
- "度数不准，眼镜白配"

### 对比反差型
- "大牌镜片5折起，品质不打折"
- "小店价格，大店服务"
- "专业验光，平价配镜"

### 情感共鸣型
- "别让近视挡住孩子的未来"
- "看清这一行，世界都清晰了"
- "一副好眼镜，改变的不只是视力"

### 口语传播型
- "南充配镜哪家强？视元堂里找答案"
- "配眼镜，我只信赖视元堂"
- "孩子配镜，妈妈放心之选"

## 2. 内容结构

```
【标题】吸睛开场（3秒抓住注意力）
【照片】真实场景（建立信任感）
【痛点】引发共鸣（戳中用户痛点）
【方案】专业解决（展示解决方案）
【信任】数据背书（用数据建立信任）
【行动】引导转化（明确行动指令）
```

## 3. 亮点呈现

### 三大亮点法则
- 每张图最多3个亮点
- 用★或✓标记
- 简洁有力，直击要点

### 示例
```
★ 不是随便测测
★ 是专业验光
★ 持证验光师
```
