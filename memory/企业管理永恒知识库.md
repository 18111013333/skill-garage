## 一、战略管理

### 1.1 战略规划框架

```
企业战略规划五步法：
┌─────────────────────────────────────────┐
│ 1. 愿景使命价值观确立                    │
│    ↓                                    │
│ 2. 外部环境分析（PESTEL）               │
│    ↓                                    │
│ 3. 内部资源能力分析（SWOT）             │
│    ↓                                    │
│ 4. 战略选择与制定                       │
│    ↓                                    │
│ 5. 战略实施与评估                       │
└─────────────────────────────────────────┘
```

### 1.2 PESTEL分析模型

| 维度 | 分析内容 | 应用场景 |
|------|----------|----------|
| **P政治** | 政策法规、政府态度 | 建筑行业资质政策 |
| **E经济** | 经济周期、利率汇率 | 投资决策时机 |
| **S社会** | 人口结构、消费习惯 | 健康管理市场需求 |
| **T技术** | 技术变革、创新趋势 | 视力训练新技术 |
| **E环境** | 环保要求、气候变化 | 建筑工程环保标准 |
| **L法律** | 法律法规、合规要求 | 企业合规经营 |

### 1.3 SWOT分析矩阵

```
           │ 优势(S)        │ 劣势(W)
───────────┼────────────────┼────────────────
机会(O)    │ SO战略         │ WO战略
           │ 发挥优势抓机会 │ 克服劣势抓机会
───────────┼────────────────┼────────────────
威胁(T)    │ ST战略         │ WT战略
           │ 发挥优势避威胁 │ 减少劣势避威胁
```

### 1.4 战略执行工具

| 工具 | 说明 | 使用方法 |
|------|------|----------|
| **OKR** | 目标与关键结果 | 季度设定，每周追踪 |
| **BSC** | 平衡计分卡 | 财务/客户/流程/学习四维度 |
| **KPI** | 关键绩效指标 | 量化目标，月度考核 |
| **PDCA** | 计划执行检查改进 | 持续循环改进 |

## 二、组织管理

### 2.1 组织架构设计原则

```
组织设计五原则：
1. 统一指挥原则 - 一个下属一个上级
2. 管理幅度原则 - 7-10人为宜
3. 权责对等原则 - 有权必有责
4. 分工协作原则 - 专业分工，协同合作
5. 精简高效原则 - 扁平化，减少层级
```

### 2.2 常见组织架构

| 类型 | 适用场景 | 优缺点 |
|------|----------|--------|
| **直线制** | 小型企业 | 简单高效，但缺乏专业分工 |
| **职能制** | 中型企业 | 专业分工，但多头领导 |
| **直线职能制** | 中大型企业 | 兼顾效率与专业 |
| **事业部制** | 多元化集团 | 灵活自主，但资源分散 |
| **矩阵制** | 项目型企业 | 灵活高效，但双重领导 |

### 2.3 团队建设五阶段

```
团队发展五阶段模型：
┌─────────────────────────────────────────┐
│ 1. 形成期（Forming）                     │
│    特征：礼貌客气，试探了解              │
│    策略：明确目标，建立规则              │
├─────────────────────────────────────────┤
│ 2. 风暴期（Storming）                    │
│    特征：冲突矛盾，权力争夺              │
│    策略：调解冲突，建立信任              │
├─────────────────────────────────────────┤
│ 3. 规范期（Norming）                     │
│    特征：形成规范，凝聚力增强            │
│    策略：巩固规范，促进协作              │
├─────────────────────────────────────────┤
│ 4. 执行期（Performing）                  │
│    特征：高效协作，成果显著              │
│    策略：充分授权，激励创新              │
├─────────────────────────────────────────┤
│ 5. 休整期（Adjourning）                  │
│    特征：任务完成，团队解散              │
│    策略：总结经验，表彰贡献              │
└─────────────────────────────────────────┘
```

## 三、财务管理

### 3.1 财务三大报表

| 报表 | 反映内容 | 关键指标 |
|------|----------|----------|
| **资产负债表** | 财务状况 | 资产负债率、流动比率 |
| **利润表** | 经营成果 | 毛利率、净利率、ROE |
| **现金流量表** | 现金流动 | 经营现金流、自由现金流 |

### 3.2 财务分析指标

```
盈利能力指标：
├── 毛利率 = (收入-成本)/收入 × 100%
├── 净利率 = 净利润/收入 × 100%
├── ROE = 净利润/净资产 × 100%
└── ROA = 净利润/总资产 × 100%

偿债能力指标：
├── 资产负债率 = 负债/资产 × 100%
├── 流动比率 = 流动资产/流动负债
└── 速动比率 = (流动资产-存货)/流动负债

运营能力指标：
├── 存货周转率 = 营业成本/平均存货
├── 应收账款周转率 = 营业收入/平均应收账款
└── 总资产周转率 = 营业收入/平均总资产
```

### 3.3 成本控制方法

| 方法 | 说明 | 应用场景 |
|------|------|----------|
| **标准成本法** | 设定标准，分析差异 | 生产制造 |
| **作业成本法** | 按作业归集成本 | 服务行业 |
| **目标成本法** | 以市场为导向 | 新产品开发 |
| **价值链分析** | 识别增值环节 | 战略成本管理 |

## 四、人力资源管理

### 4.1 招聘流程

```
招聘六步法：
1. 岗位分析 → 岗位说明书
2. 招聘计划 → 渠道选择
3. 简历筛选 → 初步匹配
4. 面试评估 → 结构化面试
5. 背景调查 → 核实信息
6. 录用决策 → 薪酬谈判
```

### 4.2 面试提问技巧

| 类型 | 问题示例 | 考察要点 |
|------|----------|----------|
| **行为面试** | 请举例说明你如何处理... | 过往行为表现 |
| **情景面试** | 如果遇到...你会怎么做 | 应变能力 |
| **压力面试** | 你为什么认为自己能胜任 | 抗压能力 |
| **能力面试** | 你最大的成就是什么 | 能力水平 |

### 4.3 绩效考核方法

| 方法 | 说明 | 优缺点 |
|------|------|--------|
| **KPI考核** | 关键绩效指标 | 量化明确，但可能忽视过程 |
| **360度评估** | 多维度评价 | 全面客观，但工作量大 |
| **OKR** | 目标与关键结果 | 激励创新，但需文化支撑 |
| **BSC** | 平衡计分卡 | 全面系统，但实施复杂 |

### 4.4 薪酬设计原则

```
薪酬设计四原则：
1. 内部公平 - 岗位价值评估
2. 外部竞争 - 市场薪酬调研
3. 个人公平 - 绩效挂钩
4. 过程公平 - 制度透明
```

## 六、风险管理

### 6.1 企业风险类型

| 风险类型 | 说明 | 应对策略 |
|----------|------|----------|
| **战略风险** | 战略决策失误 | 充分调研，专家论证 |
| **运营风险** | 日常运营问题 | 流程规范，内部控制 |
| **财务风险** | 资金链断裂 | 现金流管理，融资储备 |
| **市场风险** | 市场变化 | 多元化，灵活调整 |
| **法律风险** | 合规问题 | 法务审核，合规培训 |
| **声誉风险** | 品牌形象受损 | 危机公关，诚信经营 |

### 6.2 风险管理流程

```
风险管理四步法：
┌─────────────────────────────────────────┐
│ 1. 风险识别                              │
│    方法：头脑风暴、检查表、流程分析      │
├─────────────────────────────────────────┤
│ 2. 风险评估                              │
│    方法：概率影响矩阵、定性定量分析      │
├─────────────────────────────────────────┤
│ 3. 风险应对                              │
│    策略：规避、转移、减轻、接受          │
├─────────────────────────────────────────┤
│ 4. 风险监控                              │
│    方法：风险登记册、定期审查            │
└─────────────────────────────────────────┘
```

## 七、领导力与管理技能

### 7.1 领导风格

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **指令型** | 明确指示，严格监督 | 危机时刻、新员工 |

| **变革型** | 愿景激励，引领变革 | 组织变革 |

### 7.2 沟通技巧

```
有效沟通五要素：
1. 明确目的 - 想要达成什么
2. 了解对象 - 对方关心什么
3. 选择方式 - 面谈/书面/会议
4. 表达清晰 - 简洁明了
5. 反馈确认 - 确保理解一致
```

### 7.3 决策方法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| **SWOT分析** | 优势劣势机会威胁 | 战略决策 |
