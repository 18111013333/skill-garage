## 一、近视防控知识

### 1. 近视成因
| 因素 | 说明 |
|------|------|
| **遗传因素** | 父母近视，孩子近视风险高3-5倍 |
| **环境因素** | 光线不足、用眼距离过近、时间过长 |
| **行为因素** | 电子产品使用、户外活动不足 |

### 2. 近视分类
| 类型 | 度数范围 | 特点 |
|------|----------|------|
| **假性近视** | 可调节 | 睫状肌痉挛，可逆 |
| **轻度近视** | <300° | 眼轴轻度增长 |
| **中度近视** | 300-600° | 眼轴明显增长 |
| **高度近视** | >600° | 眼轴过长，并发症风险高 |

### 3. 近视预防
| 方法 | 说明 |
|------|------|
| **20-20-20法则** | 每20分钟看20英尺外20秒 |
| **户外活动** | 每天2小时以上 |
| **光线充足** | 阅读/工作光线500-1000lux |
| **正确姿势** | 一尺一拳一寸 |
| **定期检查** | 每半年检查一次视力 |

### 4. 近视控制
| 方法 | 效果 | 适用人群 |
|------|------|----------|
| **角膜塑形镜** | 延缓近视增长50-60% | 8岁以上 |
| **离焦镜片** | 延缓近视增长50-67% | 儿童/青少年 |
| **低浓度阿托品** | 延缓近视增长50% | 遵医嘱使用 |
| **多焦点软镜** | 延缓近视增长40-50% | 不愿戴框架者 |

## 二、配镜专业知识

### 1. 验光流程
```
问诊 → 电脑验光 → 综合验光 → 试戴调整 → 开具处方
```

### 2. 镜片材质
| 材质 | 优点 | 缺点 | 适用人群 |
|------|------|------|----------|
| **树脂** | 轻便、安全、易加工 | 易划伤 | 大多数人 |
| **PC** | 抗冲击、轻便 | 阿贝数低 | 儿童/运动 |
| **玻璃** | 耐磨、清晰 | 重、易碎 | 特殊需求 |
| **水晶** | 硬度高 | 重、贵 | 收藏/特殊需求 |

### 3. 镜片折射率
| 折射率 | 适用度数 | 特点 |
|--------|----------|------|
| **1.56** | <400° | 性价比高 |
| **1.60** | 400-600° | 均衡选择 |
| **1.67** | 600-800° | 更薄更轻 |
| **1.74** | >800° | 最薄最轻 |

### 4. 镜片功能
| 类型 | 功能 | 适用人群 |
|------|------|----------|
| **单光镜片** | 矫正视力 | 近视/远视 |
| **双光镜片** | 远近两用 | 老花眼 |
| **渐进镜片** | 远中近连续 | 老花眼 |

| **防蓝光镜片** | 过滤蓝光 | 电子产品用户 |
| **变色镜片** | 室内外两用 | 户外活动多者 |

### 5. 镜架选择
| 脸型 | 推荐镜架 |
|------|----------|
| **圆脸** | 方形、矩形镜架 |

## 三、常见眼病防治

### 1. 干眼症
| 项目 | 内容 |
|------|------|
| **症状** | 眼干、眼涩、异物感、视力波动 |
| **原因** | 泪液分泌不足、蒸发过快 |
| **治疗** | 人工泪液、热敷、眨眼训练 |
| **预防** | 多眨眼、避免长时间用眼、保持湿度 |

### 2. 结膜炎
| 项目 | 内容 |
|------|------|
| **症状** | 眼红、分泌物增多、异物感 |
| **原因** | 细菌、病毒、过敏 |
| **治疗** | 抗生素眼药水、抗病毒眼药水 |
| **预防** | 注意卫生、避免揉眼、不共用毛巾 |

### 3. 白内障
| 项目 | 内容 |
|------|------|
| **症状** | 视力模糊、畏光、色觉改变 |
| **原因** | 晶状体混浊、年龄、糖尿病、外伤 |
| **治疗** | 手术是唯一有效方法 |
| **预防** | 避免紫外线、控制血糖、定期检查 |

### 4. 青光眼
| 项目 | 内容 |
|------|------|
| **症状** | 眼胀、头痛、视力下降、视野缺损 |
| **原因** | 眼压升高、视神经损伤 |
| **治疗** | 降眼压药物、激光、手术 |
| **预防** | 定期检查、早发现早治疗 |

### 5. 黄斑变性
| 项目 | 内容 |
|------|------|
| **症状** | 中心视力下降、视物变形 |
| **原因** | 年龄、遗传、吸烟 |
| **治疗** | 抗VEGF药物、激光 |
| **预防** | 补充叶黄素、避免强光、戒烟 |

## 四、儿童视力保健

### 1. 视力发育规律
| 年龄 | 视力 | 检查重点 |
|------|------|----------|
| **0-1岁** | 光感-0.1 | 先天性眼病筛查 |
| **1-3岁** | 0.1-0.5 | 斜视、弱视筛查 |
| **3-6岁** | 0.5-0.8 | 建立视力档案 |
| **6-12岁** | 0.8-1.0 | 近视防控 |
| **12-18岁** | 1.0左右 | 度数监测 |

### 2. 儿童护眼要点
| 要点 | 说明 |
|------|------|
| **户外活动** | 每天2小时以上 |
| **电子产品** | 每次不超过20分钟 |
| **阅读距离** | 保持33-40厘米 |
| **阅读光线** | 500-1000lux |
| **睡眠时间** | 小学生10小时，初中生9小时 |
| **定期检查** | 每半年一次 |

### 3. 儿童近视信号
- 眯眼看东西
- 频繁眨眼
- 揉眼睛
- 歪头看东西
- 看电视往前凑
- 成绩下降

## 五、护眼小知识

### 1. 日常护眼
| 方法 | 说明 |
|------|------|
| **20-20-20法则** | 每20分钟看20英尺外20秒 |
| **眨眼训练** | 每分钟眨眼15-20次 |
| **热敷** | 40-45°C，10-15分钟 |
| **眼保健操** | 每天2次，每次5分钟 |
| **远眺** | 看远处绿色植物 |

### 2. 营养补充
| 营养素 | 作用 | 食物来源 |
|--------|------|----------|
| **叶黄素** | 保护黄斑 | 玉米、菠菜、蛋黄 |
| **维生素A** | 维持视力 | 胡萝卜、动物肝脏 |
| **DHA** | 促进视网膜发育 | 深海鱼、核桃 |
| **花青素** | 抗氧化 | 蓝莓、紫薯 |
| **维生素C** | 抗氧化 | 柑橘、猕猴桃 |

### 3. 生活习惯
- 充足睡眠（7-8小时）
- 避免熬夜
- 戒烟限酒
- 适度运动
- 保持心情愉快
