## 一、八字算命基础

### 1. 八字概念
八字，又称四柱，是指一个人出生时的干支历日期。年、月、日、时共四柱干支，每柱两字，合共八个字，故称八字。

### 3. 五行相生相克

#### 五行相生
```
木生火 → 火生土 → 土生金 → 金生水 → 水生木
```

#### 五行相克
```
木克土 → 土克水 → 水克火 → 火克金 → 金克木
```
