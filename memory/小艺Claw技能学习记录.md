## 二、求是技能（方法论）

### 核心价值
解决 AI "会思考但不会想问题"的根本缺陷

### 问题与解药对照表
| 问题 | 解药 |
|------|------|
| 抓不住重点 | 矛盾分析法 — 找主要矛盾 |
| 没调查就发言 | 调查研究 — 没调查没发言权 |
| 做完不自查 | 批评与自我批评 — 惩前毖后 |
| 遇阻就停 | 持久战略 — 战略藐视，战术重视 |
| 同时做多件事 | 集中兵力 — 伤十指不如断一指 |

### 核心原则
1. **实事求是作为总原则** — 先看事实，再下判断
2. **不是教条，是方法论** — 每条方法都有原著依据
3. **认真但不机械** — 能改善判断就用，增加负担就跳过

### 与七情六欲的关系
- **七情六欲** 解决"怎么说话" — 让 AI 有人味、有情感、有态度
- **求是 Skill** 解决"怎么想问题" — 让 AI 会分析、会调查、会决策
- 两者互补：一个武装嘴巴，一个武装大脑
