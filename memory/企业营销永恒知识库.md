## 一、营销战略

### 1.1 STP营销战略

```
STP三步法：
┌─────────────────────────────────────────┐
│ S - 市场细分（Segmentation）             │
│     按地理/人口/心理/行为细分市场        │
├─────────────────────────────────────────┤
│ T - 目标市场（Targeting）                │
│     评估细分市场，选择目标市场           │
├─────────────────────────────────────────┤
│ P - 市场定位（Positioning）              │
│     差异化定位，占领心智                 │
└─────────────────────────────────────────┘
```

### 1.2 市场细分维度

| 维度 | 细分标准 | 应用示例 |
|------|----------|----------|
| **地理** | 地区、城市、气候 | 南充本地市场 |
| **人口** | 年龄、性别、收入 | 儿童近视防控（6-15岁） |
| **心理** | 生活方式、价值观 | 注重健康的家庭 |
| **行为** | 使用频率、忠诚度 | 复购客户、新客户 |

### 1.3 竞争战略

| 战略 | 说明 | 适用条件 |
|------|------|----------|
| **成本领先** | 最低成本生产 | 规模经济、成本控制能力强 |
| **差异化** | 独特价值主张 | 创新能力强、品牌价值高 |
| **集中化** | 聚焦细分市场 | 资源有限、细分市场明确 |

### 1.4 波特五力分析

```
行业竞争五力模型：
                    ┌─────────────┐
                    │ 潜在进入者   │
                    │ 威胁程度     │
                    └──────┬──────┘
                           ↓
┌──────────┐        ┌─────────────┐        ┌──────────┐
│ 供应商   │ ───→  │  行业竞争   │  ←─── │  购买者   │
│ 议价能力 │        │  现有竞争者 │        │ 议价能力 │
└──────────┘        └──────┬──────┘        └──────────┘
                           ↓
                    ┌─────────────┐
                    │  替代品     │
                    │ 威胁程度     │
                    └─────────────┘
```

## 四、销售管理

### 4.1 销售流程

```
销售七步法：
┌─────────────────────────────────────────┐
│ 1. 客户开发                              │
│    渠道：门店、线上、转介绍、异业合作    │
├─────────────────────────────────────────┤
│ 2. 需求挖掘                              │
│    方法：提问倾听、观察分析              │
├─────────────────────────────────────────┤
│ 3. 产品介绍                              │
│    技巧：FAB法则、场景化描述             │
├─────────────────────────────────────────┤
│ 4. 异议处理                              │
│    方法：认同-转化-确认                  │
├─────────────────────────────────────────┤
│ 5. 促成交易                              │
│    技巧：二选一、限时优惠、从众心理      │
├─────────────────────────────────────────┤
│ 6. 交付服务                              │
│    要点：专业、高效、超出预期            │
├─────────────────────────────────────────┤
│ 7. 售后跟进                              │
│    内容：使用指导、满意度回访、复购提醒  │
└─────────────────────────────────────────┘
```

### 4.2 FAB销售法则

| 要素 | 说明 | 示例（眼镜） |
|------|------|--------------|
| **F-Feature** | 产品特性 | 蔡司镜片，德国进口 |
| **A-Advantage** | 产品优势 | 透光率高，耐磨防污 |
| **B-Benefit** | 客户利益 | 视野清晰，使用寿命长 |

### 4.3 异议处理技巧

| 异议类型 | 处理方法 | 话术示例 |
|----------|----------|----------|
| **价格异议** | 价值塑造 | "这个价格确实不便宜，但考虑到..." |
| **功能异议** | 需求匹配 | "您担心的这个问题，其实..." |
| **时间异议** | 创造紧迫 | "现在活动优惠，错过就没了..." |
| **竞品异议** | 差异化 | "我们家独特的地方是..." |

## 五、客户管理

### 5.1 客户分类管理

| 客户类型 | 特征 | 管理策略 |
|----------|------|----------|
| **VIP客户** | 高价值、高忠诚 | 专属服务、定期关怀 |
| **潜力客户** | 有消费能力、待开发 | 重点跟进、需求挖掘 |
| **普通客户** | 低频消费 | 维护关系、促进复购 |
| **流失客户** | 长期未消费 | 唤醒活动、了解原因 |

### 5.2 客户生命周期

```
客户生命周期管理：
┌─────────────────────────────────────────┐
│ 获客期                                   │
│ 目标：吸引新客户                         │
│ 策略：引流活动、首单优惠                 │
├─────────────────────────────────────────┤
│ 成长期                                   │
│ 目标：提升消费频次                       │
│ 策略：会员权益、复购激励                 │
├─────────────────────────────────────────┤
│ 成熟期                                   │
│ 目标：最大化客户价值                     │
│ 策略：增值服务、转介绍激励               │
├─────────────────────────────────────────┤
│ 衰退期                                   │
│ 目标：延长生命周期                       │
│ 策略：唤醒活动、满意度调查               │
├─────────────────────────────────────────┤
│ 流失期                                   │
│ 目标：挽回客户                           │
│ 策略：专属优惠、问题解决                 │
└─────────────────────────────────────────┘
```

### 5.3 客户满意度管理

| 指标 | 说明 | 提升方法 |
|------|------|----------|
| **CSAT** | 客户满意度 | 服务质量提升 |
| **NPS** | 净推荐值 | 超出预期体验 |
| **CES** | 客户费力指数 | 简化服务流程 |

### 5.4 客户投诉处理

```
投诉处理五步法：
1. 倾听 - 让客户充分表达
2. 共情 - 理解客户感受
3. 道歉 - 真诚表达歉意
4. 解决 - 提供解决方案
5. 跟进 - 确保问题解决
```

## 八、视元堂营销实战指南

### 8.1 目标客户画像

| 客户群体 | 需求特点 | 营销重点 |
|----------|----------|----------|
| **儿童家长** | 近视防控、专业配镜 | 科普教育、效果案例 |
| **青少年** | 时尚镜框、性价比 | 潮流款式、学生优惠 |
| **中青年** | 品质镜片、商务形象 | 品牌背书、专业服务 |
| **老年人** | 老花镜、视力保健 | 贴心服务、健康关怀 |

### 8.2 营销话术要点

```
开场话术：
"您好，请问是给孩子看眼睛还是自己配镜？"

需求挖掘：
"孩子现在视力怎么样？有没有去医院检查过？"
"平时用眼习惯怎么样？看电子产品多吗？"

产品推荐：
"根据孩子的情况，我建议选择XX镜片，因为..."
"我们家这款镜框特别适合孩子，因为..."

促成话术：
"现在活动期间有优惠，算下来很划算"
"孩子视力要趁早干预，越早效果越好"

售后话术：
"有任何问题随时联系我们"
"三个月后来复查一下视力"
```
