## V4.3.2 第三阶段最终收尾完成

### 修复的 3 个问题

#### 1. Vector 模式接通真实 Embedding 后端
- **问题**: vector_mode 仍为 degraded
- **修复**: 
  - 新增 `QwenEmbeddingEngine` 类，直接调用 Gitee AI API
  - 启动时测试 API 连接，确定模式
  - 成功连接则 `embedding`，失败则 `degraded`
- **验证**: `Vector 模式: embedding`

#### 2. LazyLoader 真正接入结果内容装配
- **问题**: 搜索结果直接 read_text()，未走 LazyLoader
- **修复**:
  - 搜索结果装配时先 `lazy_loader.register()`
  - 通过 `lazy_loader.load()` 加载内容
  - 超预算时返回截断版本
- **验证**: `LazyLoader 已加载: 2 个`

#### 3. Query Rewrite 真正参与搜索召回
- **问题**: rewrites 只生成列表，未参与召回
- **修复**:
  - `search()` 中对 rewrites 进行搜索
  - 将 rewrite 结果加入 RRF 融合
  - 权重设为 0.8（低于原始查询）
- **验证**: `Rewrites: ['搜索document', 'find文档', '查找文档', '搜索文件', 'search文档']`
